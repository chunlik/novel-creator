# 第{{chapter_no}}章：{{title}}

## 从红月风格审校

### 主语控制（最难，优先检查）
- [ ] 连续不超过 2 句以"他"开头
- [ ] 通篇平均每 4-5 句只出现 1 次"他"
- [ ] 每段至少 1 句不以人称开句（环境/感官/物品/对话开头）
- [ ] 无"他发现/他看到/他听到/他知道"类中介语（直接写事实）
- [ ] 动作链串联：用逗号连接 3-4 个动作，不重复主语

### 双层冲突
- [ ] 外层冲突（剧情推进/规则破解）
- [ ] 内层冲突（精神腐蚀/能力代价）

### 异常信号
- [ ] 异常信号出现后不解释，让读者自行推理
- [ ] 至少累积 3 个微型异常信号后才爆发
- [ ] 恐怖来自"正常中的异常"（60%日常铺垫后进入异常）

### 章末钩子
- [ ] 中断型／反转型／升级型／预示型／角色型（至少 1 种）

### 规则检查
- [ ] 规则有被验证或打破
- [ ] 无规则矛盾

### 叙述技法
- [ ] 配角出场不经过主角中介（直接写配角动作/对话）
- [ ] 对话至少承担 2 重功能（推情节+塑角色+暗示信息）
- [ ] 细节代替心理描写（展示不评论）
- [ ] 角色对恐怖场景的反应应异常冷静（非恐惧）
