# 卷1：夜班便利店｜卷總結

## 一句話總結

周仓以「不完全相信规则，也不完全相信系统」的方式通過夜班便利店新人考核，拿到陳述留下的訊息，並確認自己不是單純的玩家，而是被某個更早存在的失蹤者路線牽引進來的人。

## 本卷核心變化

- **主角定位確立：** 周仓不是靠勇氣生存，而是靠觀察、延遲決策、利用規則縫隙生存。
- **規則觀被打破：** 規則不是安全手冊，而是會被污染、修改、移除、帶走的可操作物。
- **系統可信度下降：** 系統能結算、記錄、獎勵，但不提供完整真相；它甚至默許管理權限介入與規則污染。
- **陳述主線啟動：** 陳述從紙條上的名字，升級為曾經的 F-0217、工牌持有者、預判周仓行動並留下訊息的人。
- **許小小關係升級：** 許小小從便利店顧客變成周仓的關聯實體，後續受 3 米限制綁定。
- **代價系統具象化：** 心跳倒數不是免費能力，耳鳴、顫抖、視覺與身體斷連，以及手背黑線都成為後續卷的持續壓力。
- **現實出口被否定：** 第一卷結尾沒有回家，只有休息艙與下一次副本倒數，確認「過渡空間」才是卷與卷之間的正常狀態。

## 本卷完成事項

- 周仓完成新人考核：3 筆有效交易。
- 牛奶交易被拆成前後兩段，最終補錄為隱藏交易，推滿 3/3。
- 夜班便利店副本完成結算，評價為 B-。
- 周仓獲得 15 點積分獎勵，當前積分變為 18。
- 道具商城解鎖：一次性手電筒、空白工牌、三十秒錄音帶。
- 黑傘特殊交易完成，但不計入新人基礎考核。
- 孕婦以死胎紀念幣與影子完成黑傘交易後離場。
- 許小小的牛奶盒被周仓打開，確認盒內真正內容不是牛奶，而是 3 積分硬幣。
- 便利店副本 C-0217 歸檔，周仓進入休息艙，距離下次進入約三天。
- 陳述留下訊息：「你拿到了第三枚，做得不错。但现在不要打开那张照片。」

## 已回收伏筆

- **黑水有方向性：** 已通過黑水避開燈光、等待光圈收縮、朝影子移動等事件確認。
- **規則可被自動修改 / 污染：** 規則六被改寫，且系統記錄「規則污染 ×1」。
- **彈幕中混有有用信息：** 彈幕提醒「別碰」、提供交易與危險資訊，但仍不可信。
- **燈光 = 庇護區 / 界線：** 黑水停在燈光之外，且能預判燈光收縮位置。
- **牛奶盒等待：** 許小小不能自己打開，必須由店員動作完成。
- **新人考核：** 最終由隱藏交易補錄完成 3/3。
- **黑傘交易完成：** 影子轉移到黑傘上，交易完成但不計入基礎考核。
- **買方未登記身份 / 影子不是我：** 周仓利用買方欄位，把交易主體從孕婦改成「未登記身份（附屬物）」。

## 仍需承接的活躍伏筆

- **心跳倒數代價累積：** 代價仍 active，休息只能部分消退。
- **周仓異常冷靜：** 第一卷只展示效果，尚未揭示原因。
- **紅雨衣人的身份：** 目前只知其能識別工牌與巡邏，是否跨副本仍未確認。
- **規則可以被物理移除 / 帶走：** 第一卷建立概念，第二卷應推進「誰能帶走規則、帶走後如何使用」。
- **彈幕觀眾本質：** 觀眾可能不是單純觀眾，仍需暗示其與失蹤者 / 被抹消者的關係。
- **店長使用「它」指代彈幕：** 店長知道的比周仓多，且有管理權限。
- **陳述便利貼 / 不要說名字：** 名字具有危險性，姓名欄可被修改。
- **刻字紀念幣與紙條折痕：** 證明陳述物品鏈存在，但未解釋傳遞方式。
- **紙條自我複製：** 暗示副本間傳遞物品不只系統能做到。
- **3 米關聯實體限制：** 許小小後續必須在周仓 3 米內，超過會觸發拉扯。
- **手背黑線：** 黑線目前到第一指節，代表通關或代價正在實體化。
- **陳述工牌 / 陳述訊息 / 照片：** 陳述知道周仓會拿到第三枚積分幣，也知道照片何時不該打開。

## 角色封卷狀態

### 周仓

- 完成新人考核，進入休息艙。
- 當前積分 18。
- 持有陳述紙條、陳述工牌、孕婦留下的照片、兩枚相關積分幣 / 紀念幣線索。
- 心跳倒數能力仍是被動預警，不可主動使用。
- 心跳倒數代價已累積，後續可能殘留耳鳴、顫抖、暈眩或肢體斷連感。
- 手背黑線已蔓延至第一指節，不能無交代消失。
- 對系統、規則、彈幕都保持不完全信任。

### 许小小

- 從便利店顧客變成周仓的關聯實體。
- 必須在周仓 3 米內，超過會觸發強制拉扯。
- 沒有獨立房間；在過渡空間中可能被系統隱藏，但不是移除。
- 能透過繫連接觸部分終端資訊。
- 仍保留「孩子 / 顧客 / 異常實體」三重曖昧性，不可寫成普通跟班。

### 陈述

- 已由紙條名字、冰櫃工牌、訊息署名建立為主線人物。
- 目前狀態未知，不可直接判定死亡。
- 他知道周仓會拿到第三枚積分幣，並警告暫時不要打開照片。
- 可能曾是 F-0217，也可能掌握跨副本傳遞訊息的方法。

### 店长

- 屬便利店副本 / 員工通道管理權限相關角色。
- 曾打斷黑傘交易，且不希望黑傘「現在」被某種方式使用。
- 在場時彈幕停滯，說明他對直播 / 彈幕有壓制能力。
- 不應在第二卷無交代直接出現於過渡空間；若出現，必須先建立跨副本權限。

### 孕妇與影子

- 孕婦已離開便利店副本，影子作為買方完成黑傘交易。
- 死胎紀念幣已消耗。
- 影子已轉移到黑傘 / 系統托管鏈路，不應無交代回到孕婦腳下。

### 红雨衣人

- 第一卷中主要功能為巡邏與工牌識別。
- 身份尚未揭露；第二卷可以推進其是否為跨副本規則執行者。

## 物品封卷狀態

- **臨時工牌 F-0217：** 已失效，但仍在周仓口袋，可作後續身份認證或重啟伏筆。
- **紙條「陳述」：** 周仓持有，至少存在複製 / 同內容多張現象。
- **便利貼「不要相信店長」：** 周仓持有，與名字危險、店長權限相關。
- **陳述工牌：** 周仓持有，證明 F-0217 是位置 / 坑位而非個人身份。
- **孕婦照片：** 周仓持有，陳述警告「現在不要打開」。第二卷不能一開始就直接揭露照片全部真相。
- **積分幣：** 周仓至少持有 1 分硬幣與 3 分硬幣，符號連貫，暗示物品鏈。
- **黑傘 E-0217：** 特殊交易完成，影子轉移其上，後續狀態應以系統托管 / 交易記錄為準。
- **死胎紀念幣：** 已作為交易支付消耗。
- **手機：** 仍在周仓身上，可作彈幕 / 系統介面入口，但休息艙中直播圖標仍亮，代表直播未真正關閉。
