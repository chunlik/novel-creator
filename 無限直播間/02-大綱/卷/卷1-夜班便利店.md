# 卷1·夜班便利店

## 本卷功能（寫卷前確認）

- 建立世界：副本規則、黑水、心跳倒數、交易核心邏輯
- 注入驅動力：紙條出場
- 建立人物：周倉、許小小、店長
- 埋陳述線：便利貼+工牌，有人走過同樣的路
- 埋伏筆：黑傘、紀念幣、黑水、規則污染
- 卷末狀態：周倉通關，成為F級主播，許小小成為關聯實體

## 章節結構

### Ch1 規則的第一夜
| 項目 | 內容 |
|------|------|
| 功能 | 建立世界+第一次心跳倒數 |
| 事件 | 周倉被拉入便利店，接收七條規則。探索過程觸發第一次心跳倒數 |
| 謎團 | M1（紙條出場，系統查無此人） |
| 章末鉤子 | 收銀機上有上一班留下的痕跡——有人來過，而且留下了東西 |
| 涉及場所 | 便利店前場、收銀台 |

### Ch2 牛奶
| 項目 | 內容 |
|------|------|
| 功能 | 許小小出場+交易規則展示 |
| 事件 | 許小小進店買牛奶。周倉拒絕交易。觀察到她的傷、她的眼神 |
| 謎團 | 許小小的身份（讀者：這個小孩為什麼會出現在這裡？） |
| 章末鉤子 | 門口出現紅雨衣人——新來的，不是客人。他站在門口不動 |
| 涉及場所 | 便利店前場 |

### Ch3 冰櫃底下的水聲
| 項目 | 內容 |
|------|------|
| 功能 | 黑水出現+規則開始不對勁 |
| 事件 | 紅雨衣人巡邏。規則異常。黑水從冰櫃下方滲出。彈幕開始活躍 |
| 謎團 | M3（黑水） |
| 章末鉤子 | 第三排貨架深處有人——是孕婦。她在翻找什麼 |
| 涉及場所 | 便利店前場、冰櫃區 |

### Ch4 黑傘
| 項目 | 內容 |
|------|------|
| 功能 | 黑傘交易核心+規則深層展示 |
| 事件 | 孕婦用黑傘交易。規則開始顯露交易的本質 |
| 謎團 | 黑傘代表什麼？交易真正的代價是什麼？ |
| 章末鉤子 | 店長第一次出現——他不是來換班的，是來"查看進度"的 |
| 涉及場所 | 便利店前場 |

### Ch5 燈光下
| 項目 | 內容 |
|------|------|
| 功能 | 衝突升溫+陳述線索出現 |
| 事件 | 店長在場時規則收緊。彈幕活躍。周倉發現收銀台下有便利貼——陳述留下的 |
| 謎團 | 陳述的便利貼（"不要相信店長"） |
| 章末鉤子 | 便利貼背面有字——"他在聽。不要說名字。" |
| 涉及場所 | 收銀台 |

### Ch6 打烊
| 項目 | 內容 |
|------|------|
| 功能 | 店長清算+規則反轉 |
| 事件 | 店長揭示部分真相——交易不完成會怎樣。黑水從門縫滲入。規則開始自相矛盾 |
| 謎團 | 店長到底是什麼？他是被同化的人嗎？ |
| 章末鉤子 | 收銀機自己打開——裡面有一枚紀念幣，背面刻著"陳" |
| 涉及場所 | 便利店全場 |

### Ch7 通關
| 項目 | 內容 |
|------|------|
| 功能 | 高潮+通關+許小小成為關聯實體 |
| 事件 | 周倉完成交易。心跳倒數爆發。便利店消失。結算頁面出現 |
| 謎團 | 周倉到底交易了什麼？（他自己也不確定） |
| 章末鉤子 | 白光亮起。系統音："F級主播，恭喜通關。你的關聯實體已綁定。" |
| 涉及場所 | 便利店→結算空間 |

### Ch8 沒有窗戶的房間
| 項目 | 內容 |
|------|------|
| 功能 | 建立過渡空間世界觀+喘口氣+伏筆 |
| 事件 | 周倉進入休息室。探索終端機、抽屜。進入大廳，見到其他主播（1-2人，不說話） |
| 謎團 | M1（紙條再次出現——在休息室的抽屜裡。他明明沒放進去。紙條自己來了） |
| 章末鉤子 | 紙條折痕裡有一個微小標記——和陳述工牌背面的一模一樣 |
| 涉及場所 | 過渡空間（休息室→大廳） |
