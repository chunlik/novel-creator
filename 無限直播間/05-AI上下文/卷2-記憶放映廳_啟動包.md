## 第一章開場狀態

周仓站在休息艙內，不是在家，也不是回到現實世界。屏幕上仍有下次進入倒數，直播圖標仍亮。這代表休息不是完全安全，只是副本與副本之間的緩衝區。

許小小不應以普通人方式敲門或走進來。她可以先被系統隱藏在視野外，當周仓移動到 3 米邊界時被拉出，或從終端機反光 / 屏幕角落短暫顯現。她的出現要服務於「關聯實體不是同伴，而是限制」這個核心。

周仓身上應保留：陳述紙條、陳述工牌、孕婦照片、1 分與 3 分積分幣 / 相關硬幣線索、失效工牌、手機。右手或手背黑線到第一指節，心跳倒數代價仍有殘留。

## 第一章必須承接

- 第一卷最後畫面：休息艙、倒數三天、直播圖標仍亮。
- 陳述訊息：「你拿到了第三枚，做得不错。但现在不要打开那张照片。」
- 周仓不能立刻打開照片；若要觸碰照片，必須被系統、陳述訊息或副本規則阻止。
- 許小小 3 米限制必須出場，且不能無交代解除。
- 周仓不能把心跳倒數當成主動技能。
- 直播沒有結束，彈幕可以缺席，但觀看狀態要留下痕跡。

## 第一章要建立的新規則

新副本「記憶放映廳」的第一條可見規則應和座位 / 放映 / 回頭 / 字幕有關。建議第一章只展示部分規則，不要一次把全套講完。

建議首批規則：

1. 入場後請依票根入座，不可交換座位。
2. 放映開始後，請勿回頭查看觀眾席。
3. 字幕與畫面不一致時，以字幕為準。
4. 若畫面中出現本人，請勿確認其身份。

真正陷阱：字幕可能被污染；票根可能不是周仓的票根；「本人」的判定不是肉眼，而是系統身份欄。

## 第一章要種 / 推進的伏筆

- 票根上的名字 / 編號與周仓不一致，可能出現「陳述」或另一個被遮擋的名字。
- 觀眾席中有空位，但空位會在放映開始後被填滿。
- 字幕比畫面更早知道角色要說什麼。
- 許小小能記住周仓短暫忘掉的細節，但她不能解釋原因。
- 手背黑線在放映廳光線下顯得更深，像被投影光喚醒。

## 第一章禁止事項

- 不要直接解釋系統本質。
- 不要直接揭露陳述身份。
- 不要完整打開孕婦照片。
- 不要讓許小小變成普通助手或吐槽角色。
- 不要讓周仓主動使用心跳倒數。
- 不要讓彈幕大量解說；可以用少量延遲訊息製造不安。
- 不要把記憶放映廳寫成普通電影院；它應該像系統檔案室、審訊室與影院的混合體。

## 第一章建議結構

### 1. 休息艙餘波

周仓確認身上物品，看到直播圖標仍亮。倒數顯示三天，但某個數字跳動異常，暗示倒數可被提前。

### 2. 許小小的 3 米限制

周仓測試移動，許小小從不可見狀態被拉出或在螢幕反光裡出現。她不是自由同行，而是被系統綁定在他身邊。

### 3. 陳述訊息與照片

周仓摸到照片，系統或訊息提醒「現在不要打開」。照片可有輕微異常：邊角變熱、背面浮現座位號、或照片中的男人輪廓與休息艙屏幕反光重疊。

### 4. 進入記憶放映廳

倒數被切斷，休息艙燈光變成放映機光束。門不是打開，而是牆面變成銀幕。周仓與許小小被送入放映廳入口。

### 5. 票根鉤子

周仓拿到票根。票根上不是 `F-0217`，而是 `陈述`、或 `F-0217 / 陈述` 疊印。章末停在他找到座位，卻發現座位上已經有人坐著。
