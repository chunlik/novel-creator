# 配角体系

## 五种功能类型

| 类型 | 功能 | 对主角的作用 |
|------|------|-------------|
| **镜子型** | 反射主角——和主角形成对照 | 逼主角面对自己 |
| **诱惑型** | 提供一条危险但有吸引力的路 | 制造选择冲突 |
| **秩序型** | 代表制度/规则/理性 | 制造理念冲突 |
| **情感型** | 让主角有牵挂 | 提醒主角"为什么还想当人" |
| **主题型** | 承载一个主题侧面 | 让配角不只推动剧情，还推动主题 |

## 配角检查清单（每卷后审查）

每卷完成后，确认：

- [ ] 每个重要配角是否有自己的独立目的（不只是帮主角）
- [ ] 配角之间的关系是否有变化（不只是和主角的关系）
- [ ] 配角的对话是否有立场差异（不是每个人都讲作者想讲的话）
- [ ] 是否有配角成功推动了主角的成长（不是只在旁边喊加油）
- [ ] 是否有配角承载了一个主题侧面
- [ ] 是否有任何配角变成了工具人（需要升级或让ta退场）
