## 一、五作品核心對照

| 作品 | 副本類型 | 一句話鉤子 | 最大壓力 | 爽點類型 | 最值得你學 |
|------|---------|-----------|---------|---------|-----------|
| 《無限恐怖》 | 已知恐怖片世界 | 被丟進已知恐怖片，但知道劇情也未必能活 | 既定 IP 的不可抗力 | 情報差改變原劇情 | 讓便利店規則有「原本用途」與「反用法」 |
| 《地獄公寓》 | 血字任務 | 公寓給出血字指示，不完成就死 | 指示看似簡單，其實有坑 | 找出血字真正含義 | 系統提示不要全可信 |
| 《驚悚樂園》 | 遊戲劇本 | 主角進入恐怖遊戲劇本 | 劇本會騙玩家 | 反套路推理 | 周倉要有更鮮明解法風格 |
| 《死亡萬花筒》 | 門內世界 | 進入一扇門就是一個死亡規則世界 | 門內規則通常不完整 | 線索回收 + 情感合作 | 許小小隊友線增加情感 |
| 《全球高考》 | 考場副本 | 考試不是答題，是生死副本 | 制度本身就是敵人 | 解題反殺制度 | 收銀交易可以像考題層層變形 |

## 二、各作品技法詳解

### 作品一：《無限恐怖》— 情報差技法

**核心邏輯：** 讀者知道危險來自哪裡（咒怨的房子），但主角知道原劇情，不等於能活——因為副本會變。

#### 技法 1：給讀者情報優勢，但給主角規則盲區

| 項目 | 內容 |
|------|------|
| **原文怎麼做** | 讀者看過《咒怨》，知道進房子會死。主角也知道。但副本加了「主神任務」——完成特定條件才能活，光躲沒用。 |
| **我們怎麼用** | 讀者看到七條守則，和周倉同時讀到。讀者覺得「遵守就好」——但系統提示「請勿讓她確認你看得見她」是一條讀者和周倉都不知道真假的規則。情報優勢在於：讀者比周倉早一步懷疑這條規則。 |
| **反面案例** | ❌ 讓周倉一開始就懷疑所有規則——讀者沒有推理空間。❌ 讓系統直接解釋哪條規則是假的——情報差消失。 |
| **檢查點** | 本章是否給了讀者至少一條「周倉還沒想到，但讀者可以想到」的線索？ |

#### 技法 2：已知規則的「反用法」

| 項目 | 內容 |
|------|------|
| **原文怎麼做** | 主角利用《咒怨》原作設定（伽椰子只殺進入房子的人），反向把敵人引進房子讓鬼殺。 |
| **我們怎麼用** | 規則五說「不要讓紅雨衣人看見工牌」——周倉後來反過來用陳述工牌轉移紅雨衣人注意力。規則不是只能遵守，可以被利用。 |
| **反面案例** | ❌ 周倉用「系統漏洞」但沒付出任何代價——反用法必須有風險。❌ 反用法太複雜，讀者跟不上推理過程。 |
| **檢查點** | 周倉反用規則時，讀者能否從前文推理出他的邏輯？（不能是憑空靈感） |

#### 技法 3：隊友死亡作為代價展示

| 項目 | 內容 |
|------|------|
| **原文怎麼做** | 《無限恐怖》前期隊友大量死亡，不是為了嚇人——是讓讀者知道「規則真的會殺人」。 |
| **我們怎麼用** | 卷一不需要隊友死。但需要「上一任 F-0217」的死亡痕跡（監控殘影、冰櫃工牌、陳述紙條）讓讀者感受到：上一個人沒走出去。 |
| **反面案例** | ❌ 只靠角色口頭說「這裡死過人」——沒有實體證據。❌ 死太多人讓讀者麻木。 |
| **檢查點** | 本章是否有一處讓讀者「看見」而非「被告知」死亡曾經發生？ |

### 作品二：《地獄公寓》— 半真半假規則技法

**核心邏輯：** 血字指示永遠是真的（不遵守會死），但血字真正的「含義」從來不是表面那層。

#### 技法 4：規則字面為真，執行意圖為假

| 項目 | 內容 |
|------|------|
| **原文怎麼做** | 血字說「在指定時間去指定地點」——表面是任務，實際上是把人送到鬼面前。遵守是死，不遵守也是死。生路在於找出「血字沒說的部分」。 |
| **我們怎麼用** | 系統提示「請勿讓她確認你看得見她」——字面上是一條保護性提示。實際上可能是隔離規則：阻止周倉和許小小建立隊友關係。周倉遵守了整卷，卷末才發現：不遵守也沒有懲罰。 |
| **反面案例** | ❌ 讓規則「明顯有問題」（比如矛盾、錯字）——讀者太容易識破。❌ 所有規則都是假的——讀者失去推理基準。 |
| **檢查點** | 本章的系統提示 / 規則中，是否至少有一條的「真正目的」和「字面意思」不同？ |

#### 技法 5：限制本身就是線索

| 項目 | 內容 |
|------|------|
| **原文怎麼做** | 血字的限制條件（時間、地點、人數）本身就是推理線索。比如「必須三個人去」——不是刁難，是因為鬼只會攻擊第四個人。 |
| **我們怎麼用** | 「不要讓穿紅色雨衣的人看見你的工牌」→ 反推：紅雨衣人不是看臉，是看工牌。工牌才是被鎖定的關鍵，不是人。這條規則教你怎麼躲，同時也教你紅雨衣人的運作邏輯。 |
| **反面案例** | ❌ 限制條件純粹為了刁難主角而設——沒有任何可推理的資訊量。 |
| **檢查點** | 本章的規則 / 限制，讀者能否從中反推出至少一條關於「敵人 / 系統如何運作」的資訊？ |

### 作品三：《驚悚樂園》— 主角解法風格化技法

**核心邏輯：** 封不覺（主角）不是最強，但他用反常識、嘴炮、推理拆劇本的方式無人能複製。讀者爽的不是「他贏了」，是「他這樣也能贏」。

#### 技法 6：主角的解法有簽名

| 項目 | 內容 |
|------|------|
| **原文怎麼做** | 封不覺的每一次通關都有強烈個人印記——他會用恐怖遊戲的設計邏輯反推 GM 意圖，用嘴炮拖時間，用荒誕行為測試規則邊界。 |
| **我們怎麼用** | 周仓的簽名應該是「觀察 → 找規則縫隙 → 用系統自己的邏輯反打」。不是靠暴力，不是靠運氣，不是靠前人筆記——是他自己看出來的。比如他把買方從「孕婦」改成「未登記身份（附屬物）」——這就是簽名式解法。 |
| **反面案例** | ❌ 周仓的解法和其他無限流主角沒區別（靠運氣 / 靠筆記 / 靠隊友救）。❌ 解法風格搖擺不定——有時靠推理，有時靠直覺，有時靠蠻力。 |
| **檢查點** | 本章周仓做的關鍵決策，換成其他無限流主角會不會做出一樣的選擇？如果會——解法還不夠有簽名。 |

#### 技法 7：規則的「設計意圖」可以被反向閱讀

| 項目 | 內容 |
|------|------|
| **原文怎麼做** | 封不覺會想：「GM 設計這個劇本時，預期玩家怎麼做？」然後他故意不做那個。 |
| **我們怎麼用** | 周仓可以想：「系統設計這七條規則時，預期店員怎麼做？」——全部遵守。但他可以選擇性地利用規則，而不是被規則利用。 |
| **反面案例** | ❌ 周仓的「反向閱讀」變成作者直接告訴讀者答案（「他意識到系統想讓他…」）——必須讓讀者從行動中看到反向閱讀，不是從獨白中讀到。 |
| **檢查點** | 周仓是否在某個時刻做出了「系統預期之外」的選擇？這個選擇的推理鏈讀者能否追蹤？ |

### 作品四：《死亡萬花筒》— 隊友情感線技法

**核心邏輯：** 無限流不止是解謎通關。《死亡萬花筒》證明：讀者會為了「兩個人能不能一起走出去」而追完整本書。

#### 技法 8：隊友不是工具人——有獨立動機和限制

| 項目 | 內容 |
|------|------|
| **原文怎麼做** | 阮南燭不是林秋石的附屬品——他有自己的目標、判斷、和行動。兩人之間的信任是逐步建立的，不是一見面就組隊。 |
| **我們怎麼用** | 許小小不是周仓的「隨身助手」。她在卷一有自己的行動邏輯：她想被當成「人」。她把牛奶盒推到周仓面前——不是因為她在幫他，是因為她需要店員的動作才能完成交易。她幫周仓通關的同時也在完成自己的交易。 |
| **反面案例** | ❌ 許小小變成「周仓需要什麼她就提供什麼」的工具人。❌ 許小小主動解釋世界觀、給攻略——她不是嚮導。❌ 兩人的信任來得太快——卷一應該保留距離感。 |
| **檢查點** | 本章許小小（或其他配角）的行動，能否用「她有自己的目的」來解釋？如果只能用「她在幫主角」來解釋——她已經是工具人了。 |

#### 技法 9：關係遞進靠「共同經歷」，不靠「對話交代」

| 項目 | 內容 |
|------|------|
| **原文怎麼做** | 林秋石和阮南燭的關係不是靠「我信任你」這種對話建立的——是靠一起過門、一起面對死亡、一起做選擇建立起來的。 |
| **我們怎麼用** | 周仓和許小小的關係遞進節奏應該是：交易對象 → 系統標記的不可接觸者 → 被迫一起完成交易鏈 → 周仓選擇打破隔離規則 → 潛在隊友。每一步都有對應的事件，不是對話。 |
| **反面案例** | ❌ 許小小說「你是我第一個願意相信的人」——關係靠台詞交代。❌ 周仓內心獨白「他發現自己開始信任她」——關係靠獨白交代。 |
| **檢查點** | 本章是否能刪掉所有關於「兩人關係」的台詞和獨白，而讀者仍能從行動中感受到關係變化？ |

### 作品五：《全球高考》— 日常制度恐怖化技法

**核心邏輯：** 最恐怖的不是怪物——是你每天都在做的事，突然變成了生死考題。

#### 技法 10：日常動作變成規則觸發點

| 項目 | 內容 |
|------|------|
| **原文怎麼做** | 考試——每個人從小做到大的事——變成會死人的規則場。答題、交卷、被監考盯著看——全是日常，但全部致命。 |
| **我們怎麼用** | 便利店收銀——掃碼、收錢、找零、退貨——全是日常動作。但每個動作都是一筆「交易」，每次交易都可能改變積分、觸發考核、改變玩家身份。掃碼不是掃碼，是「系統確認這筆交易有效」。 |
| **反面案例** | ❌ 把交易寫成「周仓按下確認鍵，系統發出提示音」——沒有儀式感。❌ 交易太順利——沒有讓日常動作產生緊張感。 |
| **檢查點** | 本章是否至少有一個日常動作（掃碼 / 收錢 / 開抽屜 / 看監控），讓讀者感受到「這不只是普通動作——這是在遵守或違反一條規則」？ |

#### 技法 11：制度本身是敵人，不是執行者

| 項目 | 內容 |
|------|------|
| **原文怎麼做** | 《全球高考》的恐怖不在監考官——在考場系統本身。規則、扣分機制、答題邏輯——這些「制度」才是真正的壓迫者。 |
| **我們怎麼用** | 便利店副本的恐怖不在紅雨衣人或店長——在「交易系統」本身。系統定義什麼是有效交易、什麼不算、誰是買方、誰是賣方。店長只是管理權限，系統才是規則本身。 |
| **反面案例** | ❌ 把店長寫成終極反派——他只是系統的一個權限層級。❌ 讓周仓「打敗」某個具體敵人就能通關——真正要面對的是系統邏輯。 |
| **檢查點** | 本章的威脅來源，讀者能否感受到「不是某個人要害他——是這個地方的規則本身會吃人」？ |

#### 技法 12：交易 / 考題的層層變形

| 項目 | 內容 |
|------|------|
| **原文怎麼做** | 《全球高考》的考題會變形——第一題是選擇題，第二題變成填空，第三題變成實作。同樣是「考試」，形式不斷翻新。 |
| **我們怎麼用** | 便利店的三筆交易不是三筆同樣的買賣——第一筆是銷售（牛奶），第二筆是退貨（異常空盒），第三筆是回收（空瓶押金）。全都是收銀動作，但每一筆的系統邏輯不同。讀者看完卷末才意識到：三筆交易剛好涵蓋了便利店的完整收銀循環。 |
| **反面案例** | ❌ 三筆交易都是「顧客拿東西→周仓收錢」——沒有變形，讀者覺得重複。❌ 變形太刻意——讀者覺得作者在炫技。 |
| **檢查點** | 本章的交易 / 事件，和前章相比是否有「規則層面的變化」而不只是「換了一個人 / 換了一個物品」？ |

## 三、技法速查：按場景選用

| 你的場景 | 優先使用技法 |
|----------|------------|
| 系統給出新提示 / 規則 | #4 半真半假 → #5 限制是線索 → #11 制度是敵人 |
| 周仓做出關鍵決策 | #6 解法有簽名 → #7 反向閱讀設計意圖 |
| 周仓發現 / 利用規則漏洞 | #2 反用法 → #7 反向閱讀 → #10 日常動作觸發 |
| 許小小 / 配角出場 | #8 獨立動機 → #9 關係靠行動 |
| 展示副本死亡威脅 | #3 死亡痕跡 → #11 制度吃人 |
| 交易 / 收銀場景 | #10 日常恐怖化 → #12 層層變形 |
| 章末鉤子 / 反轉 | #1 情報差 → #4 表面為真意圖為假 |
| 讀者已知道危險但周仓不知 | #1 情報差 |

## 四、每章自查（從五作品角度）

寫完一章後，快速掃這 5 題：

1. **情報差：** 讀者是否比周仓多知道或早懷疑某件事？（《無限恐怖》）
2. **規則真假：** 本章的系統提示中，是否有至少一條的「真正目的」和「字面不同」？（《地獄公寓》）
3. **解法簽名：** 周仓的關鍵選擇，換成其他無限流主角會不會做一樣的事？（《驚悚樂園》）
4. **隊友動機：** 許小小（或其他配角）的行動，能否用「她有自己的目的」解釋？（《死亡萬花筒》）
5. **制度恐怖：** 是否有一個日常動作讓讀者感受到「這不是普通動作——這是規則場」？（《全球高考》）
