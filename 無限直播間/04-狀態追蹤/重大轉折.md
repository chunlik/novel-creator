# 重大转折追踪

## 总纲：内部危机四阶段（全书骨架）

本书不是靠外部事件分段——是靠周仓对自己认知的变化来分。

每个阶段可以容纳多卷。阶段内的卷共享同一个「周仓的认知状态」，但每卷仍需要推进主线一步。

```
阶段0             阶段1               阶段2               阶段3
「我不知道        「我在追            「名字可能          「我接受了」
 少了什么」        陈述——但              是我」
                   为什么我
                   做的事和他
                   一模一样？」
   卷1-3            卷4-15              卷16-22             卷23-25
   (3卷·入门)       (12卷·追查+重复)    (7卷·面对)          (3卷·结局)
```

可扩展弹性：每个阶段实际卷数可根据写作节奏自由增删。阶段1写顺了加2卷完全没问题——只要每卷推进主线或内部危机一小步。

## 阶段0：我不知道少了什么（卷1-3？）

| 项目 | 内容 |
|------|------|
| 当前状态 | ⏳ 已完成卷1；卷2-3待写 |
| 阶段主题 | 进入系统，发现规则，生存下来。还不知道自己不正常 |
| 周仓的认知 | 他只觉得自己比别人冷静一点。认为自己追纸条上的名字只是因为好奇 |
| 阶段功能 | 让读者熟悉世界规则 + 建立驱动力（纸条） + 埋下"他不太对劲"的种子 |

| 本阶段终点 | 周仓通关了几个副本，找到了陈述的部分线索。但他开始注意到：自己和陈述的行为模式有奇怪的相似。这不是他在学陈述——是身体自己在做。 |

## 阶段1：我在追陈述——但为什么我做的事和他一模一样？（卷4-15）

| 项目 | 内容 |
|------|------|
| 当前状态 | ❌ 未触及 |
| 阶段主题 | 追查 + 发现重复 + 不愿承认 |
| 周仓的认知 | 他表面上还在找"陈述在哪"，但越来越多证据显示他在重复陈述的路。他開始注意到但不願意深想 |
| 阶段功能 | 长线追查阶段。跨多个副本，每卷释放一条关于陈述/系统/抹消的线索。同时让周仓和读者一起累积"他越来越像陈述"的不安感 |
| 阶段内卷数 | 约12卷。包含多种副本类型（病栋/组织遗迹/多人副本/座标线回访） |
| 本阶段终点 | 周仓无法再否认：他不是在追陈述——他是走在同一条路上。纸条上的名字不是陈述，可能是他自己 |

**阶段1·必须铺设的转折点（写卷时填充具体位置）：**

| 转折 | 内容 | 周仓的变化 |
|------|------|-----------|
| 转1 笔记会说话 | 笔记蓝色字来自碎片集体意识 | 陈述不是全知前辈——只是转录者 |
| 转2 系统会骗人 | 规则8是假的后，发现其他副本也有假规则 | 规则不可全信——你活着不靠规则靠运气？ |
| 转3 镜象确认 | 身体自然地做出和陈述一模一样的选择 | 我不是在学他——我就是同一种人 |
| 转4 陈述的结局 | 确认陈述被抹消/被同化 | 我也会这样吗？ |
| 转5 座标线回访 | 回到之前经过的副本，发现它变了/被黑水吃了 | 通关不是结束——副本还在长 |
| 转6 遇到其他主播 | 见到和陈述/周仓同类型的人，或者相反类型的人 | 系统在量产同一种通关者？ |
| 转7 旧记录的发现 | 组织留下的文档，发现"纸条名单" | 纸条不是写给周仓的——是一份名单。上面有陈述也有周仓 |
| 转8 碎片对话 | 周仓开始也能听到碎片的声音（不只是陈述的笔记） | 他開始擁有和陳述一樣的"體質"——這不是巧合 |

（阶段1的转捩点具体落在哪几卷、哪一章，写该卷时决定。只需要确保：每2-3卷至少触发一个转折。）

## 阶段2：名字可能是我（卷16-22）

| 项目 | 内容 |
|------|------|
| 当前状态 | ❌ 未触及 |
| 阶段主题 | 面对真相 + 我是谁 + 归零逼近 |
| 周仓的认知 | 他知道了纸条上的名字是自己的另一种可能。他必须面对：自己是一个"消失的人"的重置版/替代品 |
| 阶段功能 | 剥最后一层。不再追查外部线索——每卷只做一件事：剥开一层对自己的认知 |
| 阶段内卷数 | 约7卷。短而密——不需要再建立新副本，直接进入核心 |
| 本阶段终点 | 周仓理解了全部真相。他知道自己能选择什么：通下去（被同化）vs 停下来（不接受但也不妥协） |

## 阶段3：我接受了（卷23-25·结局）

| 项目 | 内容 |
|------|------|
| 当前状态 | ❌ 未触及 |
| 阶段主题 | 选择 + 接受 + 活下去 |
| 周仓的认知 | 他找到了答案。他没有变回正常人——他接受了"不正常就是我的正常" |
| 阶段功能 | 不是战斗——是选择。三卷分别对应：看清真相 → 做出选择 → 接受后果 |
| 阶段内卷数 | 3卷（结局三卷曲） |
| 结局画面 | 周仓没有离开系统，也没有变成店长。他停在了一个边缘位置——不再通关，但也不消失。他找到了一个让"不正常"的自己活下去的方式 |

## 全书结构一览（25卷版）

| 阶段 | 卷 | 功能 | 每卷节奏 |
|------|----|------|---------|
| 阶段0 | 卷1-3 | 入门三部曲——建立世界+驱动力+埋下"不对劲"种子 | 每卷1个副本+内部危机微小推进 |
| 阶段1 | 卷4-15 | 长线追查——12卷跨多个副本，每2-3卷一转折 | 正常副本节奏，每卷释放一条主线索 |
| 阶段2 | 卷16-22 | 短而密的自我面对——不再建立新副本 | 节奏加快，每卷一剥离 |
| 阶段3 | 卷23-25 | 结局三卷曲 | 节奏最密——选择+接受+后果 |

总量 25 卷，每卷以 4-5 万字估算，总字数在 100-130 万区间。弹性——阶段1可加，阶段2可加，不鎖死。

## 每卷最低要求（100万字不散的底線）

每卷必须做到以下三件事的**至少一件**：

- [ ] 推进主線謎團一步（见謎團管理.md）
- [ ] 推进內部危機一步（周仓对自己认知的变化）
- [ ] 建立一个新的、后续会回收的伏笔

如果一卷写完后，读者回顾时发现：主线没动、周仓没变、也没有新伏笔——这卷就是多余的。
