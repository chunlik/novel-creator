# 規則連續性

> 記錄各副本規則、規則適用範圍、已知變化、不可跨副本誤用的限制。  
> **原則：** 副本規則預設只在該副本有效。除非正文或系統明確說明，不能把便利店規則直接帶到下一副本。

## 跨副本可延續的規則型概念

### 1. 名字 / 身份危險

**來源：** 卷1·Ch4+Ch5  
**可延續：** 是  
**用法：** 第二卷可以把名字、票根、座位號、字幕身份欄、本人確認等設計成危險點。  
**限制：** 不可直接說「知道真名就會死」，除非新副本規則正式建立。

### 2. 假規則 / 隔離規則

**來源：** 卷1·Ch1+Ch2，「請勿讓她確認你看得見她」  
**可延續：** 是  
**用法：** 第二卷可以揭示某些系統提示不一定是保護主播，而是為了隔離潛在隊友、阻止關係建立、讓主播只把對方當作「交易對象」或「未確認身份」。  
**限制：** 不可直接說所有規則都是假的。假規則必須有可回推證據，例如：遵守後讓周仓失去盟友資訊；違反後沒有懲罰；許小小能用行動證明她不是敵人；彈幕或陳述留下反向提示。

### 3. 可見性 / 身份確認危險變體

**來源：** 卷1·Ch1+Ch2 的表層提示  
**可延續：** 可以作陷阱或變體  
**用法：** 第二卷仍可設計「不要確認銀幕中的自己」「不要讓觀眾席確認你聽得見」等規則，但必須讓讀者不確定它是真規則還是假規則。  
**限制：** 不可再把「周仓不能承認看得見許小小」當成硬限制。許小小預定為卷2隊友，周仓可以逐步直接與她互動，但最好在正文中先讓他意識到第一副本提示可能是誘導。

### 4. 規則文字不可靠

**來源：** 規則六污染、結算記錄「規則污染 ×1」、假規則候選  
**可延續：** 是  
**用法：** 放映廳字幕、票根、銀幕規則可被污染、替換，或故意用半真半假的方式誘導。  
**限制：** 不可讓所有規則都任意改變；每次改變需有可觀察痕跡或後續代價。

### 5. 交易不只買賣

**來源：** 牛奶銷售 / 退貨 / 回收、黑傘特殊交易  
**可延續：** 是  
**用法：** 第二卷可以把回答問題、提交片名、確認身份寫成「交易 / 補錄 / 歸檔」。  
**限制：** 必須有系統欄位或結算提示支撐。

### 6. 觀察比解釋重要

**來源：** 周仓以燈光、黑水、工牌、買方欄位推理  
**可延續：** 是  
**用法：** 第二卷應讓周仓透過字幕延遲、觀眾席反應、票根細節推理。  
**限制：** 不可讓 AI 直接旁白解釋真相。

## 不可跨副本直接使用的規則

- 不可把「黑水不能進燈光」直接套到記憶放映廳，除非放映廳重新建立光線規則。
- 不可讓紅雨衣人在第二卷繼續按 15 分鐘巡邏，除非卷2重新建立巡邏機制。
- 不可讓關東煮機器、後門、便利店貨架規則直接出現在卷2，除非作為放映畫面或回放記憶。
- 不可讓便利店收銀機交易流程完整複製到放映廳；放映廳應有自己的提交 / 問答 / 歸檔機制。
- 不可把「請勿讓她確認你看得見她」當成卷2硬限制；它現在是可揭示的假規則 / 隔離規則候選。
- 不可為了讓許小小成為隊友，就無證據否定所有規則；只能先否定這一條或揭示它的另一層目的。

## 第二卷寫作檢查

寫卷2時，每章至少檢查：

- 本章用到的規則是否已在正文呈現？
- 是否把便利店規則錯套到放映廳？
- 是否讓規則任意改變但沒有可觀察痕跡？
- 是否讓系統解釋太多？
- 是否把 `planning` 規則寫成讀者早已知道？
- 是否把 `pending_canon` 規則寫成已發生？
- 是否把所有規則都寫成假的？這會削弱規則怪談張力，禁止。
- 若許小小成為隊友，是否有場景證據證明「請勿讓她確認你看得見她」是誘導 / 隔離規則，而不是無理由推翻？

## pending_canon

| 規則概念 | 狀態 | 落地方式 |
|----------|------|----------|
| 規則可被物理帶走 | pending_canon | 可在記憶放映廳中透過「被剪掉的一條規則」「被拿走的字幕膠片」正式落地 |

| 手背黑線 | pending_canon | 若保留，需作為周仓首次明確發現的身體規則落地 |
