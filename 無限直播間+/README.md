# 無限直播間+

> 本資料夾為《無限直播間》的新版設定總庫。
>
> 目的：在正式重寫／續寫正文前，先建立一份可長期維護的詳細設定聖經。

## 使用原則

1. **先定設定，再拆 Part，再拆卷／副本。**
2. 目前不以「卷」作為最高結構，而以 **Part／大篇章** 管理全書階段。
3. 所有已確認設定放入 `01-設定工坊/`。
4. 尚未確認內容會明確標記為「未決」。
5. 正文寫作前，優先查閱：
   - `01-設定工坊/00-設定索引.md`
   - `01-設定工坊/01-核心決策總覽.md`
   - `01-設定工坊/04-未登記主播篇設定.md`
