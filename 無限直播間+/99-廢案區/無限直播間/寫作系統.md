### 本书的核心问题

> 一个被规则吞噬的世界里，通关的人都会变成规则的一部分。而一个靠观察活下来、但不知道自己少了什么的人——他能不能在消失之前，找到那个让他还是「人」的东西？

> 这个世界的规则，照做就能活。但所有真正懂规则的人，最后都消失了——而我开始明白他们为什么消失。

## 2. 核心设定（8维）

| 维度 | 内容 |
|------|------|
| 主角定位 | 一个靠观察活下来的人 |
| 驱动力 | 手里有一张纸条，上面只有一个名字。系统查不到这个人，但每个副本角落都出现同一个标记 |
| 主题天花板 | 规则本身才是怪物。通关不等于胜利——而是被系统同化 |
| 单元结构 | 一卷≈一个副本（短副本可压缩） |
| 基调 | 中等冷度（主角会推理，但系统本质留白） |
| 现实世界 | 无回家，只有过渡空间（结算页面/等待室） |
| 叙事视角 | 第三人称近距离（全程跟随周仓） |

## 3. 写作规则

- 连续不超过2句以"他"开头，平均每4-5句出现1次"他"
- 周仓（全名）和"他"交替使用
- 替代方案：环境开句→动作链串联→对话打断→配角独立出场→物品开句
- 每个动作必须有所属主语，不出现悬浮动作
- 引号使用""（大陆规范）
- 异常不解释，让读者自己推理
- 细节代替心理描写
- **知识边界自洽**: 角色的结论不能超过他当前拥有的信息量。不能因为"读者知道这是系统"就默认角色也知道。每个推理/猜测必须有对应的事件或观察作为依据。角色不知道的事，他不会去想。特别检查：主角在第一章有没有做出需要后期信息才能成立的推断
- **因果链验证**: 每个"所以"（角色的推断）必须追溯到一个"因为"（角色亲眼看到或亲耳听到的事实）。推断的层级不能超过事实一步。例如：看到规则奇怪→"这不是正常员工守则"✓（一步推断）；看到规则奇怪→"规则能让人消失"✗（两步跳跃，中间缺少"规则被违背后有惩罚"的观察）
- **角色身份一致性**: 角色行为必须符合他的背景设定。不要因为作者需要推进剧情就写出角色不会做的事。写之前先问：以这个角色的经验/技能/性格，他这时候会怎么做？例如：周仓是便利店打工的，他应该自然地去扫条码查价格，而不是问顾客"多少钱"——那是生客才会做的事
- **对话逻辑自洽**: 角色说的每一句话，读者都能理解"他为什么这么说"。不要在对话中使用模糊指代（"她""它""那个"）来制造悬念——读者困惑和读者好奇是两回事。角色提到某个关系（欠/付/交易）之前，这个关系必须在上下文中已被建立或对话本身能说清楚。检查：把这段话单独拿出来读，不依赖上下文，能不能看懂谁在跟谁做什么交易？
- **情感缺口可视化（反機器人規則）**: 周仓的異常冷靜不是「沒有感覺」，而是「感覺到了但不理解那是什麼」。他每次遇到應該有情緒反應的事件時，必須在描述中讓讀者看到他知道自己應該有反應但沒有——例如：「他等了一下，想知道自己會不會難過。沒有。他知道這不正常，但他拿不出東西來填那個位置。」情感缺口的感受不是「角色很冷」，是「角色知道自己不該這麼冷，但他沒辦法變熱」。每卷至少出現 2-3 次此類自覺。
- **禁止空動作重複**: 同一個物品同一章內不重複觸碰/確認，除非以下至少一項成立：A) 物品狀態改變了；B) 角色發現了之前沒注意的新細節；C) 角色對物品的理解被顛覆了。上述條件都不滿足時——直接跳過。讀者記得物品在哪，不需要每章重新摸一遍口袋。

## 4. 角色矛盾一览

### 周仓

| 维度 | 内容 |
|------|------|
| 表面形象 | 边缘人型。独来独往，不主动交流，被问话时会礼貌回答 |
| 隐藏异常 | A. 对危险的反应是"麻烦"而非"恐惧"；B. 观察模式自动开启，控制不了开关；C（隐藏）断片；D. 知道自己没感觉——每次应该难过/害怕/愤怒的时候，他会停下来等，等不到，然后走开。读者看到的不是冷漠，是他对自己冷漠的困惑 |
| 内在伤口 | 不记得伤口在哪里。知道少了什么，情感有缺口，但不记得为什么 |
| 想要 vs 需要 | 想要：找到纸条上那个人 / 需要：接受那人可能就是自己 |

### 许小小

| 维度 | 内容 |
|------|------|
| 本质 | X被抹消后留下的情感碎片 |
| 记忆免疫 | 没有自己的记忆可以被偷 |
| 记得陈述 | X残留的情感烙印，不是她自己的记忆 |
| 本質感知 | 能直接看到人的情绪——因为她是"情感的碎片" |
| 想要 vs 需要 | 表层：想知道进入便利店前的身份 / 深层：填补心里的空洞 |

### 陈述

| 维度 | 内容 |
|------|------|
| 表面形象 | 偏执型——用笔记和准备来压制恐惧 |
| 隐藏异常 | 写过的笔记会自己修改。他不是作者——他是转录者 |
| 内在伤口 | X被抹消，他没能保护她。所有后续行动都是在赎罪 |
| 想要 vs 需要 | 想要：把X捞回来 / 需要：接受X已经不在了 |

### X

| 维度 | 内容 |
|------|------|
| 本质 | 陈述从便利店带出的NPC，和陈述相恋，在后续副本中被抹消 |
| 遗留 | 情感碎片变成许小小；抹消时有一部分沉到了黑水里 |
| 对陈述的意义 | 驱动他走向深渊的人 |
| 对周仓的意义 | 许小小的本体 + 证明抹消机制不完美 |

## 10. 高潮设计（每卷路径）

### 通用原则
- **行动高潮**：读得见的事件
- **情感高潮**：主角面对最怕的事
- **真相高潮**：谜团部分揭露
- **价值观高潮**：主角要回答"我要成为什么样的人"
- **代價**：不可逆。主角赢了，但他也永远改变了

### 卷级路径

| 卷次 | 行动高潮 | 情感高潮 | 真相高潮 | 价值观高潮 | 代價方向 |
|------|---------|---------|---------|-----------|---------|
| 卷1 | 阻止交易+通关 | 选择帮助许小小 | 店长＝被同化者 | 庇护他人＞自保 | 轻（托管+关联关系） |
| 卷2 | 关掉放映机 | 面对银幕上的自己——被偷走的那部分 | 规则8是假的+笔记碎片来源 | 相信自己的观察 vs 相信笔记 | 损失记忆/许小小差点被拷贝 |
| 卷3 | 待定 | 发现自己正在变成陈述 | 系统有意识+组织被反噬 | 继续通关 vs 停下来 | 不可逆的代價值叠 |
| 卷4+ | 待定 | 面对"纸条可能是自己" | 归零与黑水的本质 | 接受自己的真实身份 | 失去某个重要的部分 |

### 卷2·记忆放映厅 climax 结构（写作用参考）

```
Ch5-6 冲突攀升 → Ch7 真相揭露 → Ch8 高潮+结算

行动：找到放映机钥匙→不被规则8骗→手动关掉放映机
情感：银幕上的周仓在对他说话——被偷走的记忆在播放。
      他看到自己不记得的过去。选择：继续看（找记忆但被拖住）
      还是转头不看（损失信息）
真相：规则8是假的 + 蓝色笔迹来自碎片集体意识 + 陈述的最后一笔是恐惧中写的
价值观：相信什么？笔记？银幕上的自己？还是自己的判断？

      周仓选择牺牲哪段记忆
```

## 12. 真相分层揭露体系

### 五层框架

| 层次 | 内容 | 功能 | 揭露时机 |
|------|------|------|---------|
| L1 异常存在 | 世界不正常——副本、规则、抹消、黑水 | 建立惊奇感 | 卷1展示 |
| L2 异乡有规则 | 异常有逻辑——规则三层、代价机制、道具分级 | 可推理 | 卷1展示+后续深化 |
| L3 有人操控 | 组织创造→被反噬→系统有自我意识 | 进入阴谋线 | 卷3-4 |
| L4 主角和真相有关 | 纸条名字、陈述镜像、冷静的真相 | 增加故事重量 | 卷2-4逐步释放 |
| L5 主题答案 | 通关＝被同化？正常和异常的边界？ | 完成主题 | 结尾 |

### 揭露方式规则（禁止说明书式揭露）

| ✅ 正确方式 | ❌ 错误方式 |
|-----------|-----------|
| 用事件逼出真相 | 安排角色坐下来讲十章 |
| 用角色背叛揭露 | 突然丢出新名词 |
| 用主角失控揭露 | 和情感无关的真相 |
| 用灾难证明真相 | 揭露后剧情毫无变化 |
| 用选择迫使主角面对 | 真相只靠对话传递 |

### 每卷自查

每卷写完后，问：
- 这卷结束时，读者对前面某件事的理解是否被颠覆了？
- 如果没有——该卷的真相揭露不够。

具体揭露路径（每卷的"怎么揭露"）见 `04-狀態追蹤/謎團管理.md → 卷级线索分布`

## 13. 主题体系

### 核心主题（一句话）

> **通关的代价不是死——是变成自己不认识的东西。**

这不是结论，是全书中反复提出的问题。不需要在文本中直接说出口——通过剧情和角色选择让读者感受到即可。

### 主题支柱（三个侧面）

| 侧面 | 对应设定 | 在故事中的表现 |
|------|---------|--------------|
| **被同化** | 店长 = 被消化完的通关者 | 每个副本的管理员都曾是活人。通关次数越多，越接近那个位置 |
| **消失不等于没有** | 抹消机制 + 碎片集体意识 | 陈述消失了，但他的工牌还在、笔记还在、碎片还在跟周仓说话。许小小本身就是一个"没被删干净的人" |
| **我正在变成什么东西** | 周仓的内部危机 + 陈述镜象 | 周仓发现自己在重复陈述的行为模式。他追的那个人可能就是他将来的样子 |

### 主题如何藏在每层冲突里

| 冲突层 | 主题体现 |
|--------|---------|
| 外部冲突（通关 vs 消失） | 系统在跟你做交易：通关给你活路，但你在过程中交出了记忆/规则/人性 |
| 内部冲突（追名字 vs 接受名字可能是自己） | 周仓最想找到的答案，可能会证明他不再是自己 |
| 终极冲突（通下去 vs 停下来） | 真正的选择不是"能不能活"——而是"如果我活下来的代价是变成怪物，我还活吗" |

### 主题和主角命运的绑定

周仓每一卷都在被追问同一个主题的不同侧面：

| 卷 | 主题拷问 | 怎么呈现 |
|----|---------|---------|
| 卷1 | 规则是用来保护你的，还是用来吃掉你的？ | 便利店规则=交易逻辑。遵守规则的人最后都变成了交易的一部分 |
| 卷2 | 你相信的东西——规则、笔记、银幕——会不会骗你？ | 规则8是假的。笔记的蓝色字来自陌生人。银幕上的"自己"是被偷走的部分 |
| 卷3 | 你在变成另一个人吗？ | 发现和陈述的相似不是巧合——可能是同一种淘汰模式的重复 |
| 卷4 | 如果名字是你自己，你还继续找吗？ | 纸条意义的真相揭露。周仓面临选择 |
| 卷5+ | 通关的尽头是毕业，还是被消化？ | 归零逼近 + 系统的终极真相。周仓要回答通不通下去 |

### 主题和世界观的支撑关系

| 世界观元素 | 主题功能 |
|-----------|---------|
| 规则三层结构 | 表层的规则是"安全的假象"，真正的规则是"吃掉你的机制" |
| 抹消机制 | 系统的消化方式——你不是死了，是被擦掉了 |
| 碎片集体意识 | 被吃的人没有真正消失——他们在底部活着，但已经不完整了 |
| 心跳倒数 | 能力越用越靠近失去自己——代價本身就是主题的物理化 |
| 直播间/观众 | "观众"是被吃的人在看新的人走同样的路。你通关后也会坐在那个位置 |
| 过渡空间 | 没有窗户的房间+破旧的大厅——你不是在休息，你是在"等着被吃下一口" |

### 主题不靠对白，靠选择

```
❌ 错误写法：
周仓说："我担心自己正在变成怪物。"

✅ 正确写法：
周仓发现自己操作收银机的手法和陈述笔记里描述的一模一样。
他甚至没有学——手自己就动了。
```

### 每卷自查

每卷写完后，问：
- 这卷有没有让周仓面对"我在变成不是我"的问题？
- 这个面对有没有通过事件/选择/冲突来表现，而不是通过对话？
- 读者看完这卷后，会不会想"如果是我，我会不会也不知不觉变了？"

## 14. 写卷流程（标准步骤）

### 写卷前
- [ ] 读 寫作系統.md（全规则复习）
- [ ] 读 重大轉折.md（确认当前阶段+本卷功能）
- [ ] 读 謎團管理.md（确认本卷要释放的谜团层数）
- [ ] 读 配角體系.md（确认哪些配角会出场+他们的独立目的）
- [ ] 读 非線性時間記錄.md（如涉及跨时间事件）
- [ ] 写 该卷的大纲（章节结构+每章功能）

### 写卷中
- [ ] 每章结束时检查：这章改变了什么？（位置/信息/关系/危机程度）
- [ ] 章末留有悬念或新疑问

### 写卷后
- [ ] 该卷是否推进主线谜团/内部危机/新伏笔至少一件？
- [ ] 该卷结束时读者对前面某件事的理解是否改变了？
- [ ] 更新 謎團管理.md（释放层数+新问题）
- [ ] 更新 重大轉折.md（阶段进度+转折触发状态）
- [ ] 更新 非線性時間記錄.md（如有跨时间事件發生）

- [ ] 更新 時間線.md
- [ ] 更新 道具系統.md
- [ ] 更新 配角體系.md（关系变化）
- [ ] 检查：能力代价连续性？（前文的颤抖/耳鸣后文有体现吗）
