# 03-角色長線：周倉、許小小、林█、陳述、X

## 1. 周倉

### 目前定位

周倉是主角，也是 F-0217 的現任持有者。

但 F-0217 不應被理解為單純個人編號，而更像一個功能位／位置／責任鏈。

### 已定方向

- 周倉在第一 Part 進入直播間。
- 他在第一 Part 被許小小救過。
- 他與許小小留下未完成交易／未結清歸屬。
- 他後續將因追查陳述與干涉抹消，觸發自身抹消程序。
- 他會成為「未登記主播」。
- 終局方向上，他不是摧毀直播間，而是改造觀看與通關規則。

### 核心變化

周倉的主線不是「升級變強」，而是：

1. 從求生者變成規則破解者。
2. 從主播變成未登記主播。
3. 從被觀看者變成能反向利用觀看的人。
4. 從想通關，變成質疑通關本身。
5. 最後嘗試改寫直播間規則。

## 2. 許小小

### 目前定位

許小小不是普通 NPC，也不是第一 Part 結尾直接同行的隊友。

**已定：第一 Part 中，她不離開便利店。**

但她必須在第一 Part 中救過周倉，並與周倉建立真實連結。

### 第一 Part 狀態

- 她留在便利店。
- 她與周倉之間留下未完成交易／未結清歸屬。
- 她不是周倉的助手。
- 她是周倉欠下的一筆帳，也是後續回訪便利店的重要理由。

### 禁用設定

- 不使用 3 米限制。
- 不在第一 Part 結尾讓許小小直接離開便利店。
- 不把許小小寫成普通隊友或吐槽助手。

### 長線狀態

許小小後期會與林█、X、黑水底層、未登記主播篇產生關聯。

她不是 X 本人，但她來自 X 被抹消後留下的情感殘留之一。

### 核心恐懼

> 「不要把我還回去。」

一開始讀者可能以為這句話指：

- 不要把她還回便利店。
- 不要把她還給媽媽／孕婦。
- 不要把她還給黑水。

後期真相是：

> 她害怕被還原成本體 X 的一部分，從此不再是「許小小」。

## 3. 林█

### 目前定位

林█不是單純新角色。

**已定：林█後期逐步揭示為許小小的未來／重組形態。**

但她不完全等於第一 Part 的小女孩許小小。

### 暫定定義

林█是許小小經過以下因素後形成的存在：

- 非線性時間痕跡。
- 記憶殘留。
- 抹消修補。
- X 的部分殘留。
- 觀眾側／黑水底層的影響。

她可以被理解為：

> 許小小未來可能成為的某種倖存版本／重組人格／時間錨點殘影。

### 使用節奏

| 階段 | 林█呈現方式 |
|---|---|
| 前期 | 刻字、缺字署名、座位痕跡、票根、短句 |
| 中期 | 間接幫助周倉，但不承認身份 |
| 後期 | 開始與許小小、X 的關係浮出水面 |
| 真相期 | 揭示她不完全是許小小，也不完全是 X |

### 可用台詞方向

- 「我不是她。至少，不全是。」
- 「你以為救她，就是把她帶回原來的地方？」
- 「不要把她還回去。」

## 4. 陳述

### 目前定位

陳述是前任／舊路／鏡像角色。

他曾經走過一條與周倉相似的路。

### 已定方向

- 陳述曾帶出過一個關聯實體 X。
- X 之於陳述，類似許小小之於周倉。
- 陳述相信副本中的人可以被帶出去。
- 後來 X 被系統抹消。
- 陳述開始研究黑水、抹消、時間錨點、筆記與未完成交易。
- 陳述留下線索，不只是為了逃生，而是為了證明被系統刪掉的人可能被找回。

### 陳述失敗原因

已定：D，B + C 混合。

即：

- 陳述確實犯了錯。
- 他太想救回 X，忽略了 X 已經不再是原本那個人。
- 他想把 X 還原成本體，卻差點抹掉 X 後來長出的獨立人格。
- 系統利用他的錯，誘導他走向「回收本體」而不是「保留新人格」。

這是周倉後期必須避開的錯誤。

## 5. X

### 目前定位

X 是陳述曾經帶出的關聯實體。

她不是許小小，但許小小來自她被抹消後的情感殘留之一。

### 已定方向

- X 原本是某個副本中的 NPC／關聯實體。
- 陳述曾成功把 X 帶出副本。
- X 後來被系統抹消。
- X 沒有完全消失，一部分沉入黑水底層。
- 她的殘留與其他「等人來接的小孩／被遺棄者／被帶出又失敗的 NPC」碎片混合。
- 許小小由此形成。

### X 與許小小的關係

許小小不是完整 X。

許小小是從 X 的殘留中長出來的獨立人格之一。

因此後期救人困境是：

> 找回 X，是否會吞掉許小小？

周倉最終要學會：

> 救回許小小，不等於把她還原成 X。

---

## 6. 五人關係總覽

| 角色 | 核心功能 |
|---|---|
| 周倉 | 現任 F-0217，未來未登記主播，改造直播間的人 |
| 許小小 | 第一 Part 留在便利店的情感債，後期被揭示為 X 的殘留人格之一 |
| 林█ | 許小小未來／重組形態，通過非線性時間留下痕跡幫助周倉 |
| 陳述 | 周倉的前路鏡像，曾試圖救回 X 但失敗 |
| X | 陳述曾帶出的關聯實體，許小小的源頭之一 |

---

## 7. 核心鏡像結構

| 陳述線 | 周倉線 |
|---|---|
| 陳述帶出 X | 周倉與許小小建立未完成交易 |
| X 被系統抹消 | 許小小後期為救周倉而散掉 |
| 陳述想救回 X | 周倉想找回許小小 |
| 陳述差點抹掉 X 的新人格 | 周倉必須避免把許小小還原成 X |
| 陳述留下筆記與痕跡 | 林█與周倉留下跨時間痕跡 |

---

## 8. 後續待定

1. 周倉的真實身份是否與陳述有更深關係。
2. 林█的完整名字。
3. X 的原始副本與具體人格。
4. 陳述現在是活人、碎片、同化者，還是筆記集體意識的一部分。
5. 許小小最終是否能以「許小小」身份回歸。
