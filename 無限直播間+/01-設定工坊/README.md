## 設定狀態標記

- **已定**：已由作者確認，可作為後續寫作依據。
- **暫定**：方向已明確，但可在後續細化時調整。
- **未決**：仍需討論，不可直接寫入正文作為定案。
- **禁用**：明確不採用。

## 重要原則

- 目前先使用 **Part／大篇章** 管理全書，不先鎖死卷數。
- 「卷」只在後續拆分正文時使用。
- 每個設定需要說明：功能、限制、可用伏筆、不可違反之處。
