# 題材分析

## 文件狀態

【已定案】

本文分析《無限直播間+》的題材組合在中文網文市場中的定位、優勢與風險。

本文不是設定文件——設定查閱 [[已定案設定總表]]。本文是市場策略文件。

---

## 題材組合

《無限直播間+》不是單一題材。它是以下題材的複合體：

| 層級 | 題材 | 市場標籤 | 權重 |
|---|---|---|---|
| 主層 | 無限流 | 副本通關、規則破解、生存 | 40% |
| 主層 | 規則怪談 | 詭異規則、恐怖氛圍、智力對抗 | 30% |
| 副層 | 直播流 | 彈幕互動、觀眾視角、表演性 | 15% |
| 底層 | 存在恐怖 | 人格存續、系統真相、抹消 | 10% |
| 調味 | 懸疑推理 | 伏筆回收、真相拼圖 | 5% |

### 題材定位一句話

> 無限流 + 規則怪談 + 直播間的複合體，以「人格存續與存在校正」為深層主題。

## 各子題材市場分析

### 1. 無限流

**市場狀態**：成熟期。

- 優勢：讀者基數大、接受度高、副本結構天然適合長篇連載。
- 劣勢：高度同質化。常見套路為「主角進入副本→通關→獲得獎勵→變強→進入下一副本」。
- 機會：讀者渴望「不一樣的無限流」——不是變強循環，而是有深層敘事目標的副本結構。
- 本作的差異化：周倉不是靠變強通關——他是靠規則破解和情報推理。副本不是目的，是發現世界觀真相的工具。

### 2. 規則怪談

**市場狀態**：成長期→熱潮期。

- 優勢：近兩年最熱門的恐怖子題材之一。讀者對「規則→試探→代價→真規則」的認知快感有強烈需求。
- 劣勢：氾濫風險。大量跟風作導致讀者疲勞。多數作品只能玩「規則恐怖」，缺乏深層世界觀支撐。
- 機會：規則怪談 + 系統世界觀（存在校正）的組合幾乎未見。多數規則怪談不解釋「為什麼有規則」——本作可以。
- 本作的差異化：規則不是背景設定——規則是系統「存在校正」的具體執行方式。每一條規則背後都有世界觀邏輯。

### 3. 直播流

**市場狀態**：成熟期，但創新停滯。

- 優勢：彈幕是高互動性的敘事工具、讀者代入感強。
- 劣勢：多數直播流把彈幕當背景噪音或廉價笑點。很少作品認真設計「觀眾為什麼看」。
- 本作的差異化：觀眾不是背景——他們是觀測者池的一部分，是系統因果確認的工具。彈幕有世界觀功能（不是裝飾）。

### 4. 存在恐怖／人格存續

**市場狀態**：利基市場。

- 優勢：深度讀者忠誠度極高。這類作品的粉絲會反覆閱讀、討論、解謎。
- 劣勢：門檻高。不是所有人都有耐心讀「人格被刪除後是否仍然存在」。
- 本作的策略：這層是最深的主題——不寫在表面上。表面是規則怪談＋直播間求生。底層才是存在恐怖。讀者可以只享受表面層，深度讀者可以挖掘底層。

## 題材組合優勢

| 優勢 | 說明 |
|---|---|
| **多層讀者** | 休閒讀者看規則破解和彈幕互動；深度讀者看存在恐怖和系統真相。一本書覆蓋兩種讀者 |
| **結構穩定** | 無限流提供連載架構（副本章節天然分段），規則怪談提供每章衝突，直播間提供互動層，存在恐怖提供終局重量 |
| **差異化明顯** | 規則怪談 + 系統世界觀的組合少見；「系統不是敵人也不是工具，而是一個有邏輯但不人道的存在」這個定位很獨特 |
| **伏筆深度** | 存在恐怖主題允許極深伏筆——一個細節可以在 50 章後回收。適合長篇 |

## 題材組合風險

| 風險 | 等級 | 緩解策略 |
|---|---|---|
| **前期太重** | 🔴 高 | 前 50 章以規則怪談＋求生為主；存在恐怖主題只在細節中暗示，不直接討論 |
| **哲學設定嚇跑讀者** | 🔴 高 | 永遠不要讓角色討論「存在主義」。讓恐怖透過情節浮現，不是透過對話 |
| **彈幕可能變噪音** | 🟡 中 | 嚴格控制彈幕密度（見 [[行文風格規範]]）；每條彈幕要有功能 |
| **規則怪談審美疲勞** | 🟡 中 | 每個副本有不同的恐怖類型（便利店 = 空間恐怖，檔案館 = 記憶恐怖，不要重複） |
| **中段節奏下滑** | 🟡 中 | Part05（未登記主播篇）是節奏重啟點——周倉從被動變主動。必須確保 Part03-04 不拖 |

## 與競品的差異化定位

| 競品常見做法 | 本作做法 |
|---|---|
| 主角靠變強通關 | 周倉靠觀察、推理、規則漏洞——情報型主角 |
| 系統是中立的工具 | 系統有目的——它不是工具，是角色 |
| 死亡 = 重來／復活 | 死亡 = 抹消 = 從來沒有存在過。唯一的例外是周倉（且只有一次） |
| 彈幕是笑點／背景 | 彈幕有世界觀功能——它來自觀測者池，是因果確認的一部分 |
| 規則只是副本機制 | 規則反映世界底層公理——存在校正 |
| 結局是打敗最終 boss | 終局方向是改寫規則，不是打敗誰 |
| 情感線是附屬 | 許小小是核心——她不是獎品，不是助手，不是被救的公主。她是周倉欠下的帳 |

## 讀者畫像

### 主要讀者

- 18–30 歲男性
- 喜歡智力對抗 > 純戰鬥
- 喜歡規則破解的「啊哈」瞬間
- 能接受恐怖但不喜歡純血腥
- 對「不一樣的無限流」有需求
- 平台：起點 / 番茄

### 次要讀者

- 喜歡懸疑推理的讀者
- 喜歡世界觀深挖的設定控
- 喜歡情感線但不喜歡純言情的讀者（許小小線）
- 喜歡「重讀會發現新東西」的深度讀者
