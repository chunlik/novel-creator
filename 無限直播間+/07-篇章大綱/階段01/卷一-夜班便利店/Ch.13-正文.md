# Ch.13：回应

## 1

第二天。

周仓醒过来。

他——睡了 12 小时。

他坐在休息室的床上，打开系统。

系统提示弹出：

```
[清除者追踪状态]
异常关联者：P-0217 周仓
追踪状态：已丢失
原因：目标时间流异常——玩家在副本内的主观时间与主系统时间不同步
清除者预计到达时间：未知
安全期剩余：6 天 12 小时
提示：时间异常导致清除者追踪信号减弱——但不会永久失效
```

时间异常让他暂时摆脱了清除者。
但——只是暂时。

系统又弹出一条提示：

```
[追踪状态更新]
清除者状态：已进入失物档案馆
当前坐标：档案馆入口区
预计行程：约 7 天（副本时间）
```

清除者进了档案馆——但需要 7 天走完他走过的路。

周仓——看着这条提示。
清除者——进来了。
他以为 7 天安全期是绝对的——但系统说的是「相对安全期」。
不是清除者不能进——是他需要更多时间走完周仓走过的路。
清除者已经进了档案馆——但他还在入口区。
而周仓——在深处。

他在找——林东有没有回复。

他打开「副本内部消息」。

```
周仓：

我收到了你的消息。我看了五遍——哭了很久。

我一直以为他们恨我。我不敢回去看他们的档案——我怕看到「你害死了我」。

但你告诉我他们没有骂我。他们只是希望我承认错了。

我一直不敢承认。我怕承认错了意味着我是个坏人。

但我错了。

我设计了规则让他们死了。我错了。

赵雪说喜欢我？我不知道。我以为她只是我的朋友。

陈旧说希望我能原谅自己？我不知道能不能。

但你说许小小陪我三年。我知道。我每天都知道。她在便利店里等我。我不敢进去——我怕她骂我。但你说她没有骂我——她只是陪我。

我承认了。我不值得她等三年。但她等了。

我会好好对她。我会告诉她我错了，告诉她所有人的话。

谢谢你把真相带给我。

如果你以后还会来便利店——我们会等你。

便利店永远开着。但不会再有红雨衣了——因为我不需要了。

我是林东。不是红雨衣。

谢谢你。
```

周仓关掉消息前——注意到信的末尾有一行被涂黑的字。

他把屏幕对着光——眯着眼——勉强辨认出「不是他的错——是我让他......」

后面的字——完全看不清了。

周仓把信纸举起来——对着光。不够。他把系统界面调成紫外扫描模式——对准信纸。黑色涂改层下面——一行字浮现出来：

「不是他的错——是我让他这么做的。」

弹幕飘过：
```
？？？？
還有隱藏訊息？？？
我就說林東那封信有問題
不是他寫的——是有人在他之後塗的
```

周仓愣住了。他重新看了一遍林东的档案。林东写的是「我设计了规则」。但让林东设计规则的那个人——才是真正的幕后推手。

他打开系统对话：「系统。这行字是谁写的？」

```
┌──────────────────────┐
│  系统：              │
│  信息层级不足。      │
│  玩家当前不可查询。  │
└──────────────────────┘
```

「那你告诉我——这行字是什么时候写的？」

```
┌──────────────────────┐
│  系统：              │
│  2023-11-07。        │
│  副本崩溃之前。      │
└──────────────────────┘
```

2023-11-07。副本崩溃之前。

不是许小小写的——是另一个人——在所有人进副本之前——就已经留下了这行字。

周仓关掉了对话。他知道系统不会给他更多了。但他拿到了一个坐标：这个人的存在——被系统承认了。

周仓看着这条消息。

他——读了很久。

林东——承认了。

他——错了。

他——愿意——告诉许小小真相。

他——愿意——好好活下去。

他——不再是红雨衣了。

他是——林东。

便利店——永远开着。

周仓——松了一口气。

他——做到了。

他——把真相——带给了林东。

他——让林东——承认了错误。

他让林东愿意好好活下去。

他让许小小三年的等待有了意义。

弹幕刷过：

```
他做到了
他把真相带给了林東
林東承認錯誤
許小小的三年等待有了意義
便利店不再有紅雨衣了
```

## 2

周仓洗了把脸。

他从休息室出来，走到主系统大厅。

大厅里——还是很多人。

但他注意到——他醒来的时候——大厅里有了一些变化。

他看到——大厅的中央——多了一块屏幕。

屏幕上显示着：

```
[主系统大厅 - 公告]

各位玩家：

P-0217 周仓 玩家
在便利店副本中
达成了「特殊通关」：
不通过战斗，改变了副本的运行方式。
让副本内的失败者——林东——承认了错误。
让副本内的失败者——许小小——得到真相。

这是主系统历史上
第一个「通过改变关系」而非「战斗」
完成的副本。

P-0217 周仓 玩家
获得特殊奖励：

1. 主系统大厅 - 永久 VIP 休息室
2. 主系统大厅 - 专属 NPC：便利店店员
3. 下一个副本 - 难度降一星

便利店副本
正式标记为「已完成」
所有副本内的失物——已经获得归宿。

感谢 P-0217 周仓 玩家。
```

周仓看着这个公告。

他——成了主系统大厅的「名人」。

他——获得了永久 VIP 休息室。

他——获得了专属 NPC。

他——下一个副本难度降一星。

弹幕炸了：

```
他是名人
主系统大厅第一个
「通过改变关系」完成的副本
这种人
以前没有过
```

周仓看着公告。

他——不觉得自己做了什么特别的事。

他只是——把真相带给了应该知道的人。

他只是——让林东承认了错误。

他只是——让许小小——不再被「等待」。

他只是——做了他觉得——应该做的事。

他走向任务大厅。

他要接——下一个副本。

他打开副本列表。

```
[任务大厅 - 新副本列表]

1. 副本：失物档案馆 [已完成]
2. 副本：末班地铁
   类型：规则限制类
   难度：四星 → 三星（已降星）
   奖励：1200 积分
3. 副本：旧医院七楼
   类型：恐怖类
   难度：五星
   奖励：2000 积分
4. 副本：开学日
   类型：日常类
   难度：二星
   奖励：800 积分
5. [新] 副本：失物招领处
   类型：特殊类
   难度：不定
   奖励：不定
   备注：仅「秘密携带者」称号玩家可接
```

周仓看着第 5 个副本。

失物招领处。

仅「秘密携带者」可接。

他——刚刚获得这个称号。

他——点开了详情。

```
[副本详情 - 失物招领处]

类型：特殊类
难度：不定
时长：不定

简介：
失物招领处——不是档案馆。
档案馆记录「失物」。
失物招领处——是「失物被找到」的地方。

本副本的玩家——不是来找失物的。
是来「被失物找到」的。

通关条件：
被 5 件失物找到。

注意：
本副本的「敌意」不是来自玩家。
是来自——失物本身。
失物找到玩家是为了
被玩家「收留」。
如果玩家——不愿意收留——
失物——会——
```

周仓看完介绍。

他注意到了一个细节。

介绍——没有写完。

「失物——会——」

会什么？

他想了一下。

他接了这个副本。

在他點下接受的同時——走廊深處——傳來一聲輕響。

像是腳步。

他回頭——走廊空的。

他沒多想。白光出現。

```
[已接受副本] 失物招领处
开始时间：现在
```

白光——再次出现。

他——进入了——下一个副本。

━━━

【系统奖励】主系统大厅 VIP 休息室 / 专属 NPC / 难度降星 / 特殊称号「秘密携带者」
