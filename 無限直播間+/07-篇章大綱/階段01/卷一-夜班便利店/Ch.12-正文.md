# Ch.12：档案

## 1

周仓走出特殊档案室。

他还差 4 个失物主人。

系统提示弹出：

```
[追踪状态更新]
清除者状态：已锁定副本入口
安全提示：副本时间流速异常——清除者无法立即进入
当前安全期：7 天
```

周仓确认了时间显示——失物档案馆的 7 天，是他现实中的安全窗口。

但他——需要想清楚一件事。

失物档案馆——5 个失物主人——要找到谁？

林东和许小小——他已经找到了。

还有——周远（CS-0001）。

王可（CS-0004）。

刘一鸣（CS-0005）。

赵雪（CS-0006）。

陈旧（CS-0007）。

六个人。

加上林东和许小小——八个人。

但他只需要找 5 个。

他想了一下。

他需要找的——不只是「失物的主人」。

他需要找的——是「知道真相的人」。

周远、王可、刘一鸣、赵雪、陈旧——他们——知道 2023-11-07 那天发生了什么吗？

他们——和林东、许小小——一起进了副本——然后——都「死」了。

他们——是「死亡」状态吗？

还是「残留」状态？

他回到档案馆大厅。

他去找守门人。

「守门人。」

「嗯。」

「周远、王可、刘一鸣、赵雪、陈旧——他们——在档案馆里吗？」

守门人看着他。

「你——」他说，「要找他们？」

「是。」

「为什么？」

「因为——」周仓说，「我想知道——那天到底发生了什么。」

守门人沉默了一会儿。

「他们的档案——」他说，「不在 B 区。」

「在哪？」

「C 区。2022-2024。」

守门人说这句话的时候他的手指在柜台上敲了两下。
很轻。
但周仓——注意到了。

周仓没有离开。
他看着守门人：「你刚才说C区封了三年——但你敲桌面的节奏是『快』。你希望我快点去C区。」
守门人的脸色变了。
「你——你怎么——」
「我是来调查的。」周仓说，「不是来被忽悠的。」
他转身往C区走。这次——守门人没有拦他。

「他们——」周仓说，「都'死'了？」

「对。」守门人说，「完整回收。」

「完整回收——意味着什么？」

「意味着——」守门人说，「他们的档案——是'完整'的。」

「完整——」

「完整——意味着——他们留下了完整的记录。」守门人说，「他们记得那天发生了什么。」

「他们的档案——」周仓说，「可以看吗？」

「可以。」守门人说，「但——你需要付出代价。」

「什么代价？」

「积分。」守门人说，「每个完整档案——需要 1000 积分。」

1000 积分一个。

5 个人——5000 积分。

他只有 5010 积分。

刚好够。

但如果他都买了——他就剩 10 积分。

他没有其他积分来源了吗？

他打开商城。

```
[档案馆商城]

完整档案（周远）    1000 积分
完整档案（王可）    1000 积分
完整档案（刘一鸣）  1000 积分
完整档案（赵雪）    1000 积分
完整档案（陈旧）    1000 积分

注：每个完整档案包含 2023-11-07 事件的当事人视角记录。
```

5 个档案——5000 积分。

周仓看了三秒。

他全买了。

```
[已购买] 5 个完整档案 -5000 积分
[剩余] 10 积分
```

5 个档案出现在他的系统背包里。

他——没有积分了。

但他——有了所有的真相。

弹幕刷过：

```
他全部买了
他花光了所有积分
为了真相
```

## 2

周仓走到档案馆大厅的一个角落。

那里有一张阅读桌。

他坐下来。

他拿出第一个档案。

周远。CS-0001。

他打开周远的。

「我叫周远，第一个进副本的。林东设计了一套隐藏规则——每条规则都有隐藏成本。我不知道。我触发了。被副本吞噬。死的时候听到林东在笑——不是得意的笑——是在笑自己设计的规则太完美了。我不恨他。但他得承认他错了。」

打开王可的。

「我知道林东会害死我们。他的眼睛在说对不起。我问他副本危险吗，他说不危险。我死了。我恨他。但我理解他。别告诉他我恨他——他会难过。」

刘一鸣的。

「我傻。我相信他。他哭了，在门口哭。他不是故意的。我原谅他。但他得承认。」

周仓的手开始发抖。他打开赵雪的。

「我叫赵雪。我是那批人里唯一没有恨林东的。我进副本是因为我喜欢他。我知道他想证明规则可以被打破。我支持他。我愿意为他试一次。我死了。但我不后悔。如果有人能看到这份档案——请告诉林东——我喜欢他。我希望他能好好活着。」

周仓的手——抖得厉害。

赵雪——喜欢林东。她是自愿的。她不后悔。

他打开最后一个。陈旧。

「我是最后一个。我进去的时候林东已经开始哭了。他说你不要进去。但已经来不及了。我不怪他。我怪自己。请告诉他——希望他能原谅自己。」

周仓合上最后一个档案。

五个人。五段最后的话。都在说——「请告诉林东——」

周仓——那个人。

他——是那个——能把他们的「话」——

带出去的人。

弹幕刷过：

```
他會告訴林東嗎
還是——會帶進棺材
```

周仓站起来。

他没有直接走向特殊档案室。他先绕到了 C 区。

C 区——2022-2024 年的档案区域。

在走廊尽头，他看到了一个锁着的铁柜。柜门上刻着编号：C1。密码锁。

他看着密码锁。0217。不是周仓的编号——是铁柜的标号。柜门上的标签写着：02-17。

他输入了 0217。

咔哒。

柜门弹开。里面是空的。但柜子底部有一行刻上去的字：「有些失物——不想被找到。」

他记住了这行字。

他关上了柜门。然后他做了一件事——他从系统背包里拿出许小小的档案盒，抽出一张便签纸，写了一行字：

「C区 B3架 第五盒——你要找的编号。」

他把便签贴在铁柜内侧，然后重新锁上柜门。

他不是留给自己的。是留给赏金玩家的。如果对方真的在跟踪他，他们迟早会搜到这个柜子。这个假线索会让他们浪费至少半天时间去查一个根本不存在的档案。

他走向特殊档案室。

他推开门。

林东的档案还在桌子上。

他拿起档案盒，把五份「最后的话」放进了林东的档案盒里。

然后他合上了盒子。

他要把这些「话」带给林东。

## 3

周仓走出档案馆。

白光开始出现。

档案馆——开始消失。

系统公告弹出：

```
[副本结算]
副本：失物档案馆
结算评价：S
原因：收集了所有失物主人的「最后的话」
积分奖励：+3000
特殊称号：「秘密携带者」
纪念物品：林东的完整档案（已获得）
```

白光——吞没了周仓。

他——回到了主系统大厅。

休息室的小床——还在那里。

他——躺下来。

他很累。

他——需要休息。

但在他睡着之前——他做了一件事。

他打开了系统。

他找到了一项功能：

「副本内部消息」

这项功能——可以让玩家向副本内部发送消息。

他输入了一条消息：

```
收件人：便利店副本 / CS-0002 林东
消息内容：

我去了失物档案馆。
我找到了所有人想对你说的话。
周远说——他不恨你，但他希望你承认你错了。
王可说——他不原谅你，但他能理解你。
刘一鸣说——他原谅你，他希望你承认你错了。
赵雪说——她喜欢你，她希望你好好活着。
陈旧说——他希望你能原谅自己。
还有许小小。
她——在便利店副本里——陪了你三年。
她——知道真相。
但她——没有离开。
她留下来陪你等你。
她值得你承认你错了。
你——害了她三年。
但她——没有离开。
你——欠她——一个道歉。
不只是——「对不起」。
是——「我错了」。
你——愿意——吗？

——周仓
P-0217
```

他点了发送。

系统提示：

```
[已发送]
消息已送达便利店副本内部
收件人：CS-0002 林东
状态：待接收
```

周仓——关掉系统。

他——躺下来。

他——睡着了。
