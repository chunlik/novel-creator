# Ch.11：失物档案馆

## 1

周仓睁开眼。

他站在一个——档案馆的大厅里。

不是便利店。

是一个真正的档案馆。

木质地板。深色木质书架。吊灯——老式的水晶吊灯——发出暖黄色的光。

空气中有一股旧纸张的味道。

和便利店不一样。

便利店的空气是冷的、塑料的、人工的。

档案馆的空气是——旧的。真的旧的。像是时间本身被装在了瓶子里。

周仓环顾四周。

系统提示弹出：

```
[清除者追踪状态]
异常关联者：P-0217 周仓
追踪状态：已锁定——目标进入新副本
清除者出动倒计时：30 小时（主系统时间）
——安全期说明——
副本时间流速异常——清除者无法在副本内部直接追踪。
玩家在副本内享有约 7 天的相对安全期（副本时间）。
安全期结束后——清除者将获得副本入口权限。
剩余安全期：7 天（主系统时间）
建议：在安全期内完成副本。
```

7 天。
他——需要在 7 天内——完成这个副本。
否则——清除者——会进来。

他——把这条提示——记在了脑子里。

档案馆的大厅很大——至少 200 平米。书架从地板一直到天花板。书架上摆满了——档案盒。

不是文件夹。是那种老式的、牛皮纸的档案盒。盒子上贴着标签。

他走近一个书架。

标签上写着：

```
2018-2020 / 失物登记 / 个人物品
```

他抽出一个档案盒。

盒子里是一些物品——一个钱包。空的。

一把钥匙。钥匙上挂着一个小牌子：「xx 小区 3 栋 502」。

一张照片。照片上是三个人。

照片背面有手写的字：

「失主：刘雨。2019 年 3 月 12 日在地铁站失物招领处登记。」

周仓看着这些物品。

他注意到了一个细节。

物品上有标签。但标签——没有「失主」的信息。

钱包的标签是「刘雨」。

但照片上——不是同一个人。

照片上三个人——都不像「刘雨」。

他打开了档案盒的盖子。

盖子内侧——有一行手写的字：

「失主档案仅供参考。实际失主以最终登记为准。」

周仓看着这行字。

最终登记。

失物档案馆——收集失物——但失物的「主人」——不是「物品上标签写的人」——

是「最终登记的人」。

什么意思？

他想了三秒。

他明白了。

失物——不只是物品。

失物是——「被遗忘的关系」。

照片上的三个人——不是「刘雨」。

他们——是「刘雨」曾经认识的人。

「刘雨」失物了——不是她丢了东西——是她——被她的关系网络「遗忘」了。

她——不再被人记得。

所以她——是「失物」。

弹幕滚过：

```
失物不是物品
失物是人
是「被遗忘」的人
他们来档案馆——是为了——被找到？
```

周仓关掉档案盒。

他看到档案馆的大厅中央——有一个人。

一个穿灰色制服的人。

他站在柜台后面。

他看着周仓。

「欢迎。」他说，「新登记的玩家？」

「是。」周仓说。

「我是档案馆的管理员。」他说，「你可以叫我'守门人'。」

「失物档案馆——」周仓说，「要做什么？」

「找到 5 件失物的主人。」守门人说，「这是你的通关条件。」

「5 件失物的主人——」周仓说，「是要我帮他们找到吗？」

「不是。」守门人说，「是你——找到他们。」

「什么意思？」

「失物——」守门人说，「是被遗忘的人。他们来到档案馆——是因为他们被遗忘了。」

「你要做的——不是'帮他们'——是'找到他们'。」

「找到他们——然后呢？」

「然后——」守门人说，「你可以选择：让他们被找到 / 让他们继续被遗忘。」

「让他们被找到——」周仓说，「有什么后果？」

「他们会被——送回去。」守门人说，「送回他们被遗忘的地方。」

「送回去——他们会怎样？」

「他们会——被那些遗忘他们的人——找到。」守门人说，「但——不是所有遗忘他们的人都想找到他们。」

「所以——」

「所以——」守门人说，「送回去——有时候是好事，有时候是坏事。」

「取决于——」

「取决于——他们被遗忘的原因。」

周仓沉默了。

他想到了许小小。

她——是被遗忘的人吗？

她在便利店副本里待了三年——是因为——她被「遗忘」了？

三年前——和她一起进副本的人——都「消失」了。

她的家人朋友他们还记得她吗？

他问了守门人。

「守门人。」

「嗯。」

「失物——」他说，「能被送回主系统大厅吗？」

守门人看着他。

「你在问——一个特定的人？」

「是。」

「一个叫许小小的人？」

周仓的呼吸停了一秒。

「你怎么知道？」

「因为——」守门人说，「你来这个副本——不是随机的。」

「什么意思？」

「失物档案馆——」守门人说，「不是普通副本。它是——特定玩家的特定副本。」

「特定玩家——」

「你是——」守门人说，「第一个完成便利店副本的、获得'副本改变者'称号的玩家。」

「所以——」

「所以——」守门人说，「失物档案馆是——为你打开的。」

「为我——」

「为了让你——」守门人说，「找到许小小被遗忘的原因。」

弹幕刷过：

```
失物档案馆是為他打開的
為了讓他找到許小小被遺忘的原因
這是系統的安排
```

周仓看着守门人。

「许小小——她为什么——被遗忘？」

守门人没有立刻回答。

他看了看大厅的某个方向。

「她的档案——」他说，「在 B 区。第三排。第七个书架。」

「你可以去看。」

## 2

周仓走向 B 区。

档案馆的布局很清楚：

A 区：2018-2020 年失物
B 区：2020-2022 年失物
C 区：2022-2024 年失物
D 区：2024-2026 年失物
E 区：特殊档案

他走到 B 区。

第三排。第七个书架。

书架上的档案盒——比其他书架上的要多。

他看着标签：

「CS-0003 许小小 - 副本残留 - 失物登记」

他抽出来。

档案盒比其他的重。

他打开。

里面的物品——一张便利店的工牌。照片上是许小小。名字是「许小小」。编号是空的。

一张纸条。纸条上写着：「许小小——如果我们没能出去——记得你是我最好的朋友。——林东」

一个空的马克杯。杯子上贴着便利店的标签。

一份「副本结算报告」。报告上写着：

```
玩家：许小小（CS-0003）
副本：夜班便利店
进入时间：2023-11-07
结算时间：2023-11-08
结算结果：未完成回收 - 数据残留
备注：玩家意识与副本结构深度耦合，无法单独回收。
       建议处理方式：等待副本自然关闭。
```

周仓看着这份报告。

「等待副本自然关闭」——

但便利店副本——因为林东——不会自然关闭。

所以许小小——被「等待」了三年。

她——不是「被遗忘」。

她是——「被系统等待」。

但系统没有说——她会被永远等待。

系统说——「等待副本自然关闭」。

但副本不会自然关闭——所以她——会永远被等。

她——被「等待」了。

不是被「遗忘」。

是——被「永远等待」。

弹幕飘过：

```
她不是被遺忘
她是被永遠等待
系統沒有處理她
系統只是——讓她等
但她等的——不是希望
她等的是——副本崩潰
但副本不會崩潰
所以她會永遠等
```

周仓合上档案盒。

他问了系统一个问题。

「系统。」

```
┌──────────────────────┐
│  系统：              │
│  说。                │
└──────────────────────┘
```

「许小小——她为什么会被'等待'？系统为什么没有处理她？」

```
┌──────────────────────┐
│  系统：              │
│  因为——她不应该被遗忘。│
└──────────────────────┘
```

「什么意思？」

```
┌──────────────────────┐
│  系统：              │
│  她是2023-11-07 那批人里│
│  唯一一个——不应该被遗忘的。│
└──────────────────────┘
```

「为什么？」

```
┌──────────────────────┐
│  系统：              │
│  因为——她——           │
│  知道——那一天——       │
│  到底——发生了什么。   │
└──────────────────────┘
```

周仓的心跳加快了。

许小小知道——2023-11-07 那天——发生了什么。

那天她和林东、周远、王可、刘一鸣、赵雪、陈旧七个人一起进了便利店副本。

她知道——为什么林东要自己进副本。

她知道——为什么他们会「死」。

她知道——但她——没有说。

她——把真相带进了便利店副本。

她在副本里——守了三年的秘密。

周仓关掉系统对话框。

他看着档案盒。

他想到了一个事。

许小小——她被「等待」——不是系统的问题。

是——她自己想被等待。

她想守在便利店副本里因为她要等一个能听她说的人。

等——一个能「带她出去」的人。

但「带她出去」不是「带她离开副本」——

是——「带她的秘密出去」。

她想让她的秘密——被人知道。

周仓拿起档案盒。

他把它收进了自己的系统背包里。

```
[物品] CS-0003 许小小 完整档案
       状态：已获得
       说明：包含许小小的所有失物记录
```

他拿到了第一个失物的主人档案。

他还需要 4 个。

但他现在知道了一件事：失物档案馆不是关于「物品」的，是关于「被等待的秘密」的。他要找的不是失物的主人，是失物主人藏着的秘密。

他走出 B 区。

他看到档案馆大厅的另一个角落——有一个单独的小房间。房间的门上写着「特殊档案 - 需授权进入」。

他没有权限。

但他注意到了一个细节。小房间的门口刻着两个字——「林东」。

林东也来过这里。这个房间是林东的档案所在地。

周仓转身走向柜台。

「守门人。」

「嗯。」

「林东的房间——给我权限。」

守门人看着他，没说话。

「你刚才说失物档案馆是'为我打开'的。」周仓说，「那你应该知道我需要进那个房间。给我权限。还是说——你希望我去系统前台举报「隐瞞副本核心支柱信息」？」

守门人沉默了三秒。然后他伸手指了指柜台上的登记簿：「在下面。」

周仓翻开登记簿。最后一页的背面用铅笔写着一行字：

「临时权限码：B-1107」

「你早就准备好了。」

守门人没有回答。他只是看着周仓，眼神里有一种周仓读不懂的东西——不是敌意，是期待。

周仓拿着权限码，走向林东的房间。

他把手放在门把上。系统提示：

```
[权限检测]
检测到凭证：B-1107
匹配记录：CS-0002 林东 临时权限（已过期）
系统判定：权限码匹配——可以进入（功能受限）
```

他推开了门。

门后——是一个小房间。

房间里只有一张桌子。

桌子上放着一个档案盒。

档案盒的标签上写着：

```
CS-0002 林东 - 副本程序员 - 特殊档案
警告：此档案涉及副本设计机密
警告：此档案涉及 2023-11-07 事件真相
```

周仓打开档案盒。

里面——有一份文件。

文件标题是：

「2023-11-07 事件报告 - 林东 视角」

他打开文件。

第一行写着：「如果你能看到这份文件——说明你已经知道了一些事。」

「但你可能不知道——2023-11-07——到底发生了什么。」

「那天——我们七个人——不是意外进入副本的。是有人故意让我们进去的。」

「那个人——是我。」

「我——设计了一个规则。一个专门为我们七个人设计的规则。那个规则——我没有告诉系统——我把它藏在了便利店副本里。」

「我想——证明——规则是可以被打破的。」

「但——我失败了。」

「我设计的规则——太完美了。完美到——连我自己都无法打破。」

「我们七个人——都死了。许小小——是唯一活下来的。」

「她知道——规则是我设计的。她知道——我们是被我害死的。」

「但她没有恨我。她只是——留下来——陪我——等一个能把我带走的人。」

「三年了。我每天站在门口——不是为了维持副本——是为了让她知道——我还在。」

「我还在——这里——陪她——等她——带她出去。」

「但我知道——我等不到。我不能自己走出去——因为我设计的规则——不允许我离开。」

「许小小——陪我——等了三年。」

「她——值得——被带出去。」

「如果你能看到这份文件——请告诉她——我不值得她等。」

「但——我——很开心——她——等了我。」

文件——结束了。

周仓看着这份文件。

他明白了。

那个人——是周仓。

弹幕刷过：

```
林東設計了規則
他害死了所有人
但許小小留下來陪他
三年
等一個能打破規則的人
那個人是周倉
```

周仓合上档案盒。

他拿到了林东的完整档案。

他知道了——2023-11-07 那天——到底发生了什么。

他——需要做的——不只是「找到 5 个失物的主人」。

他需要带许小小把她的秘密公之于众。

让所有人知道——她等的不是假希望。

她等的是——真相。

━━━

【章末钩子】周仓在失物档案馆找到了林东和许小小的完整档案。真相是：林东设计了一个专门针对他们七个人的规则，想证明规则可以被打破，结果害死了所有人（包括他自己）。许小小知道真相，但选择留下来陪他。等一个能打破规则的人——就是等周仓。
