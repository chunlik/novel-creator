# Ch.05：第一次攻击

## 1

05:00。

便利店进入新的时段。

规则里的「夜班」是 00:00-08:00。但系统没有说白天是什么规则。

周仓在交接本上写下：

```
04:45 - 调查 CS-0002 林东。
        - 林东是副本的程序员。
        - 他设计完副本第二天就自己进入测试。
        - 他违反了自己的规则。
        - 他死后变成红雨衣。
        - 他每天站在门口 = 维持自己的存在。
```

他放下笔。

许小小已经从柜台上消失了。

他环顾四周——她不在收银台、也不在货架区。

他走到后场门——门是关着的。

他敲了敲门。

「许小小？」

没有回应。

他试着推门——门锁着。

弹幕滚过：

```
她不见了
他刚说要带她走，她就不见了
诡异
```

周仓没有慌。

他走回收银台，看监控。

监控画面里——后场门是开着的。

但他刚才看到的是关着的。

监控里的时间戳是 05:03。

他看了一眼系统时间：05:05。

差距在拉大。

他深吸一口气。

「许小小。」他又叫了一次。

后场门开了一条缝。

许小小的脸从门缝里探出来。

「你——」她的声音有点沙哑，「你刚才说什么？」

「我没说什么。我叫了你两声。」

「哦。」她推开门走出来，「我以为我听到你说……算了，没什么。」

她的脸色不太好。

「你还好吗？」

「我没事。」她说，「我去休息一下。」

她走向后场门——门这次是开着的。

她推开门——走了进去。

门在她身后关上。

周仓站在收银台后面，看着那扇关上的门。

弹幕里有人开始打字：

```
她的状态不对
她是不是知道什么
```

周仓打开任务列表。

他注意到——一个新的被动任务已经启动。

```
[自动] 在便利店内存活 24 小时
      进度：05:23:45 / 24:00:00
      状态：进行中
```

他还有不到 19 个小时。

但他注意到——另一个任务已经标记为「已完成」。

```
[已完成] 完成 10 次顾客服务
        奖励：500 积分 + 1 能力点
        状态：可领取
```

他完成了？他没感觉到自己服务过任何顾客。

他点开任务详情：

```
完成明细：
1. 03:00 侧目观察红雨衣（系统自动判定为服务）
2. 03:10 监控画面分析
3. 03:30 系统对话
4. 04:00 调查值班表
5. 04:30 与 NPC 对话（许小小）
6. 04:45 商城浏览
7. 05:00 交接本记录
8. 05:10 货架巡视
9. 05:15 后场门观察
10. 05:20 监控死角分析
```

十次。

系统把他做的每件事都算作一次「服务」。

他点了领取。

```
[获得] 500 积分
[获得] 1 能力点

请在以下选项中选择一项永久能力：

A. 【侧目观察】- 在不直视的情况下看清更多细节
B. 【监控回溯】- 可以回看监控录像 30 分钟内的内容
C. 【时间校准】- 主观时间与副本时间的差异减少 50%
D. 【副本感知】- 可以感知副本里隐藏的规则漏洞
```

周仓盯着这四个选项。

弹幕立刻刷起来：

```
选 D！副本感知！
不对，选 B。监控回溯
C 才是最强的
B 是工具，A 是本能
```

周仓没有犹豫。

他选了 C。

```
[已选择] 【时间校准】
效果：主观时间与副本时间差异减少 50%
说明：你发呆时损失的时间将减少一半
```

他松了一口气。

这是他目前最需要的能力——他的时间在被副本偷走。

## 2

05:30。

周仓正在做时间校准测试的时候——风铃响了。

他立刻抬头。

门口——红雨衣。

它站在门口。全身包在红色塑料雨衣里。雨帽压得很低。

但这一次——它不是站在门口不动。

它在往里走。

一步。

周仓立刻把视线从它的脸移开。他看向门口旁边的落地玻璃——玻璃上映出了它的轮廓。

两步。

三步。

它走进来了。

它绕过了收银台——没有看周仓——直接走向货架区。

周仓的呼吸稳着。

它来了。

他不动。

四步。

五步。

它停在了第二排货架前。

他盯着玻璃上的倒影——看着它的轮廓。

它开始检查货架。

它伸手——红雨衣的袖子底下——伸出一只苍白的手。

它把那只手放在了货架上的一包薯片上。

薯片是 CH-0217。

它把那包薯片拿起来——举到雨帽下——像是在看。

然后它做了一件事。

它把那包薯片塞进了自己的雨衣里。

弹幕炸了。

```
它拿东西了
它是顾客吗
它偷东西！
红雨衣偷东西！
那包 CH-0217
它为什么要那一包
```

周仓的脑子转过。

CH-0217——这个编号他之前在监控上看到过。

CH 开头——是商品编号。

红雨衣是副本里的 NPC——但它在拿商品。

它在拿什么？

它的雨衣下面是什么？

红雨衣在货架前又站了一会儿。

然后它转身。

它朝收银台的方向走。

周仓没有动。

他盯着地面。不直视它。

它走到收银台前。

它站在他面前。

距离——不到一米。

周仓能看到它雨衣下摆的红色塑料在轻微地颤动。

他能看到它的鞋——雨鞋，黑色，湿的。

便利店外面没有下雨。

它的鞋是湿的。

「你——」周仓开口了，「要结账吗？」

他没有看它。

他对着空气说话。

弹幕里有人笑：

```
他在收银台问顾客要不要结账
就算顾客是鬼也要做生意
```

红雨衣没有回答。

它站在收银台前。

然后它做了一件事。

它伸出了那只手——那只刚才拿过 CH-0217 薯片的手——伸向了周仓。

周仓没有动。

他感觉到了一个触感——冰冷的。

像有人把一块冰放在了他的手腕上。

是它的手指。

它在摸他的手腕。

周仓的手腕内侧——什么都没有。

但它摸的位置——正是编号应该在的位置。

弹幕在疯狂滚动：

```
它在摸他的手腕
它在找编号
```

红雨衣的手指在他的手腕上停了三秒。

然后它收回手。

它转身。

走回门口。

推开门。

消失在门外的晨光里。

风铃响了。

然后安静了。

周仓站在收银台后面。

他的手腕——被它摸过的地方——有一圈红印。

不是疼。是那种「被冻过」之后的红印。

但他注意到了另一件事。

他手腕的红印——在慢慢消失。

但红印消失的位置——正在浮出一点东西。

一个很浅的、灰色的图案。

像编号的第一个字——「CH」

他的手腕上开始长出编号了。

━━━
