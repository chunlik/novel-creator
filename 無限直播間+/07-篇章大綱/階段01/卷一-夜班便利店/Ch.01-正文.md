# Ch.01：醒来

## 1

冷。

不是空调的那种冷。是瓷砖地面、玻璃门、白色日光灯一起包裹着的冷。像把一个活人塞进一个冰柜里——但冰柜是开着的，没有盖子。

周仓睁开眼。

他躺在一张折叠床上。折叠床靠着墙壁，墙壁是白瓷砖。头顶正上方，一根日光灯管嗡嗡地响。

他坐起来。

一个便利店。

货架。收银台。落地玻璃。关东煮的锅子在冒热气——但现在已经没人会来买关东煮了。凌晨三点，整条街上只有路灯亮着。

他不知道自己怎么来的。

他记得的最后一件事是下班。回到出租屋。洗澡。然后——

然后是这里。

他的脑子还是清醒的。没有迷药反应，没有宿醉感。他试着动了动四肢——正常。

他站起来。

脚下的折叠床在他起身时发出一声金属摩擦的响。

便利店的自动门突然开了一次。

没有人进来。

门又关上了。

周仓的脚步顿了一下。

他的视野右上角弹出一行字——淡蓝色，半透明，像游戏里的 UI：

```
┌──────────────────────────────────┐
│  副本：夜班便利店                  │
│  类型：规则限制类                   │
│  难度：三星                         │
│  存活要求：7 天                    │
│  副本次数：1/100                    │
│  状态：已开播                       │
│  观众在线：1,247                   │
└──────────────────────────────────┘
```

周仓盯着那行字看了五秒。

副本。规则。存活。

他被丢进了一个所谓的「副本」里。

而他正被一千多个观众看着。

视野右侧出现了一个滚动的字幕条——弹幕。

```
新人？
又是便利店。我看了五个了
这是 24 小时便利店，难度三星
三星？不是说一星吗
三星是改版以后。死亡率 73%
73%？新人能活吗
看脸
```

周仓扫了一眼弹幕，没有说话。

他首先做了三件事：

第一，他检查身上的口袋。空的。没有手机，没有钱包，没有任何个人物品。他被丢进来的时候什么都没带。

第二，他环顾四周。便利店大约 80 平米，标准大小。货架五排。收银台一台。冰柜一个。后场一扇门，挂着「Staff Only」的牌子。地上有四台监控，角度覆盖所有位置。

第三，他看天花板。没有摄像头。日光灯的另一边，有一个球形的黑色小装置——像烟感报警器，但不是。它在闪。每三秒闪一次蓝光。

那就是直播的设备。

他被一千多个人看着，24 小时不停地。

弹幕还在滚动。

```
他终于看到了
反应还行。没尖叫
你看他的眼神
不是……他在观察摄像头
这个人是观察型的
观察型的新人能活七天吗
不好说
```

周仓没有理弹幕。

他走到落地玻璃前。玻璃外面是一条普通的商业街。药店、面包店、洗衣店都关了。路灯亮着，照出一小段空荡荡的人行道。

玻璃上映出他自己的脸。

脸色不太好。但还撑得住。

他的视野左上角弹出新的提示：

```
副本规则（共 5 条）：

【规则 #1】
夜班期间（00:00-08:00），请勿直视门外的任何人或物。

【规则 #2】
03:00-04:00 期间，请勿靠近便利店冰柜。

【规则 #3】
任何时候听到「欢迎光临」自动语音，请立即回应「请慢走」。

【规则 #4】
货架上的商品不能跨区域移动。

【规则 #5】
若听到有人在喊「林东」，请不要回头。
```

五条规则。

周仓把每一条默念了三遍。

他注意到了三件事：

第一，规则都用了「请」字。「请勿」「请不要」——这不是命令，是请求。请求意味着有商量的余地。

第二，规则里有具体的对象（冰柜、欢迎语音、「林东」），但没有具体说明违反会怎样。系统没说违反规则会死。系统只说了「请勿」。

第三，第五条里出现的「林东」——不是规则的制定者，是规则里的一个角色。这意味着「林东」是便利店的一部分。

他正准备研究第一条规则的细节——

便利店的门又开了一次。

这次不是自动门。是被人推开的。

风铃响了。

门口站着一个人。

一个穿着红色雨衣的人。

周仓没有直视它。

他的视线落在了门口的地面上——离它大约三米远。他用余光看到它的轮廓：红色塑料雨衣、雨帽压得很低、看不到脸。

它在门口站了三秒。

然后它转身——

走了。

风铃又响了一声。

它从自动门进来，但走的是门。

不是自动门。

是手动推开。

周仓站在原地，盯着它离去的方向。

心率在加速。但他的呼吸很稳。

这就是他被丢进来的东西。

他深吸了一口气，走到收银台后面。他需要一个起点——一个能让他在 7 天后走出去的起点。

他看到了柜台上有一本交接本。本子的封面上用记号笔写着：

```
夜班交接本
```

他翻开。

第一页是空白的。第二页也是。他从第一页翻到最后一页——

全部是空白的。

但本子有被翻过的痕迹。书脊有折痕。某一页的边角微微卷起来。

他合上本子。

弹幕又说话了。

```
他在看交接本
里面有什么？
空的
什么内容？
空的
他翻了好久
他是不是发现交接本是空的？
对
就 73% 的人会忽略这一点
```

周仓把交接本放回原处。

他盯着空白的封面看了两秒。

然后他开始做今晚的第一件事。

他走到货架区，开始一排一排地数商品。

他没有说话。但他的动作很稳。系统提示里的「存活」两个字在他的脑子里一直转。

七十二小时不到。

一千二百四十七个人看着。

如果他死在这里——一千多个人会看到。

他不允许自己死。

━━━

【章末钩子】红雨衣第一次出现，周仓没有直视它——他保住了自己。但交接本是空白的，有人翻过。便利店不是从今天开始的。

【系统奖励】正确反应：未直视门外来人 +100 积分

【积分余额】100
