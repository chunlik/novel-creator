# Ch.01：进入直播间

## 1

地板是冷的。

不是瓷砖的那种凉——是冷气从地底往上渗，穿过衣服，贴着皮肤，一直钻进骨头缝里的那种。

周仓睁开眼。

灰白色的天花板。日光灯——有几根在闪，发出细微的嗡嗡声。空气里有消毒水的味道，还混着茶叶蛋和关东煮的气味。

便利店。

他坐起来。后脑勺有点痛，像被人从后面敲了一下。他摸了摸——没有伤口，没有血。但痛是真的。

最后的记忆：床。被子。枕头。普通的星期二晚上。

然后是这里。

他环顾四周。标准的夜间便利店。货架排列整齐，饮料柜亮着白光，收银台在门口方向，两面都是落地玻璃。外面很黑。路灯照出一小段街道，没有人。

"这是……"

话没说完。

他的视野左上角浮出一块半透明的黑色方框。

```
─── 直播间已连线 ───
主播：未登录
　　　状态：待确认
　　　观众：就绪
━━━━━━━━━━━━━━━━━
```

方框停留了三秒。然后像墨水被水冲淡一样，缓缓褪去。

周仓盯着那片虚空看了好一会儿。方框没有再出现。

"什么东西。"

右侧视野边缘，几行小字开始往上滚。

```
来了来了 新人
这副本没见过
便利店 看起来很和平嘛
```

字体是灰白色的，大小不一，像有人在屏幕上打字。没头没尾。没有上下文。就这样突然出现，又消失在视野边缘。

周仓闭上眼。再睁开。

字还在。

他深吸一口气。没有惊慌——至少表面上没有。心跳快了一点，但他没有让它继续加速。他站起来，拍了拍裤子上的灰。

名字：周仓。

年纪：二十七岁。

职业：不记得了——不对，不是不记得。是他从没想过这个问题。反正就是一个普通人，做一份普通的工作，过普通的生活。

直到三秒前。

三秒前他还在家里睡觉。现在他站在一间不认识的便利店的瓷砖地板上，眼前有半透明的系统提示框，右边有人在打字。

先确认状况。

不要慌。慌没有用。

他走向门口，想确认门能不能打开。

右手边的弹幕又更新了。

```
他动了
第一步往门口走 正常操作
赌他不够五分钟就会尖叫
```

周仓停了下来。

"你们看得到我？"

没有回应。字继续滚。

```
三分钟吧
那扇门打不开的 副本都这样
```

他等了五秒。没有人回答。那些文字就像自言自语——不是要跟他说话，只是在他旁边说给他听。

好吧。

他转身打量收银台的方向。也许那里有电话。也许有监控。也许——

柜台后面站着一个人。

## 2

一个穿着便利店制服的女人。年纪不大，大概二十五、六岁。白衬衫加绿色背心，领口别着名牌。黑头发扎成马尾。

她看着周仓。

表情很平静——不是惊讶，也不是警戒。就是普通的、正在值班的店员的表情。

"你是新来的夜班？"

声音不大，语气平常。

周仓愣了一下。"什么？"

"夜班。便利店的。"她微微歪了歪头，"还是你只是来买东西的？"

"我——"周仓看了看自己身上。他穿着睡觉时的 T 恤和短裤。没有钱包。没有手机。没有钥匙。"我不知道我为什么在这里。"

"哦。"

她把一瓶矿泉水放在柜台上，推过来。

"先喝点水。你脸色不太好。"

水是凉的。瓶身上没有品牌——但周仓现在不想计较这个。他拧开瓶盖，喝了一口。

"这里是哪里？"

"夜班便利店。"她说。

"我是说——"周仓停顿了一下，"算了。你叫什么名字？"

"许小小。"

"店员？"

"不然呢。"

她笑了一下——很浅，几乎算不上是笑。然后低头继续整理收银台上的东西。动作很熟练，像做过一百次一样。

周仓等了几秒，看她没有继续说话的意思，就拿起矿泉水走到旁边的货架区。

弹幕又开始了。

```
店员小姐姐耶
这副本有 NPC？这么和平？
不要立 flag 便利店副本都是先给你糖再给你刀
```

周仓看了几行，抓到了大致的节奏。他们在讨论他。不是每条都重要——大部分是废话，但他可以从中筛选信息。

"便利店副本。"

他低声重复了一遍这个词。

所以这是一个副本。

所以他在一个——游戏？直播？测试？——里面。

他站在货架前。架上摆着零食、泡面、饮料。跟世上的便利店一模一样。他抽出一包薯片——原味。包装的颜色和字体看起来正常，但他注意到右上角有一个很小的标示，写着：

`生产编号：CH-0217`

不是日期。是编号。

他把薯片放回去。

"诶——"

柜台方向传来声音。许小小在叫他。

"你的班表。"

她指了指收银台旁边墙上挂着的一张纸。

周仓走过去看。值班表上写着本周的排班，用黑色圆珠笔填的。字迹工整。夜班栏位有两个人名。

`周一：林东`
`周二：林东`
`周三：林东`
`周四：林东`
`周五：周仓`

他的名字在上面。

"这不是我写的。"

许小小耸了耸肩。"但你在上面。"

周仓看着那张表。周一到周四是一个叫林东的人。周五是他。没有说明。没有交接信息。就是一个名字。

他伸手想拿——指尖碰到纸面的瞬间，视野正中央弹出一块系统提示框。

比刚才的大。字体白色，背景深黑。

```
━━━━━━━━━━━━━━━━━━━━━━━
　　夜班副本：红雨衣便利店
　　副本规则：
　　
　　1. 夜班期间，请勿直视门外。
　　
　　详细规则将于副本过程中逐步揭示。
━━━━━━━━━━━━━━━━━━━━━━━
```

## 3

周仓看着那几行字。

第一条规则："请勿直视门外。"

他抬起头。

便利店的两面墙——从他站的位置看出去，左边一整面是落地玻璃，正面收银台旁边也是落地玻璃。外面是街道。路灯。没有人。

这是被玻璃包围的空间。

从外面，看得到里面。

从里面，也看得到外面。

规则说：不要看门外。

但他站的地方——视线随便一扫，就会扫到外面。他不需要刻意去看。他的余光里永远有那面玻璃、那条街、那片黑暗。

他把视线移回提示框。

框消失了。但他脑中的矛盾还在。

"不要看外面。"

但便利店是透明的。

这是规则的漏洞？还是规则本身就是问题？还是他只是漏看了什么细节？

他没有说话。他把这个矛盾记在心里——像在纸上折一个角，提醒自己这里有东西。

右侧的弹幕显然没有放过这个瞬间。

```
等等 这规则？
等等等等
落地玻璃耶 这不就看得到外面了吗
干 对耶
我刚注意到 这店是透明的
不要看外面 笑死 你他妈在玻璃盒子里叫不要看外面
```

然后有一条，语气不一样。

```
不要提醒他。让他看。
```

周仓的视线在那条弹幕上多停了一秒。

不要提醒他。

这条规则——可以是个陷阱。可以是个测试。也可以是一个玩笑，系统在测试他会不会蠢到遵守一条不可能遵守的规则。

他决定先不动。

他把视线从窗外移开——不是因为规则，是因为他需要整理信息。

目前知道的：

- 这里是"红雨衣便利店"。
- 他在被直播。
- 有人在看。
- 有一条规则，和环境有明显矛盾。
- 有一个女店员，叫许小小。

他不知道的：

- 这是谁的直播间。
- 他为什么在这里。
- 这条规则是真的还是假的。
- 如果违反规则，会发生什么。
- 那个叫林东的人在哪儿。

系统提示框没有再出现。便利店的日光灯继续嗡嗡地响。空调的声音从墙角传来。关东煮的锅子在冒热气。

许小小在收银台后面用手机看视频。看起来完全不像在值一个诡异的夜班。

太正常了。正常到不对劲。

周仓走回收银台前。他没有看向窗外——至少没有让自己停下来看。但他的余光知道那片玻璃在那里。

规则说不要看。

但他已经看到了。

他看到的是空无一人的街道，和一盏故障的路灯，在夜色中一闪一闪。

不过——

他刚才瞥到的那一瞬间。

就在他从货架走回收银台的路上，眼角扫过窗外的瞬间——好像有什么东西在动。

不确定。太快了。可能是自己的倒影。

他没有停下来确认。

他不想冒险——还不想。

系统提示框上的第一条规则还留在他视野的残影里。白色字，黑色底，没有消失。

```
夜班期间，请勿直视门外。
```

但便利店是玻璃的。

他在玻璃盒子里。

规则要他不要看。

周仓站在收银台旁边，手里握着那瓶没有品牌的矿泉水。他的余光里，落地玻璃的边缘有一道反光——街灯透了进来，在瓷砖地板上拉出一条长长的影子。

他没有转头看。

但那个影子，在他没有看着它的时候——

好像动了一下。

━━━

【章末钩子】第一条规则的矛盾细节——"不得直视门外"但便利店两面落地玻璃。读者在周仓之前就注意到规则与物理空间的不一致。窗外有什么东西在动，但周仓没有转头确认。
