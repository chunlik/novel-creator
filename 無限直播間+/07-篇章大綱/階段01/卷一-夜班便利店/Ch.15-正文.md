# Ch.15：归宿

## 1

周仓看着 5 条未读消息。

他先打开了第一条。

```
[失物主人：小宝]

周仓：

你是收留我妈妈钱包的人吗？

我妈妈——三年前——离开了家。
她没有说去哪里。
她只留下了她的钱包。
钱包里有她和我拍的照片。
我——把钱包放在了失物招领处。
我——等她回来拿。

但她——没有回来。

三年了。

我——已经不抱希望了。

但你——收留了那个钱包。

谢谢。

我——我想——问你一个问题：

你能帮我告诉她我想她我希望她回来但如果她不想回来我也会好好活着。

——小宝
2026-11-16
```

周仓看着这条消息。

他想了三秒。

他回复：

```
小宝：

我收到了你的消息。

我不知道你妈妈——在哪里。
但——我可以帮你找。

你——想让我帮你找她吗？

如果你想我会尽力。

——周仓
P-0217
```

他发送了。

他打开了第二条。

```
[失物主人：孩子]

周仓：

你收留了我送给爸爸的雨伞。

爸爸三年前和妈妈离婚了。

爸爸——没有——带我走。

我把雨伞放在了失物招领处。

我希望爸爸会回来拿。

但他——没有。

你——收留了——雨伞。

谢谢。

我——不想——见爸爸。

但我想知道他还好吗。

——孩子
2026-11-16
```

周仓看着这条消息。

这个孩子——想见又不想见他的爸爸。

他——只是想知道爸爸还好不好。

他回复：

```
孩子：

我收到了你的消息。

我不知道——你爸爸在哪里。

但如果你想我可以帮你问。

如果你不想我也可以不问。

你——想——怎样？

——周仓
P-0217
```

他发送了。

他打开了第三条。

```
[失物主人：李大伟]

周仓：

你收留了我的工牌。

我是这个地铁站的检票员。

我——三个月前——失踪了。

我不知道我是死了还是什么。

但我的工牌在失物招领处出现了。

这意味着我已经被遗弃了。

我——有一个——女儿。

她叫李小燕。

她——15——岁。

如果你能找到她请把工牌给她。

我不需要她原谅我。

我只希望她知道我很想她。

——李大伟
2026-11-16
```

周仓看着这条消息。

李大伟——失踪了三个月。

他的工牌——出现在了失物招领处。

这意味着——他可能已经死了。

但他的灵魂——还在等待。

他——想把工牌交给他的女儿。

他回复：

```
李大伟：

我收到了你的消息。

我会把你的工牌交给李小燕。

你——放心。

——周仓
P-0217
```

他发送了。

他打开了第四条和第五条。

那两条——是来自两个小孩的。

```
[失物主人：小孩 1]

我想妈妈[失物主人：小孩 2]

我想回家
```

周仓看着这两条消息。

他——不知道怎么回复。

两个小孩——想妈妈——想回家。

但他们——已经在失物招领处了。

他们的「家」——

是哪里？

他——不知道。

他想了三秒。

他回复：

```
[回复 小孩 1]

我会帮你找到你妈妈。

[回复 小孩 2]

我会帮你回家。

——周仓
P-0217
```

他发送了。

5 个失物——他都回复了。

弹幕飘过：
```
居然一个一个回
認真了
他真把他們當人看了
```

他——正准备——关掉消息界面。

系统弹出一条警告：
```
[安全提示]
检测到异常查询：有玩家正在通过主系统接口搜索您的当前坐标。
来源：未知玩家
建议：尽快更换位置。
```

周仓看着这条警告。有人——在找他。

他关掉警告。一条新消息——弹了出来。

```
[新消息]
发件人：许小小

P-0217 玩家：
我是许小小。
我在便利店副本里收到了你的消息。
你什么时候回来看我？
```

周仓——看着这条消息。
他——注意到了——一个细节。
许小小——从来不会叫他「P-0217 玩家」。
她——叫他——「周仓」。

这条消息——不是许小小发的。
是——守门人。

他——没有回复。
他——关掉了消息界面。

守门人——在监视他。

周仓——关掉了消息框。
他没有回复。
但他——打开了「消息追踪」功能。
他回复了一条：「好。明天下午三点——老地方见。」
然后他——守在屏幕前。
三分钟后——系统提示：「消息已读。」
又过了五分钟——守门人休息室的门——开了一条缝。
守门人探出头——看了看大厅——又缩了回去。
周仓——笑了。
他知道——是谁发的了。

他关掉消息框。站起来。

門被踢開了。一個人站在門口——黑色戰術背心，手臂上有系統編碼——玩家。

彈幕炸了：
```
完了完了完了
三萬積分他死定了
這不是普通玩家——是獵犬
```

「P-0217？」那人掃了他一眼——「三萬積分——值得跑一趟。」

周仓沒動。他認得這個編碼——昨天系統警告裡搜索他坐標的那個未知來源。

「主系統大廳不能攻擊玩家。」

「規則？」那人笑了——「你以為我接這種懸賞會在意規則？」

他往前走了一步。周仓後退。

「你跑不掉的。你進任何副本——我都能跟進去。」

周仓看著他——然後說了兩個字：「獵犬。」

那人的笑容頓住了。

「你——」

「順便問一句，C1鐵櫃和門縫裡的便籤好找嗎？」周倉看著他，「你繞了半天才發現那是假的，對吧？你以為你在追獵，其實你一直在踩我留下的腳印。」

獵犬的臉色變了。

「我看過你的檔案。失物檔案館——CS-0012。你是林東那批人裡唯一提前退出的。」

獵犬不笑了。

「那份檔案不存在。」

「存不存在你心裡清楚。你退出之後——林東一個人扛了所有。」

「閉嘴。」

「懸賞你的人是守門人。他在測試我——也在測試你。」

房間安靜了三秒。

彈幕笑瘋了：
```
？？？？
情報戰？？？？
他早就知道了
獵犬被他一句話壓住了
哈哈哈哈剛剛說他死定的出來
```

「你三秒之內走出這個房間——我就當沒見過你。」

周仓走了出去。

他沒回頭。但他聽到了身後那一句——「林東還好嗎？」

「他承認錯誤了。」

他走遠了。

他需要幫失物們找到歸宿。

## 2

周仓在主系统大厅待了三天。

三天里——他做了几件事。

第一，他给小宝的妈妈——发了消息。

他不知道她在哪里。但他用主系统的搜索功能——找到了一个可能的地址。

地址显示小宝的妈妈三年前去了另一个城市。

她没有回来是因为她在那里有了新的生活。

她——没有带走小宝。

她——留下了钱包。

周仓——把地址发给了小宝。

小宝——没有回复。

他——需要时间消化。

第二，他帮孩子——找他的爸爸。

搜索结果显示孩子的爸爸三年前去了另一个国家。

他——没有带孩子。

他——在国外——重新结了婚。

孩子——把雨伞放在了失物招领处。

周仓——把搜索结果发给了孩子。

孩子——只回了一个字：

「好。」

他——知道真相了。

他——接受了。

第三，他帮李大伟找李小燕。

李小燕还住在这个城市。

她——15 岁。

她——在等她的爸爸。

她——不知道她的爸爸——已经失踪了。

周仓——需要把工牌——交给她。

但他——不能直接联系她。

她——不是主系统的玩家。

他需要通过某种方式把工牌送到她手里。

他在主系统大厅——找到了一个功能：

「现实世界连接」

这项功能可以让玩家向现实世界的特定人发送物品。

但需要——支付——高额积分。

他打开了费用。

```
[现实世界连接]

向现实世界发送物品：1000 积分

1000 积分——他——有 4510 积分。

他付得起。

他——付了。

```
[已发送]
收件人：李小燕
物品：李大伟的工牌
状态：已发送
预计送达时间：3 天
```

工牌——会送到李小燕手里。

三天后——她会收到一个匿名快递。

里面——是她爸爸的工牌。

她会知道她爸爸曾经在地铁站工作。

她会知道她爸爸留了一个工牌想给她。

但她不会知道她爸爸已经失踪了。

她只会知道她爸爸想着她。

第四，他帮两个小孩——找妈妈——找家。

但他——找不到他们的信息。

他们——在失物招领处——待了太久。

他们——的「家」——

已经——不知道在哪里了。

他——没有办法。

他只能让他们继续在失物招领处等。

他知道这不够。

但他——只能做到这里。

系统提示弹出：

```
[追踪状态更新]
清除者当前坐标：档案室 B区
清除者状态：正在接近
```

周仓——皱了皱眉。
清除者——还在失物档案馆里。
但他——已经离开档案馆了。
清除者——追的是他「在档案馆里的影子」——不是他本人。
只要时间流速差还在——清除者就会一直慢 7 天。

越来越近了。

## 3

三天后。

周仓坐在主系统大厅的休息室里。

他——完成了所有失物的归宿。

他打开系统——

```
[系统公告]

玩家 P-0217 周仓
在副本「失物招领处」中
成功为 5 件失物找到归宿。

特殊成就：失物之友
成就说明：玩家不仅收留了失物，
还为失物找到了最终的归宿。

成就奖励：
1. 主系统商城 - 永久 9 折
2. 现实世界连接 - 永久免费
3. 下一个副本 - 难度再降一星
4. 特殊称号：「失物归宿者」

感谢玩家 P-0217 周仓
对失物招领处的贡献。
```

周仓看着这个公告。

他——做到了。

5 件失物——都有了归宿。

他——累。

但他——很满足。

他打开任务大厅。

```
[任务大厅 - 新副本列表]

1. 副本：失物招领处 [已完成]
2. 副本：末班地铁
   类型：规则限制类
   难度：四星 → 二星（已降两星）
   奖励：1200 积分
3. 副本：旧医院七楼
   类型：恐怖类
   难度：五星 → 四星（已降一星）
   奖励：2000 积分
4. 副本：开学日
   类型：日常类
   难度：二星 → 一星（已降一星）
   奖励：800 积分
```

周仓看着副本列表。

所有的副本——都降了星。

他——不需要担心难度了。

他可以选自己想去的副本。

他——想了三秒。

他——选了——「末班地铁」。

因为——李大伟——是末班地铁的检票员。

他想知道李大伟到底发生了什么。

```
[已接受副本] 末班地铁
开始时间：现在
```

白光——出现。

他——进入了——下一个副本。

【章末钩子】周仓为 5 件失物找到了归宿——他给李大伟的女儿寄出了工牌，给小宝和孩子找到了家人的信息。他接受了下一个副本「末班地铁」——他想查明李大伟失踪的真相。
