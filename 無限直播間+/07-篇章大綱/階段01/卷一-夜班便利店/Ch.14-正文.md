# Ch.14：失物招领处

## 1

周仓睁开眼。

他站在一个——地铁站里。

不是便利店。不是档案馆。

是一个——失物招领处。

但这个失物招领处——是地铁站的一部分。

地铁站的入口处有一个小房间。房间的玻璃门上写着：

「失物招领处」

玻璃门里——有一张桌子。桌子上摆着各种物品。

钱包、钥匙、手机、雨伞、眼镜、书籍、衣物——都是——普通的失物。

但周仓注意到了一个细节。

这些失物——没有一个有标签。

没有「失主登记」。

没有「招领编号」。

只是——一堆——没有主人的物品。

他推开玻璃门。

他走进了失物招领处。

里面——有一个人。

一个穿橙色马甲的老人。

他坐在桌子后面。

他看着周仓。

「你——」他说，「是来领东西的？」

「不是。」周仓说，「我是来——被找到的。」

老人看着他。

他的眼睛里——有了光。

「你——」他说，「是'秘密携带者'？」

「是。」

「那你——」老人说，「要小心。」

「为什么？」

「因为这个副本的失物——不是普通的失物。他们是——被遗弃的失物。」

「被遗弃——是什么意思？」

「意思是——他们曾经有主人。但主人不要他们了。他们被丢在这里。不是被遗忘——是被遗弃。」

周仓沉默了。

被遗弃和被遗忘不同。被遗忘是「忘了还有这个东西」。被遗弃是「知道这个东西——但不要了」。失物招领处收的是被遗弃的——不是被遗忘的。

弹幕滚过：
```
所以它們會攻擊來找失物的人
因為他們以為每個人都是遺棄者
```

周仓看着老人：「他们的敌意——是针对遗弃他们的人？」

「是。但——他们被遗弃是过去的事。他们的敌意——不只针对遗弃他们的人——也针对愿意收留他们的人。」

「为什么？」

「因为被收留——意味着被换主人。他们已经被一个人遗弃过一次。他们怕再被遗弃一次。所以如果有人愿意收留他们——他们会先攻击那个人——测试那个人是不是也会遗弃他们。」

周仓懂了。

失物被遗弃过一次——他们怕被再次遗弃，所以会先攻击愿意收留他们的人，测试他们是不是真的会收留。

弹幕飘过：
```
被遺棄過的人最怕再被丟掉
所以先攻擊
怪難過的
```

他需要被失物「找到」——

然后——让失物相信——他不会遗弃他们。

「5 个失物。」周仓说，「我需要被 5 个失物找到。」

「对。」老人说，「但——」

「5 个失物——」

「不一定会——」

「愿意被你收留。」

「如果他们不愿意——」

「他们会——」

「一直——」

「攻击你——」

「直到你——」

「离开副本。」

周仓看着老人。

「我明白了。」

老人说这话的时候目光没有看着周仓。
他看着左手边第三个架子。
那个架子上放着一本很旧的笔记本。
但他——没有提。

周仓没有转移视线。
他盯着老人的眼睛：「你每隔三天会擦一次那个架子。左手边第三个。上面的灰尘厚度跟其他架子不一样。」
老人的手——停了下来。
「你——」老人的声音在抖，「你怎么——」
「你刚才说『三年没人动过这些档案』。」周仓说，「你在说谎。」
老人——沉默了很久。
「你——」他最后说，「不是普通人。」

「开始吧。」

## 2

周仓走出失物招领处。

他站在地铁站的入口。

地铁站——没有人。

但他注意到——他手腕上——出现了一个新的标记。

不是 CH 编号——是 ——一个红点。

红点在缓慢地闪烁。

他打开了系统。

```
[副本提示]

玩家已被 1 件失物标记。
失物正在接近。
请玩家做好准备。
```

失物——在接近。

他环顾四周。

地铁站的入口有一个出口——出口的方向有一个小东西在动。

一个——钱包。

钱包在地板上向他移动。

不是被人推的。

是——自己移动。

像有一个看不见的人——在推它。

钱包——向他移过来。

它——到了他脚边。

它——停了。

它——在等他——捡起来。

周仓蹲下来。

他——伸手——捡起了那个钱包。

钱包——冰凉的。

比他以为的要重。

他打开了钱包。

钱包里——有一张照片。

照片上是——一个年轻女人和一个小孩。

照片背面写着：

「妈妈和小宝 2019 年」

还有——一张纸条。

纸条上写着：

「妈妈——如果你看到这张纸条——小宝很想你。」

周仓看着那张纸条。

他——明白了。

这个钱包——不是被「遗弃」的。

是——被「留下」的。

留它的人——是——孩子的妈妈。

她——离开了。

她留下了钱包钱包里有她和小宝的照片。

她——让小宝——找到她。

但小宝——不知道她去了哪里。

小宝——把钱包——留在了失物招领处。

等她——回来拿。

但她——没有回来。

三年了。

周仓——没有说话。

他——把照片——放回了钱包。

他——把钱包——收进了口袋。

弹幕刷过：

```
他收留了第一件失物
一個媽媽的錢包
孩子留的
等她回來
但她沒有回來
```

系统公告弹出：

```
[失物收留]
失物 1/5：妈妈的钱包
失物状态：已收留
玩家获得：积分 +100
玩家获得：失物主人的「感激」

特殊效果：失物主人（小宝）将通过副本内部消息与你联系。
```

周仓看着这条提示。

小宝——会联系他。

他——需要——回应小宝。

## 3

周仓继续在地铁站里走。

他需要被 5 件失物找到。

他刚收了 1 件。

还有 4 件。

他走到地铁站的候车区。

候车区有几排椅子。一把黑色雨伞靠在椅子的扶手上。

他走过去。手刚伸出——雨伞往旁边滑了一格。

他停。雨伞也停。

他再伸手。雨伞又滑开一格。

它在躲他。

周仓看着它：「你不是要被我收留吗？」

雨伞没有动。

他——换了一种方式。他没有伸手。他只是站在那里，把手放在身側——等。

雨伞犹豫了一会儿——然后——慢慢地——向他倾斜。

它靠了过来。

他拿起雨伞。伞柄上刻着字。

雨伞——比正常的雨伞轻。

伞柄上——刻着字。

「送给爸爸。2020 年父亲节。」

这把雨伞——是孩子送给父亲的。

但父亲——把它——留在了这里。

周仓看着伞柄上的字。

他——没有说话。

他——把雨伞收进了背包。

```
[失物收留]
失物 2/5：孩子的雨伞
失物状态：已收留
玩家获得：积分 +100
玩家获得：失物主人的「怀念」

特殊效果：失物主人（孩子）将通过副本内部消息与你联系。
```

2 件了。

他继续走。

经过一处存档点时，屏幕上闪烁着几行系统日志：

```
[副本后台日志 - 失物档案馆]
清除者当前坐标：档案室 C区
清除者状态：正在扫描档案记录
```

他走到地铁站的检票口。

检票口的机器——是关着的。

但机器的显示屏——在闪。

闪的是——一张图片。

图片上——是一个男人的脸。

他——在笑。

图片的下方——有一行字：

「失物 3：检票员的工牌」

周仓看着那张图片。

这个失物——是检票员的工牌。

工牌上——有检票员的名字。

「李大伟」

李大伟这个检票员他去哪里了？

他的工牌——为什么——在机器的显示屏上？

周仓伸手——他碰了一下显示屏。

显示屏——变黑了。

但工牌——没有出现在他面前。

他需要——让工牌「找到」他。

他环顾四周。

检票口——有一个小门。

门后——是检票员休息室。

他推开门。

里面——有一张桌子。桌子上——有一个工牌。

他走进去。

他——拿起了工牌。

工牌是温的。

像是——刚刚——有人戴着。

他翻看工牌的背面。

背面——有一行手写的字：

「如果我没能回来——把这个工牌交给我的女儿。」

他——需要把这个工牌——交给李大伟的女儿。

```
[失物收留]
失物 3/5：检票员的工牌
失物状态：已收留
玩家获得：积分 +100
玩家获得：失物主人的「托付」

特殊效果：失物主人（李大伟）将通过副本内部消息与你联系。
```

3 件了。

他想起了老人看向那个架子的目光。
左手边——第三个架子。
他——走回调度处。
那个架子上——放着——那本旧笔记本。
他——翻开——
笔记本里——夹着一张——纸条。
纸条上——写着一行字：

「失物——不是被遗弃的。
它们——是在等人。」

周仓看着这行字。他把纸条收进了口袋里。

他正要走出检票员休息室——余光扫到门框内侧有一道新的刮痕。金属表面被什么尖锐的东西划过，露出的金属色还没氧化——刚留下不久。

他不是赏金玩家正在追的唯一的猎物。

他蹲下来，从背包里取出便签，写了一行字贴在刮痕正下方的门缝里：

「B区 第三排 第五盒——不要碰。」

如果赏金玩家真的追到了这里，他们会看到刮痕——会认为周仓紧张了——会顺着假线索冲向B区。

还有 2 件。

他走出检票员休息室。

他看到——地铁站的出口方向——有两个人影。

两个——小孩。

他们——站在出口处。

他们——看着他。

他们——没有说话。

他们——只是——看着他。

他们——手里的——是两个气球。

两个红色的气球。

他们——向周仓——举起了气球。

他们在请求他收留他们的气球。

周仓——走过去。

他——蹲下来。

他——看着两个小孩。

「你们——」他说，「叫什么名字？」

两个小孩——没有回答。

他们只是把气球递向他。

周仓——接过了气球。

他——收留了——两个小孩的失物。

```
[失物收留]
失物 4/5：小孩的气球
失物状态：已收留
玩家获得：积分 +100
玩家获得：失物主人的「依赖」

失物 5/5：小孩的气球
失物状态：已收留
玩家获得：积分 +100
玩家获得：失物主人的「依赖」

[特殊]
玩家收留了 5 件失物。
玩家选择「愿意」的态度。
失物主人——通过副本内部消息与你联系。
```

5 件了。

他完成了通关条件。

在他準備離開的時候——他注意到失物招領處的後門——鎖是壞的。

鎖孔周圍有新的刮痕。

有人——最近——進過這裡。

他看了一眼。沒有停留。走了出去。

系统公告弹出：

```
[副本结算]
副本：失物招领处
结算评价：A
原因：成功收留了 5 件失物，没有让任何失物「失望」
积分奖励：+1000
特殊奖励：获得「失物之友」称号
纪念物品：5 件失物
```

白光——出现。

失物招领处——开始消失。

周仓——被传送回了主系统大厅。

他坐在主系统大厅的休息室里。

他——看着他的系统背包。

里面有 5 件失物。

5 件——被遗弃的失物。

他需要给这些失物找到归宿。

他打开了系统。

他看到了——5 条未读消息。

5 个失物主人——在等他。

他——开始一条一条地回复。
