# Ch.09：结算与消失

## 1

街道上的场景不对劲。

周仓走出便利店大约十米后，注意到了一个明显的事实——太安静了。不是没有人声的安静——是一切声音都消失了。他的脚步声还在，但回声不对。像是踩在一个封闭空间里的地板，不是室外的柏油路面。

他回头看了一眼。

便利店还在他身后。灯光亮着。透过落地玻璃，他能看到许小小站在收银台旁边。她也在看他。

确认了方向。他往前走。

地图显示「CS-0003 注销登记处」在东南方向两百米。他一边走，一边留意周围的建筑。

所有店铺都亮着招牌。药店。面包店。洗衣店。亮得不像正常营业——像是在告诉他：我们都在，但我们不营业。

他走了大约两分钟，到了一个路口。

招牌上写着：

```
CS 档案管理处
```

不是「注销登记处」。是「档案管理处」。

他站在门口。

门没有锁。

他推开门。

## 2

里面不像一个管理处的办公室。

像一间档案室。墙壁上全是文件柜——银色的金属柜子，一排一排地排列着。天花板上的日光灯管发出轻微的嗡鸣声。

房间正中央有一张桌子。桌上放着一台老式电脑——CRT 显示器，白色的外壳已经泛黄。屏幕是黑的。但主机箱上的电源灯亮着——绿色。待机状态。

周仓走到桌前。

电脑旁边放着一叠文件。最上面一张的右上角贴着一张标签：

```
CS-0003 许小小
```

他拿起那份文件。

不是档案——是登记表。

他翻开了第一页：

```
姓名：许小小
编号：CS-0003
状态：存续（异常）
进入副本日期：2023-11-07
最后活动记录：当前
备注：数据残留。未完成注销。
```

他看着最后两行字。

「未完成注销。」

她的注销流程没有完成——所以她被留在这里。不是被故意留下来的——是被忘记的。

他翻到下一页。

第二页上只有几行字——不像正式档案，像手写的备注：

```
CS-0003 在结算时检测到播放器异常。
主系统未确认回收。
建议处理方式：手动注销。
处理人员：待分配。
```

手动注销。待分配。

没有人来。三年了。

他拿着那份文件，站在那里。

背后传来声音——不是脚步声。是纸张翻动的声音。

他转头。

没有人。

但桌面上的文件——从文件柜里——有几份掉在了地上。

像是有人碰掉的。

但房间里没有别人。

周仓蹲下，捡起掉落的文件。

第一份：

```
CS-0001 周远
状态：已注销
注销日期：2023-09-12
备注：主动申请。完整回收。
```

第二份：

```
CS-0002 林东
状态：注销中（异常）
注销日期：—
备注：回收时检测到数据冲突。部分残留。
当前状态：副本内驻留。
```

林东。

值班表上的林东。

不是消失了——是「部分残留」。

和他想的一样。林东还在这个副本里。只是不再是人了。

他把文件放回桌上，看了一眼电脑。

他按下电源键。

显示器慢慢亮起来。不是 Windows 界面——是一个纯文字的数据库终端。

界面很简单。顶部一行字：

```
CS 回收入口 | 当前操作员：未认证
```

下面只有一个输入框：

```
请输入操作员编号：
```

他没有操作员编号。

但他有其他东西。

他输入了 CH-0217。

屏幕闪了一下。

然后跳出一行字：

```
操作员编号无效。
您没有注销权限。
如需手动注销 CS-0003，请使用 CS 系列操作员编号。
```

他盯着屏幕。

CS 系列操作员编号。

他只见过两个 CS 编号：CS-0001 已经注销了。CS-0002 是林东——部分残留。

他想了想，输入了：

```
CS-0002
```

屏幕停顿了两秒。

然后弹出一句话：

```
欢迎，林东。
```

权限通过了。

但林东已经不在了。

周仓看着那行「欢迎，林东」，手指悬在键盘上方。

现在他有了权限。

但他不是林东。

他只是一个用死人的账号打开系统的人。

他点了「注销管理」。

屏幕上展开一份列表：

```
待处理注销：
CS-0003 许小小 | 异常状态 | 上次操作：未完成

CS-0004 （未分配）
CS-0005 （未分配）

已完成注销：
CS-0001 周远 | 2023-09-12 | 完整回收
CS-0006 刘一鸣 | 2024-03-04 | 完整回收
CS-0007 赵雪 | 2024-07-19 | 完整回收
CS-0008 王可 | 2025-01-11 | 完整回收
CS-0009 （未分配）
```

他数了一下。

CS-0001 到 CS-0009。

但 CS-0003 卡住了。CS-0002 也卡住了。其他编号陆续有人进入，有人离开，有人完整回收。

只有许小小和那个只剩下名字的林东——被留在了这里。

他的手指在鼠标上停了一下。

然后他思考了一件事——

如果他把许小小的状态改为「完整回收」——会发生什么？

不是手动注销。

是帮她走完她没有走完的那一步。

他点了编辑。

在 CS-0003 的状态栏里，他选择了「完整回收」。

系统弹出一条确认框：

```
确认将 CS-0003 状态变更为「完整回收」？
此操作不可撤销。
```

他点了确认。

屏幕闪了一下。

然后弹出一行字：

```
状态已更新。
CS-0003 将在本轮副本结算时自动进入回收流程。
请在结算前返回副本起点。
```

他站在那里，看着那行字。

完成了。

他帮许小小做了三年前没有做完的事。

电脑屏幕暗了下去。

档案室的日光灯闪了一下——然后熄灭了。

他站在黑暗中，唯一的光源是从门口透进来的街道灯光。

他转身，走出了档案室。

## 3

他走回便利店的路上，街道在变。

不是物理上的变化——是画面的边缘在慢慢褪色。像是整个世界正在从边缘开始被擦除。路面上出现了一些白色的像素块，然后碎裂，然后消失。

系统提示弹了出来：

```
副本结算倒计时：00:10:00
```

十分钟。

他推开了便利店的门。

风铃响了。

店里——

许小小站在收银台前。

她的状态看起来不太一样了。不是外表上的变化——是一种感觉。她站在那里，不再是那种「漂浮」的感觉了。她踩在地面上，踩得很稳。

她看着他。

「你做了什么？」

周仓没有回答。他走到她面前。

「我给你走完了没走完的那一步。」

许小小的表情——很复杂。像是在哭和笑之间，选了第三种表情。

「你知道那是什么意思吗？」

「知道。」

「那你还做？」

「因为你不应该被留在这里。」

许小小看着他。她的眼睛里有东西——不是泪水。是一种她没有表达出来的情绪。

「谢谢你。」

她说了这两个字。

声音很轻。

但周仓听到了。

然后她手里拿出一个东西。

一盒牛奶。

没有编号的那盒。

「这个给你。」

周仓接过牛奶盒。

「我喝不到这个味道了——」她说，「但你可以。」

他低头看牛奶盒。

然后他感觉到了。

后脑勺——那阵持续的痛——突然松了一下。

不是消失。

是松动。

像是有什么东西在他的颅骨内侧——被敲开了一条缝。

他的右眼前方弹出两条提示。

一条白色：

```
┌──────────────────────────┐
│  副本结算已完成。          │
│  存活时间：7 天            │
│  通关评价：B               │
│  积分奖励：+3,000         │
│  触发隐藏事件：CS-0003     │
└──────────────────────────┘
```

一条红色：

```
⚠ 异常记录：
副本「夜班便利店」结算时检测到数据变动。

CS-0003 状态已变更。
CS-0002 账号被异地访问。

以上事件已上报系统监管。
```

周仓看了一眼红色警告。他没有关掉它。

他看向许小小。

她站在收银台旁边。

脸上带着笑。

在变透明。

从手指开始——像是像素一点一点地散去。

「许小小——」

「我走了。」她说，「这次是真的。」

「你会去哪里？」

她想了想。

「不知道。但至少——不是这里了。」

她笑了一下——这一次，笑到眼睛里了。

然后她完全消失了。

便利店里只剩下周仓一个人。

日光灯嗡嗡响。

关东煮的锅子还在冒热气。

值班表上——

周六的林东已经不见了。

墙上只剩下四行排班：

```
周一：林东
周二：林东
周三：林东
周四：林东
周五：周仓
```

周六的栏位——消失了。

整张表恢复了正常。

像是许小小从来没有存在过一样。

周仓站在那里，手里握着那盒没有编号的牛奶。

他听到风铃响了。

他转头看门口。

没有人。

但视线落在地上的时候——

门口的地垫边缘，有一小片湿漉漉的印记。

像是雨衣上滴下来的水。

红雨衣来过了。在最后的时候。

但没有进来。

它只是来看了一眼。

确认了。

他站在那里，看着那片正在变干的水渍。

然后他的视野被白光覆盖。

```
副本传送中——
```

━━━

【章末钩子】副本结算完成。周仓用林东的权限为 CS-0003 许小小完成了回收流程。她在最终结算时消失——但留给他一盒牛奶。值班表上不再有周六的栏位，像是她从未存在过。但门口地垫上有一片刚滴下的雨衣水渍——红雨衣来看了一眼最后的结果。
