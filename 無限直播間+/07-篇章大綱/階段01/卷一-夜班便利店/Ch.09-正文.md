# Ch.09：失败

## 1

10:30。

白光开始从便利店的边缘出现。货架在消失，收银台在消失，玻璃在消失。

许小小从后场走出来，走到周仓旁边。

「我——」她说，「谢谢你。」

「不用谢。」

她看了一眼门口。林东还站在门口，身体正在像素化。

「林东。」周仓叫了一声。

林东转过头。他的脸在笑。

「我很开心。」他说。

他的身体继续消失。脚没了，腿没了。

「林东！」许小小叫了一声。她的手伸出去，想抓住他——但穿过了他。

他不是实体，只是数据。

林东的脸还在笑：「许小小——我走了。」

他的手伸向许小小，但穿过了她的手。抓不住。

「林东——」许小小的声音哽咽了。

「别哭，我会回来。我会在下个副本等你。」

他的脸开始模糊。

「林东！」

她叫了一声。但他听不到了——他的身体只剩一个轮廓。

然后——那个轮廓重新开始。

像素化停止。他开始反向重组。

他的身体——重新出现。

他的脸——不再是林东的脸。

是「没有脸」。

一张漆黑的、没有五官的脸。

他——又变回红雨衣了。

系统公告弹出：

```
[副本事件]
副本崩溃 - 已中断
原因：CS-0002 林东 数据不稳定 - 无法完成回收
状态：副本结构恢复
```

副本——没有崩溃。

周仓的脑子——空白了一秒。

他刚才做的所有事——解耦许小小、关闭 03:00 锁定机制、和林东对话——都白费了。

林东的「重组」——

意味着他的「人性」——

被副本重新吸收了。

他——不再是林东了。

他是——红雨衣。

但他——不再是以前那个「站着」的红雨衣了。

他——狂暴的红雨衣。

## 2

红雨衣——重新组装完成的红雨衣——站在便利店中央。

它的轮廓——比之前更黑了。

它的「没有脸」里——不再是「空的」。

是——有东西。

是「愤怒」。

是「被骗」。

是「三年希望破灭」的愤怒。

它——转向周仓。

它朝他冲过来。

速度——比他之前见过的任何时候都快。

周仓没有时间反应。

他本能地蹲下——红雨衣从他头顶掠过，它的手抓住了货架——金属货架被它折弯了。

弹幕炸裂：

```
它狂暴了
它变强了
他彻底激怒了它
```

周仓滚到了收银台后面。

红雨衣——转向他。

它没有再冲。

它开始走。

一步。两步。

它在逼近。

周仓的脑子在转他需要逃跑但他需要想清楚为什么林东的回收失败了？

他说「我选真的机会」——但系统说「数据不稳定」——

他——反悔了？

还是——他——在最后一刻——他「不想走了」？

周仓想起了林东的话：

「我会——记得你——们——」

「我会——回来——」

他在最后一刻——想起来了什么。

他想起了——副本给他的「假希望」。

他想起了——他「站了三年」的意义。

他想起了——他——不想让这一切消失。

他——在最后一刻——自己反悔了。

他不想被回收。

他想继续站着。

他想继续——等。

红雨衣林东三年来第一次不想离开了。

因为——离开意味着——失去「等待」的意义。

他等了三年。

他不想承认——他等的是——假希望。

弹幕飘过：

```
他反悔了
他不想走了
三年的等待——他不想让它没有意义
他宁愿被副本吞噬——也不想承认自己被骗了
```

周仓看着逼近的红雨衣。

他明白了。

林东不是「被副本强行留下」的。

他是「自己选择留下」的。

他自己——不想走。

他不是不能走。

是——不想。

周仓站在那里。红雨衣在他面前两步。林东在看着他，没有脸的脸里没有表情，只有愤怒。

「林东。」周仓叫它。

红雨衣没有停下。它继续走——不想听他说话了。

「林东！」周仓加大了声音。

红雨衣停了。它站在他面前，距离不到一米。

「你骗我。」它的声音愤怒、流畅、扭曲。

「我没有骗你。」

「你说让我知道真相——」

「是。」

「但真相是我三年白等。」

「是。」

「那为什么——」

「因为白等不是坏事。」

红雨衣愣了一下。

「白等是好事？」

「白等是你活过的证明。」

「什么意思？」

「你站在门口三年，你以为你在等机会——其实你在等许小小。」

红雨衣完全停了。

「你每天站在门口——不是因为你相信副本会回收你——是因为你想守着她。」

红雨衣的「没有脸」动了一下。

「你说——我想陪着她？」它的声音变了，愤怒消失，变成了颤抖。

「你不想离开，不是因为你在等假希望——是因为你想陪着她。」

红雨衣退后一步。姿态从愤怒变成了震惊。

「但你告诉我他们没有骂我。他们只是希望我承认错了。」

「你说——我想陪着她？」

「是。」

「但她已经走了——」
红雨衣转向便利店后面。

许小小还站在那里。没有消失。

「林东，我还在。」

红雨衣不动了。

「你还没走？」它的声音变得非常轻，带着颤抖。

「我没有走。我不走。」

红雨衣跪了下来。林东第二次在许小小面前跪下。

但这次不是因为认错。是因为他不想离开。

「你为什么不走？」他的声音里带着哭声。

「因为我也想陪着你。」

红雨衣整个身体开始颤动。林东在哭——不是没有脸的哭，是一个完整的人的哭。

「我三年——」他的声音破碎了，「我以为你在等我。我以为只要我站着你就不会消失。我不敢走——我怕我走了你就会走。」

许小小走过去，蹲下来，抱住了红雨衣——抱住了林东。

「我不会走。你可以走——你走了我也不会走。我会等你，一直等你。但你等的不是假希望——你等的是我。」

红雨衣整个身体开始颤动。林东在哭——三年来第一次在许小小面前，作为一个有脸的人，彻底地哭。

他终于可以哭了。

## 3

11:00。

便利店没有崩溃。但它不一样了。

周仓站在收银台后面，看着许小小抱着林东。林东不再是红雨衣了。他恢复了人的样子，但身体是透明的——数据已经和副本融合了。他不能完全活过来，但可以待在副本里。不是站着等死——是陪着许小小。

系统公告弹出：

```
[副本事件]
CS-0002 林东 状态：数据稳定 - 已恢复人形
CS-0003 许小小 状态：未解耦 - 重新耦合
副本核心支柱：恢复中
```

副本没有崩溃。但林东不再需要维持副本，许小小不再需要被解耦。他们只是在副本里——自由了。

周仓站在收银台后面。他做的一切——解耦、关闭 03:00、强制崩溃——都失败了。但成功了。副本没有崩溃，但林东不再需要维持副本了。许小小不再被卡在副本里了。他们在副本里——却自由了。

这是另一种成功。

弹幕刷过：

```
但——那个人——要来了吧
清除者
他——知道吗
```

周仓靠在收银台上。

他很累。

他——从醒来到现在——没有合过眼。

他——可以休息一下了。

然后——稳定了。

系统公告弹出：

```
[副本事件]
副本「夜班便利店」进入稳定期
触发条件已改变
新主线任务已解锁
```

新主线任务——周仓点开：

```
[新] 完成便利店副本
     状态：进行中
     新通关条件：
     1. 与 CS-0002 林东 完成对话
     2. 与 CS-0003 许小小 完成对话
     3. 选择：离开副本 / 继续陪伴
     4. 主系统大厅开放
```

选择。

周仓看着这个选项。他可以离开，可以留下——他可以选。

许小小走过来，看着他。

「你——」她说，「可以走了。」

「我知道。」

「你——不需要——继续待在这里——」

「我知道。」

「你——」她说，「有 1000 多人在看你——」

「我知道。」

「你——」她说，「应该——去下个副本——」

「我知道。」

「但——」她说，「如果你——想——留下——」

「我——」她说，「会——很开心——」

周仓看着她。

他想了三秒。

「我要走。」

许小小点了点头。她没有失望——早就知道他要走了。

「但我会回来看你们。」

许小小笑了。

「好。」她说，「我——等你——」

林东——也走过来。

他看着周仓。

「我——」他说，「也——等你——」

周仓看着他们两个。

他点开了「离开副本」的选项。

系统公告弹出：

```
副本结算中——
结算评价：B
积分奖励：+5000
——检测到异常数据耦合——
——玩家 P-0217 周仓 与 CS-0002 林东 数据链路异常——
——判定：异常关联者——
——数据债务：3000——
——清除者将在 72 小时后出动——
```

周仓——看着这条公告。
异常关联者。
清除者。
他——不知道这些——是什么。
但他知道这不是好事。

白光——开始出现。

「再见。」周仓说。

「再见。」许小小说。

「再见。」林东说。

白光——吞没了周仓。
