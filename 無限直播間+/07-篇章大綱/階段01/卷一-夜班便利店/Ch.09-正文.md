# Ch.09：失败

## 1

10:30。

白光开始从便利店的边缘出现。

货架在消失。收银台在消失。玻璃在消失。

许小小从后场走出来。她的脸色比之前好。

她走到周仓旁边。

「我——」她说，「谢谢你。」

「不用谢。」

「林东——」她看了一眼门口。

红雨衣——林东——还站在门口。

他的身体正在像素化。

他即将消失。

「林东。」周仓叫了他一声。

林东转过头。

他的脸——正常的人脸——在笑。

「我——」他说，「很——开心——」

他的身体继续像素化。

「我——」他说，「会——记得——你——们——」

他的脚——消失了。

他的腿——消失了。

他的腰——

「林东！」许小小叫了一声。

她的手——

伸了出去。

她想抓住他。

但她的手——

穿过了他的身体。

他——

不是实体。

他只是数据。

数据——

抓不住。

林东的身体继续消失。

他的脸——

还在笑。

「许——小小——」他说，「我——走了——」

他的手——

伸向许小小的方向。

但他的手——

也穿过了许小小的手。

他抓不住她。

她也抓不住他。

「林东——」许小小的声音里有了哽咽。

「别——哭——」林东说，「我——会——回来——」

他的身体——

继续消失。

「我会——在——下个副本——等你——」

他的脸——

开始模糊。

「林东！」

许小小叫了一声。

但他已经听不到了。

他的身体——

只剩一个轮廓。

一个——红雨衣的轮廓。

然后——

那个轮廓——

重新开始。

像素化——

停止。

他开始反向——

「重组」。

他的脚——重新出现。

他的腿——重新出现。

他的身体——重新出现。

他的脸——

不再是林东的脸。

是「没有脸」。

一张漆黑的、没有五官的脸。

他——

又变回红雨衣了。

系统公告弹出：

```
[副本事件]
副本崩溃 - 已中断
原因：CS-0002 林东 数据不稳定 - 无法完成回收
状态：副本结构恢复
```

副本——

没有崩溃。

周仓的脑子——

空白了一秒。

他刚才做的所有事——解耦许小小——关闭 03:00 锁定机制——和林东对话——

都白费了。

林东的「重组」——

意味着他的「人性」——

被副本重新吸收了。

他——

不再是林东了。

他是——

红雨衣。

但他——

不再是以前那个「站着」的红雨衣了。

他——

狂暴的红雨衣。

## 2

红雨衣——重新组装完成的红雨衣——

站在便利店中央。

它的轮廓——

比之前更黑了。

它的「没有脸」里——

不再是「空的」。

是——

有东西。

是「愤怒」。

是「被骗」。

是「三年希望破灭」的愤怒。

它——

转向周仓。

它朝他冲过来。

速度——

比他之前见过的任何时候都快。

周仓没有时间反应。

他本能地蹲下——

红雨衣从他头顶掠过——

它的手——

抓住了货架——

金属货架——

被它——

折弯了。

弹幕炸裂：

```
它狂暴了
它变强了
他彻底激怒了它
```

周仓滚到了收银台后面。

红雨衣——

转向他。

它没有再冲。

它开始走。

一步。两步。

它在逼近。

周仓的脑子在转——

他需要逃跑——但他需要想清楚——

为什么林东的回收——失败了？

他说「我选真的机会」——但系统说「数据不稳定」——

他——

反悔了？

还是——

他——

在最后一刻——

他「不想走了」？

周仓想起了林东的话：

「我——会——记得——你——们——」

「我——会——回来——」

他在最后一刻——

想起来了什么。

他想起了——

副本给他的「假希望」。

他想起了——

他「站了三年」的意义。

他想起了——

他——

不想让这一切消失。

他——

在最后一刻——

自己反悔了。

他不想被回收。

他想继续站着。

他想继续——

等。

红雨衣——林东——

三年来第一次——

不想离开了。

因为——

离开意味着——

失去「等待」的意义。

他等了三年。

他不想承认——

他等的是——假希望。

弹幕飘过：

```
他反悔了
他不想走了
三年的等待——他不想让它没有意义
他宁愿被副本吞噬——也不想承认自己被骗了
```

周仓看着逼近的红雨衣。

他明白了。

林东不是「被副本强行留下」的。

他是「自己选择留下」的。

他自己——

不想走。

他不是不能走。

是——不想。

周仓站在那里。

红雨衣在他面前两步。

它——林东——

在看着他。

它的「没有脸」里——

没有表情。

只有愤怒。

「林东。」周仓叫它。

红雨衣——

没有停下。

它继续走。

它——

不想听他说话了。

「林东！」周仓加大了声音。

红雨衣——

停了。

它站在他面前。

距离——

不到一米。

「你——」它的声音传来——不是一字一顿了——是愤怒的、流畅的、但扭曲的——「你——骗我——」

「我没有骗你。」

「你——说——让我——知道——真相——」

「是。」

「但——真相——」它的声音变得尖锐，「真相是——我——三年——白等——」

「是。」

「那——为什么——」

「因为——」周仓说，「白等不是坏事。」

红雨衣——

愣了一下。

「白等——」它说，「是——好事——？」

「白等——」周仓说，「是你——活过的证明。」

「什么意思——」

「你——」周仓说，「站在门口——三年——你以为你在等机会——」

「但其实——」

「你在等——许小小。」

红雨衣——

完全停了。

「你——」周仓说，「每天站在门口——不是因为你相信副本会回收你——」

「是因为——」

「你想——守着她。」

红雨衣的「没有脸」——

动了一下。

「你——」它的声音变了——愤怒消失了——变成了——颤抖，「你——说什么——」

「我说——」周仓说，「你——不想离开——不是——因为你想等假希望——」

「是因为——」

「你想——陪着她。」

红雨衣——

退后了一步。

它的姿态——

变了。

它不再是「愤怒的」姿态。

它是——

「震惊」的姿态。

「你——」它的声音里有了——

不是愤怒——

是——

别的什么。

「你——」它说，「你——说——我——想——陪——着她——」

「是。」

「但——」它说，「她——已经——走了——」

「她——」

红雨衣——

转向便利店后面。

许小小——

还站在那里。

她——

没有消失。

她——

还在。

红雨衣看着她。

她看着红雨衣。

他们——

对视。

许小小——

她的眼睛里有泪。

「林东。」她说，「我——还在。」

红雨衣——

不动了。

「你——」它的声音——变成了——非常轻的——颤抖的——

「你——还没——走——？」

「我没有走。」许小小说，「我——不走。」

「你——」红雨衣——它——

它——

跪了下来。

它——林东——

第二次——

在许小小面前跪下。

但这次——

不是因为认错。

是因为——

他——

不想离开。

「你——」它的声音里有了——哭声——

「你——为什么——不走——？」

「因为——」许小小说，「我——也想——陪着你。」

红雨衣——

整个身体——

开始颤动。

它——林东——

在哭。

不是「没有脸」的哭。

是——一个完整的人的哭。

「我——」它的声音变得——破碎——

「我——三年——」

「我——以为——你在——等我——」

「我——以为——只要我——站着——你就——不会——消失——」

「我——」

「我——」

「我——不敢——走——」

「因为——」

「我——怕——我——走了——你——就——会——走——」

许小小——

走过去。

她——

蹲下来。

她——

抱住了红雨衣——

抱住了林东。

「林东。」她说，「我——不会——走。」

「你——可以——走——」

「你——走了——我——也——不会——走——」

「我——会——等你——」

「我——会——一直——等你——」

「但——」

「你——必须——知道——」

「你——等的——不是——假希望——」

「你——等的是——我——」

红雨衣——

整个身体——

开始颤动。

它——林东——

在哭。

三年来——

他第一次——

在许小小面前——

彻底地哭。

不是「没有脸」的哭。

是——

一个完整的人的哭。

他有脸了。

他的「没有脸」——

消失了。

他——

重新有了脸。

他在哭。

他——

终于——

可以哭了。

## 3

11:00。

便利店——

没有崩潰。

但它——

不一样了。

周仓站在收银台后面，看着许小小抱着林东。

林东——不再是红雨衣了。

他——

恢复了人的样子。

但他的身体——

是透明的。

他的数据——

已经和副本融合了。

他不能完全「活过来」。

但他——

可以「待」在副本里。

不是「站着等死」。

是——

「陪着许小小」。

系统公告弹出：

```
[副本事件]
CS-0002 林东 状态：数据稳定 - 已恢复人形
CS-0003 许小小 状态：未解耦 - 重新耦合
副本核心支柱：恢复中
```

副本——

没有崩潰。

但它——

也不一样了。

林东——不再需要「维持副本」。

他只是——「在副本里」。

许小小——不再需要「被解耦」。

她只是——「在副本里」。

副本的三个支柱——

不再是「支柱」。

只是——

「两个人」。

他们在副本里。

他们——

自由了。

周仓站在收银台后面。

他看着这一切。

他做的一切——

解耦——关闭 03:00——强制崩潰——

都「失败」了。

但——

成功了。

副本没有崩潰。

但林东——不再需要「维持副本」了。

许小小——不再「被卡」在副本里了。

他们——

在副本里。

但他们——

自由了。

这是——

另一种成功。

弹幕刷过：

```
副本没有崩潰
但他们自由了
另一种成功
```

周仓靠在收银台上。

他很累。

他——

三天三夜没有睡了。

他——

可以休息一下了。

系统公告弹出：

```
[副本事件]
副本「夜班便利店」进入稳定期
触发条件已改变
新主线任务已解锁
```

新主线任务——

周仓点开：

```
[新] 完成便利店副本
     状态：进行中
     新通关条件：
     1. 与 CS-0002 林东 完成对话
     2. 与 CS-0003 许小小 完成对话
     3. 选择：离开副本 / 继续陪伴
     4. 主系统大厅开放
```

选择。

周仓看着这个选项。

他——

可以离开。

他——

可以留下。

他——

可以——

选择。

许小小走过来。

她看着周仓。

「你——」她说，「可以走了。」

「我知道。」

「你——不需要——继续待在这里——」

「我知道。」

「你——」她说，「有 1000 多人在看你——」

「我知道。」

「你——」她说，「应该——去下个副本——」

「我知道。」

「但——」她说，「如果你——想——留下——」

「我——」她说，「会——很开心——」

周仓看着她。

他想了三秒。

「我要走。」

许小小点了点头。

她没有失望。

她——

早就知道他要走了。

「但——」周仓说，「我会回来看你们。」

许小小——

笑了。

「好。」她说，「我——等你——」

林东——

也走过来。

他看着周仓。

「我——」他说，「也——等你——」

周仓看着他们两个。

他点开了「离开副本」的选项。

系统公告弹出：

```
副本结算中——
结算评价：A
特殊评价：改变了副本的运行方式
积分奖励：+5000
纪念物品：林东的工牌（已获得）
```

白光——

开始出现。

「再见。」周仓说。

「再见。」许小小说。

「再见。」林东说。

白光——

吞没了周仓。
