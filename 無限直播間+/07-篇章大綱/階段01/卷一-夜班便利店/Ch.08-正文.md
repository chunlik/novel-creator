# Ch.08：第二次

## 1

08:00。

夜班结束。

便利店进入白班时段。

规则 #1「请勿直视门外」失效了。规则 #2 冰柜限制失效了。规则 #6 观众限制失效了——白天没有观众。

但红雨衣——还在门口。

它没有因为夜班结束而离开。

周仓站在收银台后面，看着玻璃外的红雨衣。

它在门口站了三秒。

然后——它推门进来了。

风铃响了。

周仓没有直视它。

但他做好了准备。

红雨衣走进来。

一步。两步。

它没有往货架区走。

它直接走向收银台。

它站在他面前。

它没有伸手。

它只是站着。

然后——它开口了。

「周——仓。」

一字一顿。

但这次不是录音了。

是它——林东——在叫他的名字。

周仓的呼吸停了一秒。

它在叫他的名字。

「你——」它的声音继续，「记得——我吗？」

「记得。」周仓说。

「我——」它停了一下，「不记得你了。」

「你不记得我？」

「我只——记得三年前——」它说，「三年前——我进副本——」

它的声音变了。

「三年前——我设计的副本——我自己测试——」

「然后我死了。」

「然后我变成这样。」

它的「没有脸」里——闪过了一道光。

「我——记不清——了。」

它的声音里有了颤抖。

「我只——记得一件事——」

「什么事？」

「我不——应该进副本。」

它说这句话的时候——它的「没有脸」里——出现了——一张脸。

不是笑。

是哭。

一张正在哭的脸。

「我——」它的声音变了，「对不起——许——小——小——」

它在道歉。

对着空气道歉。

对着许小小的方向道歉。

周仓站在收银台后面。

他没有说话。

他看着红雨衣在哭。

许小小从后场走出来。

她听到了红雨衣说的话。

她站在货架区，看着红雨衣。

她没有说话。

红雨衣继续哭。

它的身体在抖。

它的雨衣在抖。

它的「没有脸」里那张哭脸——越来越清晰。

然后——它停了。

它抬起头。

它看着许小小。

它的「没有脸」里——那张哭脸——和许小小的脸——对上了。

「你——」它说，「还在——？」

许小小没有回答。

她只是站在那里。

「你——」它说，「等了——我多久？」

许小小开口了。

「三年。」

红雨衣的「没有脸」——

没有变化。

但它的身体——跪了下来。

它跪在收银台前。

它林东三年来第一次不是站着。

是跪着。

它在——认错。

弹幕刷过：

```
他跪了
红雨衣跪了
林东跪了
他在许小小面前跪了
```

## 2

08:30。

红雨衣——林东——在收银台前跪了五分钟。

然后它站起来。

它没有说话。

它转身——走回门口。

它推开门。

但这次——它没有走开。

它站在门口。

像以前一样站着。

但它的姿态——变了。

以前它是「直直地站着」——像木桩。

现在它是「靠着门框站着」——像是一个累了的人。

它在休息。

它不在「维持」副本。

它在休息。

周仓看着这个变化。

他在交接本上写下：

```
08:30 - 林东第一次在许小小面前跪下。
        - 他在道歉。
        - 他说"对不起许小小"。
        - 他说"我不应该进副本"。
        - 他现在站在门口，姿态变了。
        - 他在休息。
        - 他不再维持副本。
```

他放下笔。

许小小走过来。

她站在他旁边。

「他——」她开口，「他刚才说——'不记得我了'——」

「嗯。」

「他不记得你了——意味着——」

她停了一下。

「意味着他的记忆在被副本吞噬。」

周仓看着她。

「副本会吞噬失败者的记忆？」

「会。」许小小说，「副本需要能量。失败者的记忆——是副本的能量来源之一。」

「所以他每天站在门口——不只是为了维持副本——也是为了让副本不要吞噬他的记忆？」

「是。」

「但他今天跪下了。」周仓说，「他今天承认了错误。他放弃了维持副本。」

「所以——」

「所以——」许小小看着他，「副本会开始吞噬他。」

弹幕飘过：

```
他放弃了
副本开始吞噬他
他最后的时间不多了
```

周仓看了一眼门口。

红雨衣还站在那里。

但它的轮廓——开始变得有点模糊。

像是被一层雾遮住。

它在变淡。

「他会——怎样？」

「如果副本完全吞噬他——」许小小说，「他会变成副本的'内容物'。不是 NPC。是被副本'消化'的素材。」

「那我们还能救他吗？」

许小小看着他。

「可以。」她说，「但需要——」

「需要什么？」

「需要他——被回收。」

「怎么回收？」

「我告诉过你——」她说，「他不能被普通流程回收。强行回收——会让副本崩溃。」

「但副本崩溃——」周仓说，「我们也会被一起带走。」

「是。」

「所以——」

「所以——」许小小看着他，「你要选。」

「选什么？」

「选——」她说，「让副本崩溃——带走我们——让林东被回收。」

「或者？」

「或者——」她说，「让副本继续——让林东被慢慢吞噬——但你和我能活下去。」

周仓看着她。

「你会选哪个？」

许小小笑了。

那种笑——不是快乐的笑。

是「我早就知道你会问」的笑。

「我——」她说，「已经选了。」

「你选了哪个？」

「我选了——」她说，「让你选。」

「为什么？」

「因为——」她说，「三年了。我在这里待了三年。我没办法再选一次了。」

「所以你让我选？」

「是。」她说，「你来这里。你比我更有能力选。」

周仓看着门口的红雨衣。

他的脑子里转过很多事。

林东——一个程序員——自己设计的副本，自己进去测试，死了，变成红雨衣，每天站在门口三年。他为什么每天站着？

他在维持副本。

但他也是被副本困住的人。

他维持副本——是副本要求他维持——还是他自己想维持？

如果他自己想维持——那他想维持的是什么？

如果副本要求他维持——那副本需要他维持什么？

周仓想了一会儿。

他开口了。

「许小小。」

「嗯。」

「林东维持副本——副本会给他什么？」

许小小看着他。

「给他——」她说，「一个'回家的机会'。」

「什么意思？」

「副本告诉林东——'只要你站在门口，你就有机会被回收'。他每天站着——是因为他相信他还有机会。」

「但他——其实没有机会？」

「对。」许小小说，「副本不会回收他。他每天站着——是副本给他的'假希望'。」

「所以——」

「所以——」她看着他，「如果你让副本崩溃——他不会失去任何东西。他本来就没有机会。」

「但如果副本继续——」

「他会继续站在门口——继续相信——继续等——」

「等到被副本完全吞噬。」

「是。」

周仓沉默了。

他看了一眼门口。

红雨衣还站在那里。

它——林东——还在等一个「被回收的机会」。

但这个机会不存在。

他每天站着——是因为他相信「机会」。

如果周仓让副本继续——林东会继续站着，继续被副本骗，继续被吞噬。

如果周仓让副本崩溃——林东会失去「假希望」——但他会被强制回收——会被「真的」带走。

「许小小。」

「嗯。」

「如果副本崩溃——林东会被强制回收——不是死亡？」

「不是。」许小小说，「他会进入下一个副本。但是——没有'假希望'了。」

「他会——」周仓说，「知道真相？」

「会。」

「他会知道——他本来没有机会？」

「会。」

周仓看着门口。

他做了一个决定。

「我要让副本崩溃。」

许小小看着他。

「你确定？」

「确定。」

「为什么？」

「因为——」他说，「他应该知道真相。」

「但他会失望。」

「会。」周仓说，「但失望比假希望好。」

许小小看着他。

她的眼睛里有东西在闪。

「三年了。」她说，「你是第一个这样说的人。」

弹幕刷过：

```
他选了
他选了让副本崩溃
他选了让林东知道真相
他选了带所有人离开
```
