# Ch.08：第二次

## 1

08:00。

夜班结束。

便利店进入白班时段。

规则 #1「请勿直视门外」失效了。规则 #2 冰柜限制失效了。规则 #6 观众限制失效了。

但红雨衣——

还在门口。

它没有因为夜班结束而离开。

周仓站在收银台后面，看着玻璃外的红雨衣。

它在门口站了三秒。

然后——

它推门进来了。

风铃响了。

周仓没有直视它。

但他做好了准备。

红雨衣走进来。

一步。两步。

它没有往货架区走。

它直接走向收银台。

它站在他面前。

它没有伸手。

它只是站着。

然后——

它开口了。

「周——仓。」

一字一顿。

但这次不是录音了。

是它——林东——在叫他的名字。

周仓的呼吸停了一秒。

它在叫他的名字。

「你——」它的声音继续，「记得——我吗？」

「记得。」周仓说。

「我——」它停了一下，「不——记得——你了。」

「你不记得我？」

「我——只——记得——三年前——」它说，「三年前——我进副本——」

它的声音变了。

「三年前——我设计的副本——我自己测试——」

「然后我死了。」

「然后我变成这样。」

它的「没有脸」里——闪过了一道光。

「我——记不清——了。」

它的声音里有了颤抖。

「我——只——记得——一件事——」

「什么事？」

「我——不——应该——进副本。」

它说这句话的时候——

它的「没有脸」里——

出现了——

一张脸。

不是笑。

是哭。

一张正在哭的脸。

「我——」它的声音变了，「对不起——许——小——小——」

它在道歉。

对着空气道歉。

对着许小小的方向道歉。

周仓站在收银台后面。

他没有说话。

他看着红雨衣在哭。

许小小从后场走出来。

她听到了红雨衣说的话。

她站在货架区，看着红雨衣。

她没有说话。

红雨衣继续哭。

它的身体在抖。

它的雨衣在抖。

它的「没有脸」里那张哭脸——

越来越清晰。

然后——

它停了。

它抬起头。

它看着许小小。

它的「没有脸」里——

那张哭脸——

和许小小的脸——

对上了。

「你——」它说，「还在——？」

许小小没有回答。

她只是站在那里。

「你——」它说，「等——了——我——多久——？」

许小小开口了。

「三年。」

红雨衣的「没有脸」——

没有变化。

但它的身体——

跪了下来。

它跪在收银台前。

它——林东——

三年来第一次——

不是站着。

是跪着。

它在——

认错。

弹幕刷过：

```
他跪了
红雨衣跪了
林东跪了
他在许小小面前跪了
```

## 2

08:30。

红雨衣——林东——在收银台前跪了五分钟。

然后它站起来。

它没有说话。

它转身——

走回门口。

它推开门。

但这次——它没有走开。

它站在门口。

像以前一样站着。

但它的姿态——变了。

以前它是「直直地站着」——像木桩。

现在它是「靠着门框站着」——像是一个累了的人。

它在休息。

它不在「维持」副本。

它在休息。

周仓看着这个变化。

他在交接本上写下：

```
08:30 - 林东第一次在许小小面前跪下。
        - 他在道歉。
        - 他说"对不起许小小"。
        - 他说"我不应该进副本"。
        - 他现在站在门口，姿态变了。
        - 他在休息。
        - 他不再维持副本。
```

他放下笔。

许小小走过来。

她站在他旁边。

「他——」她开口，「他刚才说——'不记得我了'——」

「嗯。」

「他不记得你了——意味着——」

她停了一下。

「意味着他的记忆在被副本吞噬。」

周仓看着她。

「副本会吞噬失败者的记忆？」

「会。」许小小说，「副本需要能量。失败者的记忆——是副本的能量来源之一。」

「所以他每天站在门口——不只是为了维持副本——也是为了让副本不要吞噬他的记忆？」

「是。」

「但他今天跪下了。」周仓说，「他今天承认了错误。他放弃了维持副本。」

「所以——」

「所以——」许小小看着他，「副本会开始吞噬他。」

弹幕飘过：

```
他放弃了
副本开始吞噬他
他最后的时间不多了
```

周仓看了一眼门口。

红雨衣还站在那里。

但它的轮廓——

开始变得有点模糊。

像是被一层雾遮住。

它在变淡。

「他会——怎样？」

「如果副本完全吞噬他——」许小小说，「他会变成副本的'内容物'。不是 NPC。是被副本'消化'的素材。」

「那我们还能救他吗？」

许小小看着他。

「可以。」她说，「但需要——」

「需要什么？」

「需要他——被回收。」

「怎么回收？」

「我告诉过你——」她说，「他不能被普通流程回收。强行回收——会让副本崩溃。」

「但副本崩溃——」周仓说，「我们也会被一起带走。」

「是。」

「所以——」

「所以——」许小小看着他，「你要选。」

「选什么？」

「选——」她说，「让副本崩溃——带走我们——让林东被回收。」

「或者？」

「或者——」她说，「让副本继续——让林东被慢慢吞噬——但你和我能活下去。」

周仓看着她。

「你会选哪个？」

许小小笑了。

那种笑——不是快乐的笑。

是「我早就知道你会问」的笑。

「我——」她说，「已经选了。」

「你选了哪个？」

「我选了——」她说，「让你选。」

「为什么？」

「因为——」她说，「三年了。我在这里待了三年。我没办法再选一次了。」

「所以你让我选？」

「是。」她说，「你来这里。你比我更有能力选。」

周仓看着门口的红雨衣。

他的脑子里转过很多事。

林东——一个程序員——自己设计的副本——自己进去测试——死了——变成红雨衣——每天站在门口——三年——

他为什么每天站着？

他在维持副本。

但他也是被副本困住的人。

他维持副本——是副本要求他维持——还是他自己想维持？

如果他自己想维持——那他想维持的是什么？

如果副本要求他维持——那副本需要他维持什么？

周仓想了一会儿。

他开口了。

「许小小。」

「嗯。」

「林东维持副本——副本会给他什么？」

许小小看着他。

「给他——」她说，「一个'回家的机会'。」

「什么意思？」

「副本告诉林东——'只要你站在门口，你就有机会被回收'。他每天站着——是因为他相信他还有机会。」

「但他——其实没有机会？」

「对。」许小小说，「副本不会回收他。他每天站着——是副本给他的'假希望'。」

「所以——」

「所以——」她看着他，「如果你让副本崩溃——他不会失去任何东西。他本来就没有机会。」

「但如果副本继续——」

「他会继续站在门口——继续相信——继续等——」

「等到被副本完全吞噬。」

「是。」

周仓沉默了。

他看了一眼门口。

红雨衣还站在那里。

它——林东——还在等一个「被回收的机会」。

但这个机会不存在。

他每天站着——是因为他相信「机会」。

如果周仓让副本继续——林东会继续站着——继续被副本骗——继续被吞噬。

如果周仓让副本崩溃——林东会失去「假希望」——但他会被强制回收——会被「真的」带走。

「许小小。」

「嗯。」

「如果副本崩溃——林东会被强制回收——他会被传送到主系统大厅吗？」

「是。」

「那他会——」

「会被——等待下一个副本。」

「不是死亡？」

「不是。」

「他会被传送到主系统大厅——然后等下一个副本？」

「是。」许小小说，「但是——没有'假希望'了。」

「他会——」周仓说，「知道真相？」

「会。」

「他会知道——他本来没有机会？」

「会。」

周仓看着门口。

他做了一个决定。

「我要让副本崩溃。」

许小小看着他。

「你确定？」

「确定。」

「为什么？」

「因为——」他说，「他应该知道真相。」

「但他会失望。」

「会。」周仓说，「但失望比假希望好。」

许小小看着他。

她的眼睛里有东西在闪。

「三年了。」她说，「你是第一个这样说的人。」

弹幕刷过：

```
他选了
他选了让副本崩溃
他选了让林东知道真相
他选了带所有人离开
```

## 3

09:00。

周仓开始准备让副本崩溃。

他打开系统。

```
[主线] 强制副本崩溃
      触发条件：完成以下所有条件
      1. 获得林东的临时权限
      2. 找到副本的核心支柱
      3. 移除所有核心支柱
      4. 触发崩溃

进度：
✓ 1. 获得林东的临时权限（剩余时间：23 小时）
? 2. 找到副本的核心支柱
? 3. 移除所有核心支柱
? 4. 触发崩溃
```

他需要找到副本的核心支柱。

他之前在 CS 档案管理处看到了林东的记录——林东是副本的核心支柱之一。

但「之一」意味着还有其他支柱。

他问了一次系统。

```
┌──────────────────────┐
│  系统：              │
│  副本核心支柱：      │
│  CS-0002 林东        │
│  ？？？              │
│  ？？？              │
└──────────────────────┘
```

三个支柱。

他知道一个。还有两个未知。

他开始思考。

第一个支柱是林东——他每天站在门口——维持副本存在。

第二个支柱可能是——冰柜？冰柜在 03:00 停止运行——它可能是某种「容器」。

第三个支柱可能是——许小小？她被卡在副本里三年——她的数据与副本结构耦合。

如果三个支柱是：林东、冰柜、许小小。

那他要做的——

是让林东不再维持副本——让冰柜不再作为容器——让许小小不再被耦合。

但许小小是玩家——他不能让她「消失」。

他想了一下。

他打开商城。

```
┌──────────────────────────────────┐
│  商城                            │
│──────────────────────────────────│
│  道具：                          │
│  数据解耦胶囊     500 积分       │
│  说明：将玩家与副本结构解耦      │
│  副作用：解耦后玩家不再受副本保护│
│                                  │
│  状态：                          │
│  紧急护盾         500 积分       │
│  状态：暂时缺货                 │
│                                  │
│  情报：                          │
│  副本结构图       300 积分       │
│  状态：可购买                   │
└──────────────────────────────────┘
```

数据解耦胶囊——500 积分。

副本结构图——300 积分。

他现在 250 积分。

不够。

他需要更多积分。

他看了一眼任务列表——

```
[新] 在便利店内存活 48 小时
     进度：09:23:45 / 48:00:00
     奖励：1000 积分
```

48 小时——还有 38 个小时。

太久了。

他看了一下另一个任务——

```
[新] 完成 5 次与 NPC 的深度对话
     进度：3/5
     奖励：300 积分
```

他已经完成了 3 次。和许小小对话了 3 次。

他可以再完成 2 次——再赚 300 积分。

但还不够。

他需要 500 积分来买数据解耦胶囊。

300 积分来买副本结构图。

一共 800 积分。

他现在 250。

差 550。

他可以：

A. 等 38 小时拿 1000 积分（太慢）
B. 完成 5 次深度对话拿 300（还差 250）
C. 找到其他赚钱方式

他想了一下。

他打开了另一个选项——

```
[隐藏] 放弃之前的「主动降低副本关注度」操作
      效果：副本关注度回到「高」
      后果：红雨衣会更频繁地出现
      奖励：500 积分
```

放弃降低关注度——让红雨衣更频繁地出现——赚 500 积分。

但红雨衣更频繁地出现——意味着更危险。

他犹豫了一下。

他点下了「放弃」。

```
[已放弃] 主动降低副本关注度
[获得] 500 积分
[当前] 750 积分
[副本关注度] 高
```

他现在的积分：750。

他买了副本结构图——300。

```
[已购买] 副本结构图 -300 积分
[剩余] 450 积分
```

结构图在他的视野里展开。

```
[副本结构图]

便利店主结构：
  货架区：5 排货架，存放 234 件商品
  收银区：1 台收银台，1 个交接本
  冰柜：1 个冰柜，1 个隐藏保鲜盒
  后场：1 个监控室，1 个储物间
  入口：1 个自动门，1 个手动门

副本核心结构：
  支柱 1：CS-0002 林东 - 副本驻留
  支柱 2：03:00-04:00 冰柜锁定机制
  支柱 3：CS-0003 许小小 - 数据耦合

副本防御机制：
  红雨衣 - 副本核心 NPC
  行为：追捕违规玩家
  
副本传输结构：
  CS 档案管理处 - 系统节点
  状态：可访问
```

周仓看着结构图。

三个支柱确认了。

林东。冰柜。许小小。

要触发崩溃——

他需要让林东不再维持副本（他自己决定的事，今天早上已经做了）

他需要让冰柜不再作为支柱（需要破坏 03:00 的锁定机制）

他需要让许小小不再耦合（需要数据解耦胶囊）

但他现在 450 积分。

数据解耦胶囊 500 积分。

还差 50。

他看了一眼商城——

```
[限时] 紧急包 50 积分
     内容：1 个小型积分袋（50 积分）
     状态：可购买
```

50 积分买 50 积分。

没有意义。

他关了商城。

他想了一下。

他看向许小小。

「许小小。」

「嗯。」

「你——能帮我做一件事吗？」

「什么事？」

「完成 5 次深度对话——我还需要 2 次。」

她看着他。

「你——需要积分？」

「是。」

「为什么？」

「为了——」他说，「带你离开。」

她沉默了一会儿。

然后她笑了。

「好。」

她在收银台对面坐下来。

「那你问我问题吧。」

周仓看着她。

他问了一个问题。

「许小小——你三年前——为什么要来这个副本？」

她看着门口。

「因为——」她说，「我以为——这是个游戏。」

「什么意思？」

「我以为——这是个简单的游戏副本。我以为——我能赢。」

「为什么这样想？」

「因为我——」她说，「我不知道。」

她笑了。

「三年了。我还是不知道为什么我以为我能赢。」

弹幕飘过：

```
三年前的她
也是新人
她以为她能赢
但她输了
```

【系统提示】

```
[深度对话] 完成
[积分] +60
[剩余] 510 积分
```

他的积分——510。

够买数据解耦胶囊了。
