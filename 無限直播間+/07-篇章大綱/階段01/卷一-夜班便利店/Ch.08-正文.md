# Ch.08：救援

## 1

红雨衣离开后，便利店安静了整整二十分钟。

周仓坐在收银台后面，薯片包装袋还放在柜台上。他没有再吃。他在想那句话——「找到你自己的编号」。

后脑勺的阵痛已经退成一种持续的钝感。不剧烈——但存在。像是某种东西在他的颅骨内侧等待着被唤醒。

许小小从后场出来。

她没有说话。她走到收银台前，看了周仓一眼。

那种眼神——不是 NPC 看玩家的眼神——是有人在确认另一个人还活着的时候的那种眼神。

「它和你说了什么？」

周仓迟疑了一下。他在想该不该告诉她全部。

「它说——」他开口了，「我看到了不该看的东西。」

「还有呢？」

「还有……如果我想带你走，要先找到我自己的编号。」

许小小的表情没有变化。但她把手里的抹布放在柜台上，动作很慢。像在想什么。

周仓看着她。

「你知道编号是什么意思，对不对？」

许小小没有否认。

「我知道。」她说，「CH-0217。」

她说得很平静。

但周仓的心跳加快了。

「你怎么知道？」

她没有立刻回答。她把目光移向窗外——天亮了的街道上，开始有人走动了。正常的行人。正常的早晨。

「因为我也有一个。」

她卷起左手的袖子。

手腕内侧——和周仓空无一物的手腕不同——她那里有一行浅浅的灰色数字。

不是纹上去的。像是打印上去的——和薯片包装上的编号一样的位置、一样的字体。

```
CS-0003
```

周仓看着那行数字。

CS-0003。

不是 CH 开头。是 CS。

他看着许小小。

「你的编号……」

「不一样。」她说，「CH 和 CS 不一样。CH 是货架上的商品编号。CS 是——」

她停了一下。

「……是人。」

弹幕炸了。

```
CS-0003！！！
她真的有编号！！！
我靠 她真的是残留数据
CH 是商品 CS 是人的话
那林东的编号是什么
还有红雨衣的编号是什么
你们看周仓的表情 他是不是也想到了
```

周仓没有在听弹幕。他盯着许小小手腕内侧的那行灰色数字。

颜色和值班表上周六林东消失前的颜色一样——灰色。

「你这个编号——」

「会越来越淡。」她替他说完了，「每天淡一点。直到完全消失。」

「完全消失之后呢？」

她笑了一下。和之前那些笑不一样——不是轻快的。

是空的。

「不知道。我还没到那一步。」

周仓看着她。他张了张嘴，想说点什么。

但许小小先开口了。

「但你的编号还没出现。对吧？」

她说得对。他手腕上什么都没有。

「CH-0217——那是商品编号。你碰过那包薯片，你碰过那盒牛奶。但你自己还没有编号。」

「这说明什么？」

「说明——」她看着他，「你还没有被登记。」

## 2

便利店的灯突然闪了一下。

不是电压不稳的那种闪烁——是全灭了一秒。然后重新亮起来。

周仓的视野左上角弹出一条新的提示。

```
副本剩余时间：00:28:17
```

不到三十分钟。

弹幕开始变得焦躁。

```
还有 28 分钟
他要怎么完成通关？
他还没有找到自己的编号
而且红雨衣说——找到编号才能带她走
但 28 分钟够吗
他不是还有积分吗
他用完了 400 全买了地图
```

周仓也在想这个问题。

28 分钟。他还没有自己的编号。他不知道怎么找。他只知道一件事——红雨衣说过，「找到了，才能带她走」。

他现在还做不到。

他看向许小小。

她站在那里，看着他。

很平静。

像是早就知道会这样。

「许小小。」

「嗯。」

「如果我这次不能带你走——」

她打断了他。

「别说那种话。」

她的语气变了——不是柔和的。是强硬的。他在这个副本里第一次听到她用这种语气说话。

「你还有 28 分钟。你不是那种会在这里放弃的人。」

周仓看着她。

「你怎么知道？」

「因为——」她顿了一下——「我之前见过放弃的人。你不是他们。」

她说得很认真。

那份认真，不是 NPC 的程序行为。

是人。

是和他在同一条船上的、活生生的人。

周仓点了一下头。没有多余的话。

他打开积分商城。400 积分已经花完了。但他还有一样东西——

他打开了那片完整度只有 73% 的城区地图。

地图在他面前展开。

便利店在中心。周围的道路稀疏，有遮挡的区域在地图上显示为灰色条纹。

他在地图上找。

找什么？

他还不知道。

但他要找。

## 3

他找到了。

在便利店东南方向约 200 米的位置——地图上有一块灰色条纹区域。不是完全遮挡——条纹中间有一行极小的文字。小到几乎看不见。

他用手指放大那块区域。

文字显示：

```
CS-0003 注销登记处
```

CS-0003。

许小小的编号。

注销登记处。

他的手指停在那个位置。

「许小小。」

她走到他身边，看地图。

她的表情变了。

「那是什么？」

「你的编号对应的位置。」

「远吗？」

「走路——大概五分钟。」

她沉默了。

「如果我要带你走——」周仓说，「我需要先拿到那个登记处的什么东西。」

「你怎么知道？」

「我不知道。但红雨衣说找编号——不是我的，是你。」

便利店突然变暗了。

不是灯光在暗——是窗外。

明明是早晨。天色却在变暗。

像有人在外面把天拉上了一层幕布。

系统提示再次弹出：

```
⚠ 规则冲突警告。
副本时间线异常。
建议主播立即停止当前行动。
检测到违规倾向。
```

违规倾向。

周仓站在那里，看着系统警告。然后他做了一个决定。

「许小小。」

「嗯？」

「如果我现在去找那个地方——你能自己在这里待一会儿吗？」

她没有立刻回答。

「我能。」她说。

「等我回来。」

她看着他。点了点头。

周仓转身，走向门口。

他推开门。

风铃响了。

外面的街道——不是他记忆中的样子了。

天色暗得像傍晚。路灯全亮着。街道上空无一人。

但所有商店的招牌都亮着。便利店。隔壁的药店。对面的面包店。

全都亮着。

像是一个舞台布景，在他推门出来的瞬间亮起了所有的灯。

等待他走上去。

周仓站在便利店门口，深吸了一口气。

然后他迈出了一步。

━━━

【章末钩子】许小小手腕上有编号 CS-0003——她真的是残留数据。地图显示东南方向 200 米处有「CS-0003 注销登记处」。周仓推开门走出便利店——天色瞬间变暗，所有招牌同时亮起，像舞台布景在等他上场。
