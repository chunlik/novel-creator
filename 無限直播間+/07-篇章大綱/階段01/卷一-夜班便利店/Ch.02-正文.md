# Ch.02：规则建立

## 1

天亮的时候，便利店看起来没那么可怕了。

日光灯的嗡嗡声还在，但自然光从落地玻璃透进来，把昨晚那种「外面有人」的感觉冲淡了不少。周仓站在收银台旁边，手里拿着那瓶没有品牌的矿泉水——水已经喝完了，瓶子他还留着。

他不知道自己为什么要留着一个空瓶子。大概是习惯。在一个什么都不知道的环境里，先留下可以留下的东西。

昨晚他睡着过吗？

他想了想。好像有。又好像没有。便利店的时间感很奇怪——没有窗户能看到天色变化，只有天花板上的日光灯一直亮着。他的手机不在身上，收银台上也没有钟。

但系统有。

他抬起头，看着视野右上角。那里有一行小字，一直存在，只是他昨晚没注意到：

```
副本剩余时间：06:42:17
```

六小时四十二分钟。

所以他确实在这里过了一夜。还有不到七个小时，这个副本就结束了。

他环顾四周。货架上的商品和他的记忆一致——零食、泡面、饮料、关东煮的锅子在冒热气。一切正常。太正常了。

他从收银台旁边抽出一张纸——值班表下面压着的空白交接本。翻开。第一页是空白的。第二页也是。第三页——

有人写过字。

字迹很淡，像是被擦掉过又写上去的。

```
交班注意：
- 02:00-03:00 检查冰柜温度
- 货架第三排左上角缺货（已登记）
- 门锁正常
```

没有签名。没有日期。

他把本子翻到下一页。空白。

再下一页。也是空白。

他又翻回第三页——字还在。

他盯着那几行字看了一会儿，然后把本子合上。

弹幕在右边滚过。

```
他在看什么？
交接本吧
便利店有交接本很正常啊
但那本子是空的吧 我看他翻了三四页都是白的
```

周仓的视线在那条弹幕上停了一下。

空的。

观众看到的交接本是空的。

但他看到的上面有字。

他没有说话。他把交接本放回原处，走向货架区。心里记下一笔：观众视角和他的视角，不完全一致。

## 2

上午的时间过得很慢。

周仓把便利店走了三遍。第一遍记货架布局，第二遍记商品种类，第三遍看是否有变动。

结果：每次走完，货架的排列顺序都不一样。

不是全部变动——只是一两个商品的位置换了。比如第三次经过饮料柜的时候，他记得自己第一次走过时绿茶在第二层最左边，现在它在第三层中间。

他站在饮料柜前，回想自己有没有记错。

没有。他不会记错这种事。

弹幕又开始说话了。

```
他站在饮料柜前五分钟了
在想要喝什么吧
便利店副本最危险的其实是选择困难症
```

周仓没理他们。他在想另一件事。

他走到收银台旁边，看墙上的值班表。

`周一：林东`
`周二：林东`
`周三：林东`
`周四：林东`
`周五：周仓`

他昨晚看的时候，是这样的。

现在还是一样的。

但他总觉得哪里不对。他盯着那张表看了十几秒。

「林东」——他数了一下——两个字。

对。林东。两个字。

但他昨晚第一次看到这张表的时候，好像……不是两个字。

他不确定。

他把视线从值班表上移开，走回收银台。许小小不在那里——她在货架区整理东西，背对着他。

他靠在柜台上，没有叫她。

值班表上的字，好像变过。

每次他看，都和上次看有一点点不一样。

他决定——下次用手机拍下来。

然后他想起自己没有手机。

「行吧。」

他低声说了一句。弹幕立刻响应。

```
他说行吧
好冷静 这人
不是 你们不觉得他太冷静了吗
正常人在便利店副本应该已经尖叫过了吧
```

周仓扫了一眼弹幕，没有回应。

他们说得对，他太冷静了。

他自己也知道。

但他从小就是这样——越不正常的情况，他越冷静。真的慌起来反而是日常小事：赶不上公交、丢了钥匙、外卖被人拿走。

被丢进一个莫名其秒的便利店副本？反而还好。

这不是勇敢。是认知偏差。他自己清楚。

## 3

下午，他决定打开商城。

系统界面上的商城按钮一直在那里——左下角，一个小小的购物车图标。他之前没碰，因为不确定触发它会有什么后果。

但现在他需要信息。

他点了。

一块半透明的蓝色界面在面前展开，和系统提示框一样半透明的风格。左侧是商品分类，右侧是推荐。最上方显示他的余额：

```
积分余额：1,200
```

一千二。不知道是多还是少。

他扫了一眼商品列表。

```
┌─────────────────────────────────────┐
│  商城                              │
│─────────────────────────────────────│
│  情报／规则提示    800 积分         │
│  道具／一次性       500 积分        │
│  恢复／状态修复     300 积分        │
│  外观／主播皮肤     200 积分        │
│                                    │
│  推荐：情报／规则提示               │
│  ┌─────────────────────────────┐   │
│  │ 「便利店的冰柜里有什么？」   │   │
│  │  价格：800 积分             │   │
│  │  风险：情报可能不完整        │   │
│  └─────────────────────────────┘   │
└─────────────────────────────────────┘
```

情报可能不完整。

他盯着那行小字看了两遍。

商城在买之前就告诉你，你买的东西可能有问题。这不是诚信——这是免责声明。

但他还是买了。因为他需要知道。

他点了购买。积分余额从 1,200 变成 400。界面刷新，弹出一条新信息：

```
便利店的冰柜里存放着不属于便利店的物品。
打开之前，确认自己有权限。
```

就这两行。

不是情报——是警告。

他花了 800 积分，买了一条警告。

弹幕炸了。

```
800 积分买了一句废话
笑死 新手就是新手
商城情报是这样的 经常买到垃圾
但他说「不属于便利店的物品」欸
冰柜里有什么？
```

周仓没有后悔。至少现在没有。

他注意到那条情报的页面边缘——在文字消失的瞬间，右下角有一行非常淡的灰色文字，闪了一下就不见了。像是被覆盖过的痕迹。

他没有看清写了什么。但他确定那不是商城的正常内容。

他记住了那个位置。

等他再打开商城的时候，同一页面上，那行灰色文字已经没有了。

被删掉了。

在他看过的瞬间。

他关掉商城界面，走回柜台。许小小正好从后场走出来，手里拿着一盒牛奶。

她看了他一眼。

「你还好吗？」

「还好。」

「你的脸色不太好。」

周仓摸了摸自己的脸。他不知道自己的脸色怎么样。但他确实感觉有一点——不是累，是一种不舒服。

像是被人盯上的感觉。

不是观众那种「在看」的表情。

是另一种。

「没事。」他说。

许小小没有再问。她把手里的牛奶盒放在柜台上，转身走进后场。

周仓低头看了看那盒牛奶。

没有品牌。和矿泉水一样。但右上角有一个编号：

`CH-0217`

和他昨晚在薯片上看到的编号一样。

## 4

傍晚。

便利店的光线开始变暗——不是外面的天色，是店内的灯光。日光灯还是亮着的，但亮度好像降了一点。不明显，但那一点点变化足够让阴影变得更长。

周仓站在收银台前，手里握着那盒牛奶。

他还没有喝。也没打算喝。

他在想那个编号。CH-0217。两次出现，间隔十二小时。是巧合？还是这条线从一开始就在等他注意到？

他把牛奶盒放回冰柜。

然后他听到了声音。

很轻。

像什么东西碰到地面的声音。

他转过头。

门。

便利店的自动门——开了一次。

然后关上。

没有人进来。

门口没有人。外面街道上也没有人。路灯已经亮了，照出一小段空荡荡的人行道。

他看了一眼监控画面——收银台上有一台小显示器，循环播放四个摄像头的画面。收银台、货架区、冰柜前、门口。

门口的监控画面里，没有人。

和他肉眼看到的一样。

但——

他听到了脚步声。

很轻。不是从门口传来的——是从货架区深处。

他站在那里，没有动。

弹幕安静了两秒。

然后疯了一样刷起来。

```
有人进来了？？？
我没看到人 但我也听到了
不是 监控也没有啊
脚步声 货架那边
我也听到了
他为什么不跑
因为他没看到人
但声音是有的
幹 我真的听到了
```

周仓没有跑。

他站在那里，听着那个脚步声从货架区深处慢慢靠近。

一步。

两步。

三步。

然后停了。

停在他右手边——第三排货架的尽头。那里没有人。

但那里有一个空位。

货架上少了一包薯片。

就是那包编号 CH-0217 的原味薯片。

他早上经过的时候还在。

现在不在了。

他盯着那个空位看了五秒。

他没有转头看门口的方向。

但他知道——

此刻，有什么东西正在便利店里面。

和他在一起。

━━━

【章末钩子】自动门开了一次，没有人进来。但脚步声从货架区深处传来。一包编号 CH-0217 的薯片消失了。周仓确定有什么东西在他看不见的地方，正在便利店里面。
