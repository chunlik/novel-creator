# Ch.02：第一个规则

## 1

货架上的商品比看上去的要多。

周仓用了二十分钟，把五排货架全部清点了一遍。泡面、薯片、饮料、关东煮、面包、冷藏便当——品项接近 200。他没有记住全部，但他记住了两件事。

第一：每一排货架的商品都有编号。CH-0001 到 CH-0234。一共 234 件商品。

第二：编号是手写的。

不是机器打印的。是有人用马克笔在商品包装的右上角，亲自写上去的。字迹工整，但每一个「0」都有一点偏——像是手在抖。

他走到收银台，从监控上看了一遍整个便利店。

没有死角。四台摄像头的画面拼接起来，覆盖了 95% 的区域。

他注意到画面右下角有一个时间戳：

```
03:00:00
```

凌晨三点整。

规则说：03:00-04:00 期间，请勿靠近便利店冰柜。

现在——正是 03:00。

他回头看了一眼冰柜。冰柜在收银台左后方三米的位置。玻璃柜门，里面整齐地摆着冷饮、冰淇淋、冷冻便当。

03:00 的那一秒——冰柜的压缩机声突然停了。

不是缓慢的减速。是突然的、像被人按下了静音键的停止。

便利店变得很安静。

弹幕开始有人打字。

```
安静了
压缩机停了
这意味着什么？
03:00 是规则时间
00:00-08:00 是夜班时段
那 03:00-04:00 是更细分的时段
冰柜在那个时段会被「锁」住
锁住的意思是——你不能靠近
靠近会怎样
去看过的新人说过：会听见声音
什么声音
不告诉我。但那个新人进了医院
```

周仓看着弹幕滚过。

他没动。

03:00 整——他站在收银台里，离冰柜三米。他不动。

弹幕里有人开始刷礼物。

```
他不动欸
他在等
看新人什么时候能学到第一条：什么都不做
有时候什么都不做就是最对的
但他只有 100 积分
他要怎么活下去
```

03:01。

03:02。

03:03。

冰柜没有声音。

但周仓听到了另一种声音——非常轻——从冰柜的方向传出来。

不是机械声。不是压缩机。

是呼吸声。

像有什么东西在冰柜里面。

或者——贴着冰柜的内壁。

在呼吸。

## 2

周仓没有看冰柜。

他强迫自己看着监控画面。画面里，冰柜静静地立在那里。玻璃柜门反射着日光灯的光，看不清里面。

呼吸声持续了大约三十秒。

然后停了。

03:02:47——周仓看了时间戳。

呼吸声消失的那一瞬间，监控画面闪了一下。

他眨了眨眼。

画面恢复了。但他注意到冰柜的玻璃柜门里最下层、最角落的位置多了一个东西。

一个保鲜盒。

刚才不在。

他清点货架的时候——他确认过——冰柜里没有保鲜盒。

他走回货架区，绕到冰柜的侧面——从监控照不到的角度——看了一眼。

冰柜的玻璃上有一层薄薄的霜。

但玻璃门是关着的。

他没有打开。

规则说「请勿靠近」。他站在两米外。

弹幕又刷起来了。

```
他在偷看
他看监控但不直接看
他是不是知道监控有死角
从监控死角看
聪明
但冰柜里那是什么
保鲜盒
刚才还没有
每一局便利店副本冰柜里都会多东西
是副本自己放的吗
不知道
但我看了四局。每次都有人忍不住去打开
然后——就没有然后了
```

周仓退回收银台。

他在交接本上写下：

```
03:00 - 冰柜压缩机停止运行。
        - 02 分 47 秒后恢复。
        - 期间听到呼吸声。
        - 结束后冰柜内出现保鲜盒。
        - 未打开。
```

他放下笔。

脑子里转过三件事：

第一：冰柜是副本的「容器」。它会在特定时间被「填充」。填充的内容物（保鲜盒）是副本制造的。

第二：填充的时间点恰好是规则规定他「请勿靠近」的时间。这说明规则是「保护性」的——不是禁止他靠近冰柜，是告诉他冰柜那个时段会「有事发生」。

第三：他不能问任何人这个保鲜盒是什么。因为他不知道还有谁可以被问。

## 3

03:30。

冰柜已经恢复了运转。压缩机继续发出嗡嗡声。玻璃门上的霜也化了。

周仓站在收银台后面，看着监控。

03:30 的画面里，便利店一切正常。

但他注意到一个细节。

监控画面右上角的时间戳。

他记得自己第一次看监控的时候是 03:00。

现在 03:30。

时间戳从 03:00 跳到 03:30——间隔三十分钟。

但他从 03:00 之后做了一堆事。他数过——大概过了二十五分钟到二十八分钟。

监控的时间比他感知的时间快了两分钟。

他翻看监控的循环录像。录像以两小时为周期回放。他快进到自己清点货架的那段——时间戳显示：03:05-03:25。

但他自己数的是 03:00-03:20。

差了五分钟。

他放下回放键。

他知道了一个新的事——监控里的时间，和他感受到的时间，不是同一个时间。

他靠回去，闭了一下眼。

「系统。」

他开口了。

弹幕立刻响应。

```
他叫系统欸
主播要互动了？
这是新版的功能吗
系统会回应吗
```

「系统。」他又叫了一声。

视野中央弹出一个半透明的蓝色对话框。

```
┌──────────────────────┐
│  系统：              │
│  听到了。             │
│  说。                │
└──────────────────────┘
```

「监控里的时间，是什么时间？」

```
┌──────────────────────┐
│  系统：              │
│  副本时间。          │
│  你感受到的，是你的。  │
└──────────────────────┘
```

「有什么区别？」

```
┌──────────────────────┐
│  系统：              │
│  副本时间不会停。      │
│  你的会。             │
└──────────────────────┘
```

「会停多久？」

系统没有立刻回答。

对话框在闪。

三秒后：

```
┌──────────────────────┐
│  系统：              │
│  这取决于你在做什么。  │
│  如果你在思考，你不会停。│
│  如果你只是——等待。    │
│  你的时间会变慢。      │
└──────────────────────┘
```

周仓盯着那句话。

「我在思考的时候，副本时间也在走？」

```
┌──────────────────────┐
│  系统：              │
│  是。                │
│  你在思考——是有效时间。│
│  你在等待——是浪费。   │
└──────────────────────┘
```

周仓沉默了。

系统的话翻译过来是——他不能发呆。

发呆会让他的「主观时间」变慢——但副本时间继续走。结果就是：他会觉得自己只过了三十分钟，但副本已经过了一个小时。

他的七天——会比他以为的更早结束。

他必须在每一个小时都「有效」。

他要么行动，要么思考。

不能只是站着。

弹幕里突然有人刷了一句话。

```
他问出这个问题了
我看了五个新人。没人问过这个
他会不会是观察型 + 策略型
那能活七天吗
不一定。但这种人至少不会死在第二天
```

周仓关掉对话框。

他看了一眼积分：100。

然后他看到了一个新的按钮——在他视野的左下角。灰色的，没有激活。

按钮上写着：

```
【任务系统】
```

他点开。

```
┌──────────────────────────────────┐
│  任务                              │
│──────────────────────────────────│
│  [新] 主动服务 10 位顾客         │
│       奖励：500 积分 + 1 能力点    │
│       进度：0/10                  │
│                                  │
│  [新] 在便利店内存活 24 小时      │
│       奖励：300 积分              │
│       进度：01:23:45 / 24:00:00  │
└──────────────────────────────────┘
```

两个任务。

第一个需要 10 个顾客——但他被丢进来的瞬间是凌晨三点，便利店不会有顾客。

除非——有「不是顾客的人」进来。

比如红雨衣。

他盯着那个「0/10」的进度条。

他明白了。

红雨衣进来——也是一次「服务机会」。

但他不能直视它。

那他要怎么服务一个不能看的人？

他又问了一次。

系统这次没有立刻回答。

五秒后：

```
┌──────────────────────┐
│  系统：              │
│  服务的方式有很多种。  │
│  提示：              │
│  你已经完成了一次。    │
└──────────────────────┘
```

周仓看着那行字。

他已经完成了一次——他想起了红雨衣进门的那一次。

他站在收银台后面。他没动。他没说话。他没用余光盯着红雨衣看。

但他做了一件事——他把视线落在了门口的地面上。

那不是「直视」。

那是「侧目」。

系统把那次「侧目」算作了一次「服务」。

他完成了 1/10。
