# Ch.10：主系统大厅

## 1

白光消散。

周仓睁开眼。

他站在一个——大厅里。

不是便利店。不是街道。不是任何他之前待过的地方。

这是一个——

巨大的、空旷的、纯白色的空间。

天花板很高——至少二十米。地面是白色的，墙壁是白色的，远处是白色的——看不出边界。

大厅里——

有人。

不是一个人。

是——很多人。

几十个人。散布在白色大厅的各个位置。

他们——

都在做不同的事。

有人在角落里蹲着。有人在中间的柱子旁站着。有人在和身边的人说话。有人在发呆。

他们的衣服——各不相同。有的穿便利店的制服。有的穿医院的白大褂。有的穿普通的衣服。

周仓低头看了一眼自己。

他穿的还是便利店的制服——深蓝色 polo 衫，左胸口绣着店名。

他——

还是便利店副本的样子。

但他——

已经不在便利店了。

他——

在主系统大厅。

弹幕在他视野的右侧滚动：

```
他到了
主系统大厅
新人
这是他第一个副本
他拿到了 A 评价
这评价很高了
五星副本里 A 评价不到 5%
```

周仓扫了一眼弹幕。

他注意到——

弹幕里有人在说「五星副本」。

便利店副本是「三星」。

他拿到 A 评价——在五星副本里不到 5%。

但他拿的是三星副本的 A 评价。

那不是更有价值——

那是另一个问题。

他没有时间想这些。

他需要先搞清楚——主系统大厅是什么。

他打开系统——

```
[主系统大厅]

欢迎回来。
当前玩家等级：1
副本次数：1/100
总积分：5010

大厅功能：
  - 任务大厅（接受新副本）
  - 商城大厅（购买道具）
  - 休息室（恢复状态）
  - 玩家大厅（与其他玩家交流）
  - 排行榜
```

周仓扫了一眼。

任务大厅——接新副本的地方。

商城大厅——他之前在副本里买过东西，但那是副本内的商城。大厅的商城应该更全。

休息室——他需要。他三天没睡了。

玩家大厅——和其他玩家交流。

排行榜——

他点开排行榜。

```
[主系统大厅 - 玩家排行榜]

第一名：P-0001 | 副本次数：99 | 评价：全 S
第二名：P-0010 | 副本次数：87 | 评价：全 S
第三名：P-0052 | 副本次数：76 | 评价：全 A+
...
第 247 名：P-0217 | 副本次数：1 | 评价：A
```

P-0217。

他的编号。

他在第 247 名。

总玩家数——至少 247。

他的副本次数是 1。

他是最底层。

但他——A 评价。

弹幕刷过：

```
他在 247 名
P-0217 是新编号
他刚拿完第一个副本
他 A 评价
新人里罕见
```

周仓关掉排行榜。

他先去了休息室。

## 2

11:30。

休息室是一个小房间。

白色的墙壁。白色的小床。白色的枕头。白色的被子。

周仓躺在床上。

他很累。

但他睡不着。

他脑子里还在转便利店副本里的事。

许小小——林东——红雨衣——

他们——

还在便利店副本里吗？

还是——他们也被传送到了主系统大厅？

他打开系统——查询。

```
[玩家查询]
CS-0003 许小小 - 状态：在副本中
CS-0002 林东 - 状态：在副本中

注：玩家与副本结构融合后，副本即为其「常驻地点」。
他们可以选择继续待在副本里，也可以选择离开。
但默认状态下，他们会留在副本里。
```

许小小和林东——

默认会留在便利店副本里。

他们——

没有「离开副本」的意愿。

他们——

选择留在那里。

周仓——

松了一口气。

他们没事。

他们——

在一起。

他闭上眼睛。

他终于——

可以休息了。

## 3

三天后。

周仓醒过来。

他从床上坐起来。

休息室没有窗户。他不知道自己睡了多久。

但他感觉——好多了。

身体不累了。脑子也清醒了。

他打开系统——

```
[休息室 - 离开]

您已休息：72 小时
状态：完全恢复
```

72 小时。

他——

睡了三天。

他——

在便利店副本里待了三天。

他出来——

又睡了三天。

一周过去了。

他打开系统——

```
[玩家状态]

玩家：周仓
编号：P-0217
等级：1
副本次数：1/100
总积分：5010
称号：「副本改变者」

下一个副本：未接受
```

下一个副本——

他还没有接。

他在休息室里又待了一会儿。

他需要想清楚一件事。

下一个副本——

他要接什么？

他打开任务大厅。

```
[任务大厅 - 新副本列表]

1. 副本：失物档案馆
   类型：调查类
   难度：四星
   奖励：1500 积分
   备注：需要完成便利店副本的玩家

2. 副本：末班地铁
   类型：规则限制类
   难度：四星
   奖励：1200 积分

3. 副本：旧医院七楼
   类型：恐怖类
   难度：五星
   奖励：2000 积分

4. 副本：开学日
   类型：日常类
   难度：二星
   奖励：800 积分
```

周仓看着这个列表。

第一个副本——失物档案馆。

需要完成便利店副本的玩家。

而且是「调查类」。

便利店副本里——

有「失物」吗？

他想起了——

红雨衣在货架上拿过东西。

它拿的是——CH-0217 的薯片。

它把薯片塞进雨衣里。

然后它把薯片放在了便利店门口。

薯片——

是一个「失物」吗？

他想到了另一个事。

CS-0001 周远——林东之前的玩家——

在副本里「完整回收」了。

但林东——CS-0002——不是「完整回收」——是「被副本吞噬」。

周远——和林东——之间有什么关系？

失物档案馆——

会不会是——

周远和 2023 年那批人的「档案」所在地？

他点开失物档案馆的详情。

```
[副本详情 - 失物档案馆]

类型：调查类
难度：四星
时长：5 天

简介：
失物档案馆是一个位于城市边缘的档案保管机构。
它收集所有「被遗忘」的物品。
但它不收集物品——它收集「失物的主人」。

通关条件：
找到 5 件失物的主人。

注意：
本副本没有战斗机制。
但有心理压力机制。
玩家需要面对「被遗忘」的事实。
```

周仓看着这个简介。

「它收集失物的主人」——

这意味着——

失物档案馆里——

有「被遗忘的人」。

那些人的「物品」——在档案馆里。

但他们本人——

也被「收集」了。

这就是「失物」的意思。

不是物品失物了。

是人——

被「遗失」了。

周仓想起了许小小说的话：

「三年了。我每天都想离开。」

她——是「被遗失的人」。

她被「遗失」在便利店副本里。

失物档案馆——

是「被遗失的人」的档案所在地。

如果周仓去那里——

他可能会看到——

许小小三年前的档案。

她被「遗失」的过程。

她为什么会被「遗失」。

他想了一下。

他接了失物档案馆。

```
[已接受副本] 失物档案馆
开始时间：现在
```

白光再次出现。

他——

进入了下一个副本。

━━━
