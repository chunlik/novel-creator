# Ch.04：规则矛盾

## 1

凌晨三点。

便利店最安静的时候。

日光灯的嗡嗡声变小了——不是错觉，是真的变小了。像是电压不稳。灯光暗了一度，又在几秒后恢复正常。

周仓坐在收银台后面，没有睡。

他面前摊着那本交接本。上面写着他从昨晚到现在记录的所有内容：

```
22:00 - 红雨衣顾客进店。停留约2分钟。未购物。 
       离店后值班表出现周六栏位。

23:00 - 货架第三排左上角的薯片（CH-0217）确认消失。
       没有找到包装袋。

00:00 - 冰柜温度登记：正常。

01:00 - 值班表「周六：林东」字体颜色从黑变灰。

02:00 - 冰柜温度登记：正常。
       监控画面角度恢复——货架深处视角已消失。

03:00 - ???
```

他停笔。现在就是凌晨三点。

他在等那个问号变成具体的东西。

三分钟过去了。五分钟。

什么都没有。

他合上本子，站起来活动了一下肩膀。他需要再看一次值班表。

他走过去。

周六的林东还在。

但颜色比凌晨一点的时候更淡了——现在是浅灰色，接近白色的那种。

像是字正在消失。

周仓盯着它看了一会儿，然后转向旁边的监控显示器。画面顺序正常了——收银台、货架区、冰柜前、门口。每个画面都和他记忆中一致。

但他注意到了一个问题。

监控画面右上角的时间戳——比他手机记忆的时间慢了四分钟。

他没有手机，但他对时间很敏感。他记得自己最后一次看交接本是 02:00。他写完后，看了一眼监控，记下了时间。

当时监控时间：01:56。

现在他再抬头看——监控时间：03:01。

流逝了约六十五分钟。

但实际应该只过了六十分钟。

四分钟的偏差。

他记下这一点，走回收银台。但走到一半，他停住了。

货架上有一排饮料。

他早上注意过的是：绿茶、红茶、乌龙茶。现在变成：绿茶、红茶、乌龙茶——但乌龙茶的位置向右移动了一个瓶位。

不。

不是移动了。

是被动过。

他走近货架，弯下腰，看瓶盖上有没有指纹。没有。但他注意到另一个细节——

乌龙茶的包装上，标签的颜色变了。

早上的标签是绿色底、白色字。现在是绿色底、黄色字。

只是颜色变了。文字没有变。

他没有伸手去拿。他站在那里，看着那瓶乌龙茶，脑子里在拼一幅图。

值班表上的名字在变——从黑色到灰色到快消失。

监控时间和他感知的时间不一致——四分钟的漂移。

货架上的商品标签在变——颜色变了，位置变了。

三个不一致。三条线。

他在交接本上写下：

```
03:00 - 三个不一致：
        1. 值班表「周六·林东」正在消失。
        2. 监控时间比实际慢约4分钟。
        3. 货架商品标签和位置有变动。
```

他写完后，看着那三行字，思考了很长一段时间。

然后他写下了第四行：

```
问题：这些变动，是系统的错误，还是系统的线索？
```

## 2

弹幕开始分裂了。

```
他在写什么？有没有人能偷看一下
他写了三个问题
他在推理
不是 我看了十三个便利店副本了 从来没人写这个
这人是真在动脑子
还是他只是在假装很厉害
等等——你们看他的表情
他没有在装 他是真的在想
```

周仓没有在看弹幕。他在想另一件事。

值班表上有一个名字在消失。

消失——不是被涂掉，不是被撕掉。是字迹自己在变淡。

像是有人把那个人的存在，从记录里一点一点地擦掉。

他想起交接本上那几行字——只有他看得到、观众看不到的字。

还有商城情报页面边缘闪过的灰色文字——被删掉的内容。

便利店里的东西，正在被修改。

不是物理上的破坏——是记录层面的删除。

有些东西被人为地抹去了。

有些还没有。

他把交接本翻到全新的一页。在正中央，他写下了一句话：

「超市的东西在变。值班表的名字在消失。监控时间在漂——便利店不是稳定的。」

他盯着这句话看了十秒。

然后他在下面写了一行小字：

「但这个副本的通关条件，真的是「存活七天的规则」吗？」

不是。

他在本子上写下：

「通关条件可能不是遵守规则——是理解为什么规则在变。」

这是他进副本以来，第一次觉得自己抓住了什么。

爽感没有爆发——而是一种更安静的感觉。像是拼图的最后一块虽然没有落下，但他已经看到了整体的轮廓。

## 3

凌晨三点十五分。

周仓从交接本上抬起头。

他的后脑勺又开始痛了。不强烈——但存在。像有人在他的颅骨内侧轻轻敲了一下。警告性质的。

他没有理会。

他站起来，走向货架区。他需要检查一件事。

他走到第三排货架——CH-0217 薯片消失的那排。

他弯下腰，看货架底层。那里有几包泡面，标签正常。但有一包泡面的包装上——

印刷日期不对。

其他泡面的生产日期都是 2025 年之后的。但有一包泡面的日期是：

```
2023-11-07
```

两年前的日期。

在便利店里，一包生产于两年前的泡面，应该早就被下架了。

但它还在货架上。

他拿起来，翻到背面。配料表正常，营养成分表正常。但背面右下角有一行极小的字：

```
本产品所含成分可能与标签描述不完全一致。
```

正常的食品包装上不会有这句话。

他放下泡面，转身看向冰柜。

他想起商城情报的那句话：「便利店的冰柜里存放着不属于便利店的物品。」

他打开冰柜。

冷气扑面而来。

冰柜里的东西和正常便利店一样——冰淇淋、冷冻食品、冰棒。但他注意到最下层，靠近内壁的位置。

有一个保鲜盒。

不是便利店会卖的那种。是普通的家用玻璃保鲜盒，盖子上贴着一张胶带，上面用马克笔写着两个字：

```
林东
```

周仓蹲在冰柜前，看着那个保鲜盒。

他没有伸手去拿。

他想起商城情报的第二句话：「打开之前，确认自己有权限。」

他没有权限。

至少现在没有。

他把冰柜关上了。但那个保鲜盒的画面，已经印在了他的脑子里。

弹幕疯了。

```
冰柜里那是什么？？？
林东 保鲜盒上写着林东
不会是…
我想多了吧
你没想多
幹 他怎么不打开看一下
他说没有权限
但商城情报说的是真的！冰柜里真的有东西！
```

周仓站起来，走回收银台。

他坐下来，在交接本上写下：

```
03:20 - 货架：2023年泡面。包装上印有「成分可能与标签不一致」。
       冰柜：保鲜盒。标签「林东」。
       未打开。
```

他放下笔。

值班表上的「周六：林东」——现在只剩下一个非常淡的影子了。

再过不久，它就会完全消失。

和林东这个名字一样。

周仓看着那个快要消失的名字，想到一个问题——

如果林东也在这个便利店里待过。

如果林东也记录了这些规则。

如果林东发现了什么。

然后林东消失了。

那他自己呢？

他低头看自己的手。CH-0217 的编号在他的记忆里——薯片上、牛奶盒上、货架的标签上。无处不在。

如果他继续挖下去——

他会不会也变成值班表上一个正在消失的名字？

他抬起头，看向窗外。

落地玻璃外面，街道空无一人。路灯一闪一闪。

但他觉得——有什么东西在回看他。

不是红雨衣。

不是顾客。

是便利店本身。

这间便利店，不只是副本。

它是一间档案室。一间停尸间。一间装着不完整记录的黑盒子。

而周仓正坐在它的收银台后面——在这个盒子里面，试着拼出盒子的形状。

他低头看了一眼交接本。

上面最后一行字写着：

「通关条件不是遵守规则。是理解为什么规则在变。」

他从来没有这么确定过一件事。

也从来没有这么不确定过——确定之后，下一步该怎么办。

━━━

【章末钩子】周仓通过值班表、监控时间、货架商品三线不一致，推导出真规则的第一层。冰柜里的保鲜盒写着「林东」。但他没有权限打开。便利店不只是一个副本——它是一间记录着消失者的档案室。
