# Ch.04：值班表

## 1

周仓没有说话。

他站在收银台后面，消化着许小小刚才说的话。

红雨衣是林东。

林东是三年前和她一起进副本的同伴。

七个人。死了。一个完整回收。两个残留。三个变成「别的东西」。

七个同伴。现在她一个人坐在便利店里。

他没有问「为什么不离开」。

他知道答案——回收流程没有完成。她被系统认为是「已死」的人。系统不会主动回收她，但也不允许她离开副本。

她被卡在这里了。

「许小小。」

「嗯。」

「林东——他变成了红雨衣。他还有意识吗？」

她没有立刻回答。

她把手机屏幕按灭，看着自己的手腕。

「我不知道。」她说，「三年来他只进过几次店。每次都不说话。他只是……站着。站一会儿。然后走。」

「他知道你在这里吗？」

「应该知道。但他不和我说话。」

「为什么？」

她笑了一下。

「也许是因为他知道我记得他以前的样子。他不想让我看到他现在的样子。」

周仓看着她。

「你——」

「我没事。」她打断了他，「三年了。我已经习惯他那个样子了。」

她笑得很自然。

但周仓注意到了——她的手指在发抖。

## 2

04:15。

便利店进入相对平静的时段。

周仓从柜台下面翻出一张纸——是值班表。

```
周一：林东
周二：林东
周三：林东
周四：林东
周五：周仓
```

他盯着这张表。

林东的名字出现了四次。周五是他。

「这张表——」他问许小小，「原本不是这样的吧？」

许小小走过来，看了一眼。

「不是。」她说，「三年前这张表上还有六个名字。现在只剩下林东和你的。」

「其他人的名字——」

「褪色了。」她指着表格上方的某一处，「你看，那里。」

周仓凑近看。

值班表上方的空白处——原本应该有标题栏的位置——有非常淡的痕迹。

六个名字。

他眯起眼睛看。字迹已经浅到几乎看不见了——但如果仔细看，他能辨认出：

```
2023-11-07 / CS-0001 周远
2023-11-07 / CS-0002 林东
2023-11-07 / CS-0003 许小小
2023-11-07 / CS-0004 王可
2023-11-07 / CS-0005 刘一鸣
2023-11-07 / CS-0006 赵雪
2023-11-07 / CS-0007 陈旧
```

七个人。

七个编号。

他看到 CS-0002 林东——那个名字在值班表主表上还是清晰的。其他六个——包括她自己（CS-0003 许小小）——都已经褪成了几乎看不见的影子。

「为什么林东的没褪？」

「因为他在副本里。」许小小说，「他'死'了，但他的数据还在被副本使用。其他人的数据被副本清理了。」

「被副本清理了——」

「就像一个人走了，档案柜里的文件被烧掉。」她说，「但林东——他变成了副本的一部分。他的数据不能被清理。」

周仓盯着「林东」两个字。

他的数据不能被清理。

所以他变成了红雨衣——一直在便利店门口站着。

三年来没有离开过。

弹幕里有人刷了一句话：

```
他的数据不能被清理，所以他变成副本的一部分
那他有没有可能被「注销」？
像许小小那样？
```

周仓没有看弹幕。

他打开任务系统。

任务列表里多了一个新任务——

```
[新] 调查 CS-0002 林东 的过去
     奖励：300 积分 + 1 线索碎片
     进度：0/3
```

「许小小。」

「嗯。」

「林东——他以前在便利店做什么工作？」

「他不是店员。」许小小说，「他是……程序员。」

「程序员？」

「对。这个副本的程序员。」

周仓的脑子转过。

副本的程序员。

红雨衣不是被副本吞噬的失败者。

红雨衣是设计这个副本的人之一。

## 3

04:45。

周仓打开了商城。

他现在的积分是 350。

他扫了一眼商品列表。

```
┌────────────────────────────────────┐
│  商城                              │
│────────────────────────────────────│
│  情报／规则提示    800 积分         │
│  道具／一次性       500 积分        │
│  恢复／状态修复     300 积分        │
│  外观／主播皮肤     200 积分        │
│                                    │
│  [任务奖励] 线索碎片：林东的工牌    │
│             0 积分                │
│             状态：可领取           │
└────────────────────────────────────┘
```

线索碎片是任务奖励——他领了。

一块虚拟卡片在视野里展开。上面是林东的工牌照片：

```
┌────────────────┐
│  夜班便利店      │
│  系统设计组      │
│  CS-0002 林东    │
│  职位：副本程序员 │
└────────────────┘
```

他盯着那张工牌。

林东——副本程序员。

他设计了规则。他设计了红雨衣的行为模式。他设计了 03:00-04:00 冰柜停止运行的逻辑。

然后他「死」在了自己设计的副本里。

他变成了自己设计的红雨衣。

「许小小。」

「嗯。」

「林东设计的规则——他自己违反了，对吗？」

许小小的表情变了。

她沉默了很久。

「对。」她说，「他设计完副本的第二天，我们就进了副本。他是自己申请进来的。」

「为什么？」

「他说——」她的声音变得很轻，「他说他想测试一下。」

「测试什么？」

「测试他自己设计的规则——能不能被打破。」

周仓沉默了。

一个程序员设计了一个规则限制类副本。然后他自己进去测试。

测试结果——他死了。

他设计的规则不能被打破。

他变成了自己规则的牺牲品。

弹幕刷过：

```
林东是程序员
他自己设计的副本
然后自己死了
他变成红雨衣不是副本的「惩罚」——是他自己设计的机制
那他设计的机制里——他有没有给自己留一条活路？
```

周仓想到了这个问题。

他看了一眼任务列表。

```
[进行中] 调查 CS-0002 林东 的过去
        进度：1/3
```

他还有两条线索要找到。

他看了看值班表上的「林东」。

颜色是深的——但比刚来的时候浅了一点。

它在褪色。

很慢。但确实在褪色。

「许小小。」

「嗯？」

「林东的名字——会褪色吗？」

「会。」她说，「但很慢。他'死'了三年了——但他每天还在门口站一会儿。他维持着自己的存在。」

「如果他不站了呢？」

「那他的名字会消失。」她说，「消失之后——」

她停了一下。

「消失之后，他就会彻底变成副本的一部分。不能再被回收。」

「所以他每天站在门口——是在救自己？」

「不是救自己。」许小小说，「是——我不知道他为什么不放弃。」

周仓看着门口。

落地玻璃外面——天色已经微微发亮了。街道开始有车经过。

但门口的玻璃上——映出他自己的脸。

他看到了自己的眼睛。

他没有注意到——但他发现自己的眼睛下面有了一点黑眼圈。

副本的第三天还没到。

他已经开始有黑眼圈了。

他不知道许小小在这里三年——她是怎么熬过来的。

「许小小。」

「嗯。」

「你想离开这里吗？」

她没有立刻回答。

她看着门口。看着玻璃上映出来的自己。

「我——」她说，「我每天都想。」

「那我会带你离开。」

她说这句话的时候很轻。但周仓听到了。

她没有回头。

「别说这种话。」她说。

「为什么？」

「因为——」她说，「三年了。我已经不相信了。」

弹幕飘过。

```
她说不想了
三年太久了
新玩家都这样说。但最后都死了
他不一样
他眼睛里的东西不一样
我不知道。但他说这话的时候没在开玩笑
```
