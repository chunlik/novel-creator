# Ch.09：結算與消失（細綱）

## 本章目標

周倉通關→系統結算否認許小小存在→F-0217 邊緣權限殘影→卷末鉤子（周倉記得她，系統不承認）。

## 開局狀態

- 周倉知道：許小小救了他、她不是普通 NPC
- 周倉不知道：系統會否認她存在
- 讀者知道：同周倉
- 讀者知道但周倉不知道：許小小可能已經「不存在」了

## 主要衝突

- **衝突類型**：系統衝突（系統紀錄 vs 周倉記憶）、認知衝突（周倉 vs 系統判定）
- **衝突升級點**：通關→結算→系統提示許小小不存在→周倉拒絕接受→F-0217 殘影浮現

## 關鍵場景

| 場景 | 功能 | 情緒基調 | 關鍵資訊 |
|---|---|---|---|
| 1. 通關 | 周倉成功通關副本 | 短暫放鬆 | 系統顯示任務完成 |
| 2. 結算 | 系統結算清單——沒有許小小的名字 | 從放鬆→疑惑→震驚 | 系統說許小小不存在；「副本內不存在該 NPC」 |
| 3. 邊緣權限 | F-0217 殘影在周倉的 UI 角落閃過 | 懸念 | F-0217 邊緣權限——未登記者的殘留軌跡 |
| 4. 鉤子 | 周倉離開副本——他記得許小小救過他——但系統沒有她的任何記錄 | 心酸、懸念 | 卷末鉤子留存 |

## 套路應用

- T9（UI 異常）：F-0217 殘影在 UI 角落閃現
- T11（抹消 > 死亡）：系統說她不存在——這是讀者第一次接觸抹消的結果
- T18（未完成交易）：她救了他，但他無法確認她是否存在——未完成交易的重量

## 爽點 / 恐怖點 / 緩衝

- **爽點**：S8 權限階梯突破（F-0217 殘影）、S10 抹消具體化（系統說她不存在）、S12 許小小遠端接觸（系統否認她但她存在於周倉記憶）
- **恐怖點**：系統平靜地說「副本內不存在該 NPC」——最平靜的句子最恐怖
- **緩衝**：無（本章不留緩衝空間——讓讀者帶著情緒進階段二）

## 章末鉤子

- **類型**：身份／情感（雙重）
- **內容**：周倉走出副本——他記得許小小的臉、她救他的動作——但系統的結算清單上沒有她的名字。他低頭看自己的手——F-0217 的編碼在 UI 邊緣閃了一下。
