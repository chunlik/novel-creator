# Ch.08b：准备崩溃

## 3

09:00。

周仓开始准备让副本崩溃。

他打开系统。

```
[主线] 强制副本崩溃
      触发条件：完成以下所有条件
      1. 获得林东的临时权限
      2. 找到副本的核心支柱
      3. 移除所有核心支柱
      4. 触发崩溃

进度：
✓ 1. 获得林东的临时权限（剩余时间：23 小时）
? 2. 找到副本的核心支柱
? 3. 移除所有核心支柱
? 4. 触发崩溃
```

他需要找到副本的核心支柱。

他之前在 CS 档案管理处看到了林东的记录——林东是副本的核心支柱之一。

但「之一」意味着还有其他支柱。

他问了一次系统。

```
┌──────────────────────┐
│  系统：              │
│  副本核心支柱：      │
│  CS-0002 林东        │
│  ？？？              │
│  ？？？              │
└──────────────────────┘
```

三个支柱。

他知道一个。还有两个未知。

他开始思考。

第一个支柱是林东——他每天站在门口——维持副本存在。

第二个支柱可能是——冰柜？冰柜在 03:00 停止运行——它可能是某种「容器」。

第三个支柱可能是——许小小？她被卡在副本里三年——她的数据与副本结构耦合。

如果三个支柱是：林东、冰柜、许小小。

那他要做的是让林东不再维持副本让冰柜不再作为容器让许小小不再被耦合。

但许小小是玩家——他不能让她「消失」。

他想了一下。

他打开商城。

```
┌──────────────────────────────────┐
│  商城                            │
│──────────────────────────────────│
│  道具：                          │
│  数据解耦胶囊     500 积分       │
│  说明：将玩家与副本结构解耦      │
│  副作用：解耦后玩家不再受副本保护│
│                                  │
│  状态：                          │
│  紧急护盾         500 积分       │
│  状态：暂时缺货                 │
│                                  │
│  情报：                          │
│  副本结构图       300 积分       │
│  状态：可购买                   │
└──────────────────────────────────┘
```

数据解耦胶囊——500 积分。

副本结构图——300 积分。

他现在 250 积分。

不够。

他需要更多积分。

他看了一眼任务列表——

```
[新] 在便利店内存活 48 小时
     进度：09:23:45 / 48:00:00
     奖励：1000 积分
```

48 小时——还有 38 个小时。

太久了。

他看了一下另一个任务——

```
[新] 完成 5 次与 NPC 的深度对话
     进度：3/5
     奖励：300 积分
```

他已经完成了 3 次。和许小小对话了 3 次。

他可以再完成 2 次——再赚 300 积分。

但还不够。

他需要 500 积分来买数据解耦胶囊。

300 积分来买副本结构图。

一共 800 积分。

他现在 250。

差 550。

他可以：

A. 等 38 小时拿 1000 积分（太慢）
B. 完成 5 次深度对话拿 300（还差 250）
C. 找到其他赚钱方式

他想了一下。

他打开了另一个选项——

```
[隐藏] 放弃之前的「主动降低副本关注度」操作
      效果：副本关注度回到「高」
      后果：红雨衣会更频繁地出现
      奖励：500 积分
```

放弃降低关注度——让红雨衣更频繁地出现——赚 500 积分。

但红雨衣更频繁地出现——意味着更危险。

他犹豫了一下。

他点下了「放弃」。

```
[已放弃] 主动降低副本关注度
[获得] 500 积分
[当前] 750 积分
[副本关注度] 高
```

他现在的积分：750。

他买了副本结构图——300。

```
[已购买] 副本结构图 -300 积分
[剩余] 450 积分
```

结构图在他的视野里展开。

```
[副本结构图]

便利店主结构：
  货架区：5 排货架，存放 234 件商品
  收银区：1 台收银台，1 个交接本
  冰柜：1 个冰柜，1 个隐藏保鲜盒
  后场：1 个监控室，1 个储物间
  入口：1 个自动门，1 个手动门

副本核心结构：
  支柱 1：CS-0002 林东 - 副本驻留
  支柱 2：03:00-04:00 冰柜锁定机制
  支柱 3：CS-0003 许小小 - 数据耦合

副本防御机制：
  红雨衣 - 副本核心 NPC
  行为：追捕违规玩家
  
副本传输结构：
  CS 档案管理处 - 系统节点
  状态：可访问
```

周仓看着结构图。

三个支柱确认了。

林东。冰柜。许小小。

要触发崩溃——他需要让林东不再维持副本（他自己决定的事，今天早上已经做了）

他需要让冰柜不再作为支柱（需要破坏 03:00 的锁定机制）

他需要让许小小不再耦合（需要数据解耦胶囊）

但他现在 450 积分。

数据解耦胶囊 500 积分。

还差 50。

他看了一眼商城——

```
[限时] 紧急包 50 积分
     内容：1 个小型积分袋（50 积分）
     状态：可购买
```

50 积分买 50 积分。

没有意义。

他关了商城。

他想了一下。

他看向许小小。

「许小小。」

「嗯。」

「你——能帮我做一件事吗？」

「什么事？」

「完成 5 次深度对话——我还需要 2 次。」

她看着他。

「你——需要积分？」

「是。」

「为什么？」

「为了——」他说，「带你离开。」

她沉默了一会儿。

然后她笑了。

「好。」

她在收银台对面坐下来。

「那你问我问题吧。」

周仓看着她。

他问了一个问题。

「许小小——你三年前——为什么要来这个副本？」

她看着门口。

「因为——」她说，「我以为——这是个游戏。」

「什么意思？」

「我以为——这是个简单的游戏副本。我以为——我能赢。」

「为什么这样想？」

「因为我——」她说，「我不知道。」

她笑了。

「三年了。我还是不知道为什么我以为我能赢。」

弹幕飘过：

```
三年前的她
也是新人
她以为她能赢
但她输了
```

【系统提示】

```
[深度对话] 完成
[积分] +60
[剩余] 510 积分
```

他的积分——510。

够买数据解耦胶囊了。

系统弹出一条提示：
```
[警告]
解耦异常数据可能产生数据债务。
是否继续？
```

数据债务？
他——不知道那是什么。
但他——点了「是」。

他——买了。

积分扣了 500。

剩余——10。

━━━

【章末钩子】周仓决定让副本崩溃——他买了数据解耦胶囊（500 积分）。但副本结构图揭示了三个核心支柱：林东、冰柜、许小小。三个都需要解除才能让副本真正崩溃。

【系统奖励】完成 5 次深度对话任务 / 获得副本结构图

【积分余额】10
