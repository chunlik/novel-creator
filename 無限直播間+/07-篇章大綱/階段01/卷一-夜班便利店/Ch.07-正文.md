# Ch.07：危机升级

## 1

天亮之后，便利店的氛围没有变好。

日光灯还是亮的。关东煮还是热的。但那些平时让人觉得正常的东西——在周仓知道了许小小的身份之后，变得不一样了。

他坐在收银台后面，脑子里反复翻着那条红色警告。

「未授权条件下运行。」

「数据残留。」

他看了一眼许小小。她在擦货架。动作很轻，像是在对待什么东西。

他不知道她在这里轮回了多少次。

他也不知道自己还能做什么。

但有一件事他确定了——他要把她带出去。

弹幕里有人注意到了。

```
他的表情变了
变得认真了
他是不是想到什么了
他刚才一直在看许小小
他是不是想救她
兄弟 这是副本 不是剧情片
但他是主播 他有积分 他可以买情报
他只有 400 积分了
```

周仓没有看弹幕。他在积分商城。

他把仅剩的 400 积分全部花了——买了一条新情报。

```
解锁：附近城区地图（便利店周边 500 米范围）
完整度：73%
```

地图展开在他的视野里。

便利店在中心。周围有街道、几栋建筑、一条巷子。

地图的完整度只有 73%——也就是说，有四分之一的区域是被遮挡的。

他没有时间细看地图。

便利店的自动门开了。

风铃响了。

红雨衣。

又来了。

## 2

红雨衣站在门口。

和上一次一样——全身包在红色塑料雨衣里。雨帽压得很低，看不到脸。站在门口，没有进来。

但这一次和上一次有一点不同。

它手里拿着一样东西。

一包薯片。

CH-0217 的原味薯片。

周仓认出来了。就是昨天从货架上消失的那一包。

他的手按在收银台上。视线没有直视红雨衣——他用余光锁住它。

红雨衣没有动。

它就站在那里，手里拿着那包薯片。

像在等人过来。

像在要求他还原。

弹幕刷过。

```
它拿着那包薯片回来了
它在等什么？
等他去拿吧
别去
别去 兄弟
不是 他不去的话会怎样
规则没说不去会怎样
因为规则里没有这个场景…
```

周仓没有动。他在想一个问题——

红雨衣能进来吗？

规则说「夜班期间，请勿直视门外」。红雨衣第一次进来的时候，规则没有阻止——或者说，它确实没有被门挡住。

但它也没有在这里做出什么「违规」的事。它只是走了一圈，离开了。

这次不一样。

这次它手里拿着他碰过的东西。

周仓开口了。

「你想要什么？」

他的声音在安静的便利店里响起来。不大，但很清楚。

红雨衣没有回答。

但周仓注意到一个细节——它拿着薯片的手，动了一下。

不是松手。是指尖收紧了一下。

像是握紧。

它在回应他的声音。

它在表明：它知道他在这里。

它能听到他。

## 3

然后红雨衣进来了。

和第一次一样——一步、两步、三步——走进店里。路线没有变：经过收银台，左转，进入第三排货架。

但这一次，它没有停在 CH-0217 的位置。

它穿过了第三排货架，走到更深处。

到了后场门口。

红雨衣站在那里，面对着那扇关着的门。

后场。

许小小十分钟前进去了。

周仓的心脏跳了一下——不是跳得快，是跳得重。那一击像锤子敲在肋骨上。

他没有喊。他不能喊。

他站起来，绕过收银台，走向货架区。

红雨衣站在那里。背对着他。红色的雨衣在日光灯下微微反光，像一层不自然的涂层。

周仓在第 3.5 排的位置停下——离后场门还有三米。

他们之间隔着几排货架。

但他没有说话。

他从货架上拿起一件东西——一包薯片。不是 CH-0217，是同品牌、同口味的另一包。

他撕开包装。

声音在安静的便利店里异常清晰。

红雨衣的头——动了。

雨帽下沿微微抬了一下——像在看向他的方向。

周仓没有看它。他从包里取出一片薯片，放进嘴里，咬了一口。

咔嚓。

红雨衣没有动。

他咬下第二口。

咔嚓。

然后他听到了——

一声极轻的声音。

不是红雨衣发出来的。是后场门。门缝里——开了一条线。

许小小的眼睛出现在门缝里。

她在看。

周仓没有转头。他只是继续吃薯片。一片。又一片。

他在传递一个信息：这里有人。而且是活着的、正常的、可以沟通的人。

红雨衣站了大约四十秒。

然后它转身了。

不是走回门口——是走向他。

一步。两步。

周仓没有退。他手里拿着薯片包装袋。他的视线放在红雨衣的胸口位置——不直视脸，不直视雨帽下方。

红雨衣在他面前两步的距离停下来。

它比他矮——大约矮十公分。但那种压迫感不是身高决定的。是它站在那里的时候，周围的空气像是变稠了，呼吸起来有些费劲。

然后红雨衣开口了。

声音很轻。像是隔着一层塑料膜传出来的。

「你还……没……有……到……时……间。」

一字一顿。像很久没有说过话。

「但……你……已……经……看……到……了……不……该……看……的……东……西。」

周仓没有说话。

「你……看……到……了……编……号。」

「你……看……到……了……保……鲜……盒。」

「你……看……到……了……她。」

那个「她」——周仓知道它在说谁。

许小小。

他的左手手指在身后做了一个小小的动作——

示意许小小不要出来。

他没有转头确认。但他听到了后场门的缝隙被合上的声音。

她看到了。

她没有出来。

红雨衣又说话了。

「如……果……你……想……带……她……走——」

它停顿了一下。

雨帽下方，一片阴影里——

周仓觉得它笑了。

不是看出来的。是一种直觉。周围的空气变了一下。

「你……要……先……找……到……你……自……己……的……编……号。」

说完。红雨衣转身。

走回门口。推开门。消失在日光渐亮的街道里。

风铃又响了。

然后安静了。

周仓站在那里，手里还拿着那包吃了一半的薯片。

他的脑子在重放那句话。

「你要先找到你自己的编号。」

他自己的编号。

他低头看自己的手腕。那里什么都没有。

但他感觉到了。

后脑勺——痛。

不是刺痛——是持续的阵痛。

像有什么东西在那里——在他的颅骨内侧——

等着被找到。

━━━

【章末钩子】红雨衣第三次出现，主动开口说话。它知道许小小的存在。它说——「如果你想带她走，你要先找到你自己的编号。」后脑勺开始持续性阵痛，像有什么东西在等着被找到。
