# Ch.07：观众

## 1

周仓从后场走回便利店。

他关掉了 CS 档案管理处的画面——他现在有林东的临时权限 24 小时。他有时间。

但他注意到一件事。

便利店的落地玻璃外面——那条街——开始有行人了。

不是普通的行人。

他们都在看便利店。

他们不是路过。他们是停下来看的。

周仓站在收银台后面，看着玻璃外。

三个男人。两个女人。都站在便利店门口。看着里面。

他们不是顾客。

他们没有要进店的意思。

他们就只是看着。

弹幕里有人开始打字：

```
外面有人了
他们在看什么
看主角吧
他们是谁？
```

周仓看着其中一个人。

那是一个中年男人。穿着普通的衣服。手里拿着手机。

他盯着便利店看。

周仓低头看了一眼手机——不，他没有手机。

但他看着那人的嘴型——他在说话。

他看不清他在说什么。

但他的嘴型像是——「出来。」

他在叫周仓出来。

弹幕里有人刷：

```
他们在说「出来」
他们不是过路人
他们是什么人
```

周仓的视野里弹出一条新的系统提示：

```
[新规则]
请勿与观众连续说话超过 10 秒。
违反后果：观众入侵。
```

观众入侵。

周仓看着这条规则。

他现在有一条新规则。规则的对象不是红雨衣——是观众。

但观众是——在看直播的人。

观众怎么会「入侵」？

他看了一眼弹幕。

弹幕还在滚：

```
外面那些人是谁
他们也是玩家吗
不是，他们没有编号
他们是——观众？
观众可以离开屏幕吗
```

周仓的脑子里闪过一个想法。

观众——一千多个人在看直播。

但他们不只是「看」。

他们可以——入侵副本。

这意味着——他们不只是观众。

他们是参与者。

「系统。」他开口。

```
┌──────────────────────┐
│  系统：              │
│  说。                │
└──────────────────────┘
```

「外面的观众——他们为什么在看我？」

```
┌──────────────────────┐
│  系统：              │
│  他们在做选择。       │
│  他们是观众，          │
│  也是——评估员。       │
└──────────────────────┘
```

「评估什么？」

```
┌──────────────────────┐
│  系统：              │
│  评估你值不值得活下去。│
└──────────────────────┘
```

周仓看着那行字。

值不值得活下去。

观众在看的不只是「娱乐」。他们在做评估。

评估的结果——决定他的生死。

他看了一眼外面的人。

他们开始散了。

但他们离开的方式很怪——他们不是转身走掉。他们是「消失」的。

像从空气里被擦除。

一秒。

人就没了。

便利店外面又空了。

弹幕里有人开始紧张：

```
他们消失了
他们不是走掉的
他们是被系统——回收了
观众也会被回收吗
他们是「外部观众」——不是直播里的观众
直播里的观众——不会消失
但外部观众会
因为他们的功能是评估
评估完了就消失
```

周仓看着那条弹幕。

外部观众。

他们不是真正的观众——他们是「评估员」。

评估员来了，评估完了，走了。

他们的评估结果——他不知道。

但他知道一件事——观众在看的不只是「娱乐」。

他们在决定他的生死。

他做得好——他们让他活。

他做得差——他们让他死。

这个副本不是「系统对他」的考试。

是「观众对他」的考试。

## 2

07:00。

便利店进入白班时段。

规则里的「夜班」是 00:00-08:00。现在是 07:00——还有一小时夜班才结束。

周仓站在收银台后面，脑子里转过很多事。

他整理了目前掌握的信息：

```
[副本信息]
类型：规则限制类
难度：三星
存活：7 天
当前：副本第 1 天 / 剩余约 6 天

[已知规则]
#1 夜班期间（00:00-08:00），请勿直视门外
#2 03:00-04:00，请勿靠近冰柜
#3 听到「欢迎光临」请回应「请慢走」
#4 商品不能跨区域移动
#5 听到有人喊「林东」，请不要回头
#6 [新] 请勿与观众连续说话超过 10 秒

[已知 NPC]
许小小：CS-0003，三年前的玩家
红雨衣：CS-0002 林东，副本驻留

[已知物品]
紧急修复胶囊：已用
烟雾弹：剩余 1
林东的工牌：已获得

[能力]
永久：【时间校准】- 主观时间与副本时间差异减少 50%

[积分]
当前：450
```

他把本子上的信息过了一遍。

他注意到——规则 #5「不要回头」他没有违反过。

但他听到了「林东」这两个字。

是许小小说的。

但他没有「听到喊叫」——他只是听到对话。

所以规则 #5 不是说「不能听到林东这个名字」——是说「不能听到有人喊'林东'」。

「喊」是动词。

是有人专门叫林东的时候——不能回头。

他现在还没有遇到这种情况。

他继续整理。

许小小从后场走出来。

她的脸色比之前好了一点。

「你刚才——」她看着他，「去哪了？」

「CS 档案管理处。」

她的表情变了。

「你怎么找到的？」

「你在任务里给过提示。」

她笑了一下。

「那是三年前的提示。我以为你不会去。」

「为什么？」

「因为——」她想了想，「去过的人，都没有回来。」

周仓看着她。

「他们没回来——是因为他们回不来？还是——」

「因为他们选择不回来。」许小小说，「CS 档案管理处——有让人留下来的方法。」

「什么方法？」

「看到自己被回收的家人、朋友、以前的同伴——的档案。看到他们被回收的过程。看完之后——」

她停了一下。

「看完之后——」

「就会想留下来？」

「会。」她说，「会想陪着他们。」

周仓沉默了。

「你没有去档案管理处吗？」

「去过。」她说，「我看到的是 2023 年 11 月 8 号。我看到我自己被回收的过程。」

「看完之后——」

「我本来想留下来。」她说，「但我没有。因为——」

她笑了一下。

「因为林东在门口站着。他每天都在。他不离开。我不能在他站着的时候，自己走掉。」

周仓看着她。

「所以你留下来——是为了陪他？」

「不完全是。」她说，「我也想——看看谁会来。」

「来做什么？」

「来——」她看着他，「来把他带走。」

弹幕刷过：

```
她等了三年
等一个能带走林东的人
现在来了
是周仓
```

## 3

07:30。

周仓正在整理商品。

他需要完成「服务 10 次顾客」的任务。他已经完成了——但他还要积分。

他打开商城——

```
┌──────────────────────────────────┐
│  商城                            │
│──────────────────────────────────│
│  道具：                          │
│  时段冻结胶囊     400 积分       │
│  说明：冻结 5 秒时间             │
│  副作用：自己也会被冻结          │
│                                  │
│  道具：                          │
│  监控记录器       200 积分       │
│  说明：可保存 1 小时监控录像     │
│                                  │
│  情报：                          │
│  2023 副本事件    500 积分       │
│  状态：可购买                   │
└──────────────────────────────────┘
```

时段冻结胶囊——400 积分。

监控记录器——200 积分。

2023 副本事件——500 积分。

他现在 450 积分。

他买了监控记录器。

```
[已购买] 监控记录器 -200 积分
[剩余] 250 积分
```

他把记录器激活。

从现在开始——监控会保存。

他要记录接下来发生的事。

弹幕里有人刷：

```
他开始录了
他开始主动记录
他在准备什么
```

周仓没有看弹幕。

他看了一眼门口。

红雨衣——林东——还在外面。

他每天站在门口。

三年来——他每天都在。

但周仓注意到了一个细节。

他今天早上——被红雨衣追出便利店的时候——它冲进来。

它从来没有冲进来过。

但今天——它冲进来。

它追他了。

这意味着——规则变了。

红雨衣的行为模式变了。

以前——它只是「站着」。

现在——它会「追」。

它被激活了。

他激活了它。

他用紧急修复胶囊清除了 CH 标记。

他用 CS 档案管理处的权限登录了系统。

他做了太多事。

他「动」了太多副本里的东西。

他激活了副本的防御机制。

红雨衣——副本的防御机制——开始追他了。

他看了一下规则 #6：

```
请勿与观众连续说话超过 10 秒。
违反后果：观众入侵。
```

他刚才和系统说了很多话。

但他和弹幕——观众——没有说话超过 10 秒。

所以规则 #6 没有被违反。

但他感觉得到——有东西在变。

副本在变。

规则在变。

他在副本里的「存在」——

在变得——清晰。

清晰——意味着被副本「看到」了。

被副本看到——意味着被副本追。

他要怎么脱身？

弹幕里突然有人刷了一句话：

```
他被副本锁定了
他做的事太多了
系统已经把他标记为「高价值目标」
他需要——降低自己的价值
```

周仓看着那条弹幕。

降低自己的价值。

他要让系统觉得——他不重要。

怎么做？

他想了一会儿。

他打开了任务系统。

他找到了一个选项——

```
[隐藏] 主动放弃任务
      效果：放弃任务后，玩家在系统内的「重要性」降低
      副作用：放弃的任务越多，系统会认为玩家越「失败」
      注意：失败的玩家可能会被系统提前清理
```

主动放弃任务。

他可以让系统觉得他「失败」。

但失败的玩家——会被清理。

他不能让自己显得太失败。

他要找一个平衡点。

他想了一下。

他放弃了「调查 CS-0002 林东 的过去」这个任务。

```
[已放弃] 调查 CS-0002 林东 的过去
```

这条任务他还没完成。但放弃它——会让系统觉得他「不关心林东」。

不关心林东——意味着他对林东没有威胁。

对林东没有威胁——系统对他的关注度会降低。

系统的提示更新：

```
玩家重要性评估：中等
副本关注度：中等
```

中等——不是高。

他松了一口气。

但他知道——这不会持续太久。

他必须尽快找到真正的解决方案。

他看了一眼监控录像。

他刚才录的 30 分钟——现在在回放。

他看到了什么东西。

03:00 冰柜停止运行的那一段——冰柜玻璃门里不只有保鲜盒。

保鲜盒的旁边——有一个人。

站着的。

看不清脸。

但那个人的轮廓——和红雨衣一样。

当时——他没看到。
不是他没注意——是光线。
监控画面拍到的是冰柜内部的光线反射。
人在保鲜盒旁边——被阴影遮住了。
回放时——他把亮度调到最高——才看到了那个轮廓。

红雨衣——在冰柜里待过。

他不是「从外面来的顾客」。

他是「从冰柜里出来的东西」。

━━━

【章末钩子】新规则 #6：不要和观众说超过 10 秒的话。周仓发现红雨衣是从冰柜里出来的——它不是从外面来的，是冰柜里「生出来」的。副本的机制比他以为的更复杂。

【系统奖励】发现红雨衣的来源 / 主动降低副本关注度
