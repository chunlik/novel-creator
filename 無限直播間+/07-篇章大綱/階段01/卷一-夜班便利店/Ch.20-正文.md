# Ch.20：阶段结束

## 1

周仓——在主系统大厅——休息了三天。
清除者——在跨副本追踪时需要重新校准坐标。
三天——是校准时间。

三天里他接了最后一个副本。

「最后一天」。

这是一个——剧情类副本。

不是探索。不是规则限制。

是让他经历一段故事。

副本简介：

```
[副本详情 - 最后一天]

类型：剧情类
难度：三星
时长：3 小时

简介：
这是主系统大厅的最后一个副本。
也是阶段一（卷一）的最后一个副本。
玩家将在副本里——
体验一次——
作为副本 NPC 的生活。
然后——选择——你的下一步。

通关条件：
完成 NPC 的一天。
然后——做出选择。
```

周仓——接了这个副本。

白光——出现。

他——进入了——副本。

## 2

周仓睁开眼。

他——站在——便利店的大厅里。

但——不是他之前待过的便利店。

是——另一个便利店。

这个便利店——是——2018 年的便利店。

和现在——不一样。

更旧。更小。

灯光是白炽灯不是日光灯。

空气——没有消毒水的味道。

有一种很老的塑料味。

周仓——环顾四周。

他——穿着——便利店的制服。

他是这个便利店的店员。

他——看了一下系统。

```
[副本信息]

玩家身份：便利店店员
副本时间：2018 年
任务：完成一天的工作
时长：24 小时（剧情内）
```

他——要在 2018 年的便利店——工作一天。

他——开始了。

## 3

这一天——和普通的一天——一样。

早上开店。

整理货架。

收银。

关东煮。

迎接顾客。

和顾客说话。

和顾客说「欢迎光临」。

和顾客说「请慢走」。

中间——有一个老婆婆来买东西。

她——买了——一瓶水。

她——付了钱。

她没有要袋子。

她把水装进了自己的布袋里。

周仓——看着她。

她——笑了一下。

「小伙子。」她说，「你很勤快。」

「谢谢。」

「你——」她说，「会一直在吗？」

「会。」

「那——」她说，「我以后——还会来。」

她走了。

下午——有一个年轻人来买烟。

他——没有——带身份证。

周仓——不能卖给他。

年轻人——生气了。

他说——「你们便利店——真麻烦——我去别的店买。」

他走了。

周仓——没有生气。

他只是做了他应该做的事。

傍晚——有一个妈妈来买奶粉。

她的孩子在怀里哭。

她——很急。

她——让周仓——快点。

周仓帮她找了她要的品牌。

她——付了钱——说了声「谢谢」——

就走了。

晚上——末班时段。

顾客少了。

周仓——坐在收银台后面。

他——看着外面。

街上的路灯——已经亮了。

他——想起了——许小小。

她——在便利店——待了三年。

她每天看同样的路灯亮起来。

她每天看同样的收银台。

她每天和同样的空气待在一起。

她——三年。

周仓想——如果是他——他能待三年吗？

他——不知道。

## 4

午夜。

24 小时——到了。

周仓——站在收银台后面。

系统公告弹出：

```
[副本完成]

玩家已完成一天的工作。
评价：优秀
原因：忠实执行了店员的所有职责
积分奖励：+1500
特殊奖励：体验 NPC 的一天

现在——
请做出选择。
```

选择。

周仓——看到了——两个选项。

```
[选择 A]

继续作为便利店店员。
你将成为这个便利店的永久 NPC。
你将和这个便利店——永远在一起。
你将不再收到副本任务。
你将不再积累积分。
你将成为便利店的一部分。

[选择 B]

离开便利店。
你将回到主系统大厅。
你将继续作为玩家。
你将收到新的副本任务。
你将继续积累积分。
你将——继续——前行。
```

周仓——看着这两个选项。

选择 A——留下。

选择 B——离开。

他——想了三秒。

他——选了 B。

他——离开。

白光——出现。

便利店——开始——消失。

货架、收银台、关东煮的锅子、老婆婆、年轻人、妈妈、孩子——全部消失。

周仓——一个人——站在白光的中央。

他——选择了——继续前行。

白光——吞没了——他。
