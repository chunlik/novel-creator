# Ch.16：末班地铁

## 1

周仓睁开眼。

他站在一个——地铁站里。

不是便利店的入口。

不是档案馆。

不是失物招领处。

是一个真正的末班地铁的站台。

站台——没有人。

但灯光——是亮着的。

墙上挂着一个钟：

```
23:45
```

23:45。

末班地铁——还有 15 分钟。

周仓环顾四周。

站台很安静。屏蔽门是关着的。轨道上看不到任何车。

他打开了系统。

```
[副本信息]
副本：末班地铁
类型：规则限制类
难度：二星（已降星）
时长：1 小时
通关条件：乘坐末班地铁到达终点站
注：副本难度已降低，规则已简化。
```

通关条件——坐末班地铁到终点站。

但——副本是「规则限制类」。

有规则。

他打开了规则。

```
[副本规则]

1. 末班地铁只会停靠三站。
2. 你必须从第一站坐到第三站。
3. 中途下车——副本失败。
4. 不要和车上的人说话。
5. 如果有人下车——不要跟下去。
```

五条规则。

周仓看着这些规则。

规则 4「不要和车上的人说话」——

这是一个社交限制。

规则 5「如果有人下车——不要跟下去」——

这是一个行为限制。

他——不能违反这些规则。

如果违反了——副本会失败。

他想了一下。

他接受了这些规则。

他站在站台上——等末班地铁。

23:50。

远处——传来了声音。

地铁进站的声音。

由远到近。

然后灯光从隧道深处照过来。

末班地铁——到了。

地铁——停在了站台前。

屏蔽门——打开。

周仓——走进了地铁。

地铁——和他记忆中的——不一样。

不是普通的地铁车厢。

是——一节——空的车厢。

只有一节。

座位——是空的。

但——最后一排——有一个人。

一个——老人。

穿橙色马甲的老人。

他——坐在最后一排的座位上。

他——看着周仓。

他——没有说话。

周仓——也没有说话。

规则 4 说——不要和车上的人说话。

他——遵守规则。

他——在中间的位置——坐下来。

他——看着前方。

地铁——开始动了。

屏蔽门——关上了。

地铁——驶入隧道。

车厢里——很安静。

只有地铁行驶的声音。

周仓——坐在座位上。

他没有看那个老人。

他——在想——李大伟。

李大伟——是检票员。

他的工牌——出现在了失物招领处。

他——失踪了。

但他的灵魂——还在等待。

李大伟是这个副本里那个老人吗？

他——不确定。

他——等着看。

## 2

第一站到了。

地铁停了。

屏蔽门打开。

但没有人上车。

也没有人下车。

屏蔽门又关上了。

地铁继续行驶。

周仓数着站数。

第一站过了。

第二站到了。

地铁停了。

屏蔽门打开。

这一次，有一个人上车了。

一个年轻女人。

她穿着普通的衣服，手里拿着一个包。

她在周仓对面坐下了。

她看着周仓，没有说话。

周仓也没有说话。

规则 4——不要和车上的人说话。

他不违反规则。

地铁继续行驶。

在车厢另一端的阴影里，坐着一个年轻女人。她看到了周仓的编号标签，怔了一下。「P-0217？」她低声说，「有人被标记了——」旁边的人轻轻拉了她一下，她没再说。

周仓站了起来。他走到那个人面前，直接在他对面坐下。「你认识这个编号。」周仓说，是肯定句不是问句。

弹幕飘过：
```
他直接問了
規則4不是不能說話嗎
完了他要被懲罰了
```

那个人脸色发白：「我——不——」

「你刚才看到我编号的时候，瞳孔放大了。」周仓说，「你知道P-0217代表什么。告诉我。否则我现在就去广播室喊『P-0217玩家在此』——到时候所有人都知道你在跟我说话。」

那个人咽了咽口水：「P-0217是被标记的意思。」

弹幕反转：
```
？？
他沒被罰
規則4是「不要和車上的人說話」——但NPC先跟他說話的
規則是他在進
哈哈哈哈我就知道
```

年轻女人打开了她手里的包。

包是空的，但她像是在找什么。

周仓看着她的动作，注意到包的底部有一张照片——一对夫妻。那个男人他认识——李大伟。

弹幕飘过：
```
不會是他老婆吧
她找了多久啊
這副本到處都是故事
```

周仓不能说话。但规则没说不能用写的。

他打开系统界面，调出副本结构图中的工牌照片，把屏幕转向她。

年轻女人愣住了。

她看着屏幕上那张工牌——「李大伟——检票员」。

她的手在抖。

她伸手碰了一下屏幕。

屏幕上多了一行字——是她在回复：

「他是我丈夫。他失踪了三年。」

周仓点头。

他关掉屏幕。他知道答案了。第三站快到了。

地铁继续行驶。

就在这时，系统弹出提示：

```
[追踪状态更新]
清除者距离当前位置：约 3 天行程
```

3 天。

第三站快到了。

## 3

第三站到了。

地铁——停了。

屏蔽门——打开。

这是终点站。

周仓应该下车。

他站起来，准备走向屏蔽门。

但他注意到了一件事。

最后一排的老人没有下车。他还坐在那里，没有动。

周仓看着他。老人对他微微点了点头，像是在告别。

周仓走向屏蔽门。

年轻女人也站起来，准备下车。

但她走到屏蔽门的时候停了。她回过头，看着周仓，张了张嘴：「你能帮我找一个人吗？」

周仓看着她。

规则 4——不要和车上的人说话。

但她是在问找人。她是不是在问李大伟？

周仓没有说话。他遵守规则。

但他做了一个动作——他点了点头。

年轻女人笑了。

「谢谢。」她说。

她下车了。

屏蔽门关上了。

地铁继续行驶。

周仓一个人坐在空的车厢里。老人已经不在了，年轻女人也下车了。他一个人。

地铁到了终点。

屏蔽门打开。

他下车了。

系统公告弹出：

```
[副本结算]
副本：末班地铁
结算评价：A
原因：遵守所有规则，到达终点站
积分奖励：+1500
纪念物品：地铁的最后一节车厢钥匙
```

就在系统公告弹出来的同时，车站的监控摄像头转了一下。不是程序设定的那种转动，是有人在另一端操控它。

周仓看着摄像头。摄像头停了一秒，然后转向了他。

他快步走进白光。

他坐在休息室里，想着一件事。

年轻女人问他能不能帮她找一个人。她找的是李大伟吗？

他打开了系统。

他查了年轻女人的身份。

```
[失物主人：年轻女人]

姓名：林小雨
身份：李大伟的妻子
状态：在世
位置：副本内部消息
```

林小雨是李大伟的妻子。不是女儿。

她让周仓帮她找人。她找的是李大伟。她不知道李大伟已经失踪了。她在等他。

周仓打开了「现实世界连接」。

他给林小雨发了一条匿名消息。

```
[匿名消息]
收件人：林小雨

李大伟在末班地铁的最后一节车厢等过你。

他让我告诉你他想你和小燕。

——一个陌生人
```

他发送了。关掉系统。躺在休息室的床上。

他很累。

但他很满足。

他又帮了一个人。

━━━
