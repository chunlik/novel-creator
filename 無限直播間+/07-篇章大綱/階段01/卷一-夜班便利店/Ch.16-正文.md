# Ch.16：末班地铁

## 1

周仓睁开眼。

他站在一个——地铁站里。

不是便利店的入口。

不是档案馆。

不是失物招领处。

是——

一个真正的——

末班地铁的——

站台。

站台——

没有人。

但灯光——

是亮着的。

墙上挂着一个钟：

```
23:45
```

23:45。

末班地铁——

还有 15 分钟。

周仓环顾四周。

站台很安静。屏蔽门是关着的。轨道上看不到任何车。

他打开了系统。

```
[副本信息]
副本：末班地铁
类型：规则限制类
难度：二星（已降星）
时长：1 小时
通关条件：乘坐末班地铁到达终点站
注：副本难度已降低，规则已简化。
```

通关条件——

坐末班地铁到终点站。

但——

副本是「规则限制类」。

有规则。

他打开了规则。

```
[副本规则]

1. 末班地铁只会停靠三站。
2. 你必须从第一站坐到第三站。
3. 中途下车——副本失败。
4. 不要和车上的人说话。
5. 如果有人下车——不要跟下去。
```

五条规则。

周仓看着这些规则。

规则 4「不要和车上的人说话」——

这是一个社交限制。

规则 5「如果有人下车——不要跟下去」——

这是一个行为限制。

他——

不能违反这些规则。

如果违反了——

副本会失败。

他想了一下。

他接受了这些规则。

他站在站台上——

等末班地铁。

23:50。

远处——

传来了声音。

地铁进站的声音。

由远到近。

然后——

灯光——

从隧道深处——

照过来。

末班地铁——

到了。

地铁——

停在了站台前。

屏蔽门——

打开。

周仓——

走进了地铁。

地铁——

和他记忆中的——

不一样。

不是普通的地铁车厢。

是——

一节——

空的车厢。

只有一节。

座位——

是空的。

但——

最后一排——

有一个人。

一个——

老人。

穿橙色马甲的老人。

他——

坐在最后一排的座位上。

他——

看着周仓。

他——

没有说话。

周仓——

也没有说话。

规则 4 说——不要和车上的人说话。

他——

遵守规则。

他——

在中间的位置——

坐下来。

他——

看着前方。

地铁——

开始动了。

屏蔽门——

关上了。

地铁——

驶入隧道。

车厢里——

很安静。

只有地铁行驶的声音。

周仓——

坐在座位上。

他没有看那个老人。

他——

在想——

李大伟。

李大伟——

是检票员。

他的工牌——

出现在了失物招领处。

他——

失踪了。

但他的灵魂——

还在等待。

李大伟——

是——

这个副本里——

那个老人吗？

他——

不确定。

他——

等着看。

## 2

第一站到了。

地铁——

停了。

屏蔽门——

打开。

但——

没有人上车。

也没有人下车。

屏蔽门——

又关上了。

地铁——

继续行驶。

周仓——

数着站数。

第一站过了。

第二站——

到了。

地铁——

停了。

屏蔽门——

打开。

这一次——

有一个人——

上车了。

一个——

年轻女人。

她穿着普通的衣服。手里拿着一个包。

她——

在周仓对面——

坐下了。

她——

看着周仓。

她——

没有说话。

周仓——

也没有说话。

规则 4——不要和车上的人说话。

他——

不违反规则。

地铁——

继续行驶。

年轻女人——

她——

打开了手里的包。

包——

是空的。

但她——

像是在找什么。

她——

翻了很久。

然后——

她——

抬起头。

她——

看着周仓。

她——

张了张嘴——

想说什么。

但她——

没有说。

她——

又低下头——

继续翻包。

周仓——

看着她的动作。

他——

注意到了一个细节。

她的包——

虽然空——

但包的底部——

有一张照片。

他——

能看到照片的一角。

照片上——

是——

一对夫妻。

那个男人——

他——

认识。

那是——

李大伟。

年轻女人——

是李大伟的——

谁？

她——

是李小燕吗？

他——

不确定。

但他——

想起了规则 4。

不要和车上的人说话。

他——

不说话。

地铁——

继续行驶。

第三站——

快到了。

## 3

第三站到了。

地铁——

停了。

屏蔽门——

打开。

这是终点站。

周仓——

应该下车。

他站起来——

他准备走向屏蔽门。

但他——

注意到了一件事。

最后一排的老人——

他没有下车。

他——

还坐在那里。

他没有动。

周仓——

看着他。

他——

看着那个老人。

老人——

对他——

微微点了点头。

像是——

在告别。

周仓——

走向屏蔽门。

年轻女人——

也站起来——

准备下车。

但她——

走到屏蔽门的时候——

她——

停了。

她——

回过头。

她——

看着周仓。

她——

张了张嘴——

「你——」她说，「能帮我——找一个人吗？」

周仓——

看着她。

规则 4——不要和车上的人说话。

但她——

在问——

找人。

她——

是不是——

在问——

找李大伟？

周仓——

没有说话。

他——

遵守规则。

但他——

做了一个动作。

他——

点了点头。

年轻女人——

她——

笑了。

「谢谢。」她说。

她——

下车了。

屏蔽门——

关上了。

地铁——

继续行驶。

周仓——

一个人——

坐在空的车厢里。

老人——

已经不在了。

年轻女人——

也已经下车了。

他——

一个人。

地铁——

到了终点。

屏蔽门——

打开。

他——

下车了。

系统公告弹出：

```
[副本结算]
副本：末班地铁
结算评价：A
原因：遵守所有规则，到达终点站
积分奖励：+1500

```

白光——

出现。

周仓——

被传送回了主系统大厅。

他坐在休息室里。

他——

想着一件事。

年轻女人——

问他——

能不能帮她找一个人。

她——

找的是——

李大伟吗？

他——

打开了系统。

他查了——

年轻女人的身份。

```
[失物主人：年轻女人]

姓名：林小雨
身份：李大伟的妻子
状态：在世
位置：副本内部消息
```

林小雨——

是李大伟的妻子。

不是女儿。

她——

让周仓帮她找人。

她——

找的是——

李大伟。

她——

不知道李大伟已经失踪了。

她——

在等他。

周仓——

打开了「现实世界连接」。

他——

给林小雨——

发了一条匿名消息。

```
[匿名消息]
收件人：林小雨

李大伟——

在末班地铁的——

最后一节车厢——

等过你。

他——

让我告诉你——

他——

想——

你——

和——

小燕。

——一个陌生人
```

他——

发送了。

他——

关掉系统。

他——

躺在休息室的床上。

他——

很累。

但他——

很满足。

他——

又帮了一个人。
