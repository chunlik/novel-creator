# Ch.18：第十一层

## 1

一天后。

周仓——在主系统大厅——接了下一个副本。

「第十一层」。

这是一个——探索类副本。

探索类——不是规则限制。

是要他在副本里找东西。

副本简介：

```
[副本详情 - 第十一层]

类型：探索类
难度：三星
时长：1 天

简介：
一个普通的写字楼。
但它有 11 层。
而租户名册——只记录了 10 层。
第十一层没有人知道它是什么。

通关条件：
找到第十一层的「租户」。

注意：
本副本没有战斗机制。
但有「逻辑推理」机制。
玩家需要根据线索——推理出答案。
```

周仓——接了这个副本。

白光——出现。

他——进入了——副本。

## 2

周仓睁开眼。

他站在一个——写字楼的大厅里。

但这个大厅——和普通的写字楼大厅——不一样。

普通大厅——是玻璃、金属、明亮。

这个大厅——是——旧。

地板是深色的木板。墙壁是浅黄色的壁纸。灯是老式的吊灯。

像是——1980 年代的写字楼。

周仓环顾四周。

前台——有一个桌子。

桌子上——有一个名册。

名册是一本老式的牛皮纸笔记本。

他翻开名册。

```
[租户名册]

一层：大堂经理 张明
二层：会计事务所
三层：律师事务所
四层：设计公司
五层：投资咨询
六层：贸易公司
七层：广告公司
八层：信息技术
九层：市场调研
十层：总裁办公室

——没有第十一层。
```

名册——记录到第十层。

没有第十一层。

但这个写字楼——有 11 层。

第十一层——存在。

但名册上——没有记录。

周仓——走进电梯。

电梯——是关着的。

他按了按钮。

没有反应。

电梯——坏了。

又是电梯坏了。

他需要——走楼梯。

他找到了楼梯口。

楼梯——往上走。

他开始爬。

一层。

二层。

三层。

四层。

五层。

六层。

七层。

八层。

九层。

十层。

他到了第十层。

他——继续往上走。

第十层到第十一层——有一段楼梯。

但这段楼梯——和之前的——不一样。

之前的楼梯——是水泥的。

这段楼梯——是——木头的。

像是——临时加的。

像是——这栋楼——原本只有 10 层。

但有人——后来——加了一层。

加在了顶上。

周仓——继续往上走。

他到了——第十一层的门口。

门口——有一扇门。

门是木门。

老式的木门。

门上——有一个牌子。

牌子上写着：

「第十一层 - 档案室」

档案室。

他推开门。

## 3

里面——是一个小房间。

和下面 10 层——完全不一样。

不是办公区。

是——档案室。

四面墙——都是文件柜。

文件柜——是铁的。老式的铁皮柜。

每个柜子——都有一个标签。

他——开始看标签。

```
A1：2010 年文件
A2：2011 年文件
A3：2012 年文件
...
A10：2019 年文件
B1：2020 年文件
B2：2021 年文件
B3：2022 年文件
B4：2023 年文件
B5：2024 年文件
B6：2025 年文件
B7：2026 年文件
```

文件柜——按年份——排列。

但有一个柜子标签是不同的。

```
C1：特殊档案
```

C1——是特殊档案。

他——打开 C1 柜子。

里面——只有一个文件。

文件标题是：

「2018 年 - 副本计划」

周仓——打开文件。

第一行写着：「2018 年——主系统启动副本计划。」

「计划内容：在主系统内建立独立副本。计划目标：让失败者有地方可去。计划进展：已完成。」

「但——我们发现了一个问题。」

「副本——不是给失败者的——是给——活人的。」

「失败者不应该在副本里——他们应该在主系统等待下一个机会。」

「但他们不愿意等待。他们愿意在副本里陪伴其他失败者——他们把副本当成了家。」

「这是计划的意外。但我们决定接受这个意外——因为这是他们的选择。」

「如果他们愿意在副本里陪伴其他失败者——我们不应该阻止。」

「副本——不是给失败者的。是给——愿意留下陪伴的人的——家。」

周仓——看完了文件。

弹幕缓缓飘过：
```
原來副本是家
不是監獄
是他們自己選的
自願留下陪其他人
```

他——明白了。

副本——不是主系统——「惩罚」失败者的——是——失败者自己——选择的——「家」。

许小小——选择留在便利店副本——是因为她想陪林东。

林东——选择留在便利店副本——是因为他想陪许小小。

李福生——选择留在旧医院——是因为他没有家。

但医院——是他的家。

副本——不是牢笼。

是——选择。

周仓——看完了文件。

系统提示弹出：

```
[追踪状态更新]
清除者状态：已定位副本出口
——警告：清除者即将离开失物档案馆——
——玩家周仓请注意——
——清除者将在离开后重新校准坐标——
```

周仓——盯着最后一行。
重新校准。
这意味着——清除者——很快就能追上他了。
之前的 7 天流速差——只在档案馆内有效。
一旦清除者离开档案馆——他会直接出现在周仓所在的地方。

他——没有太多时间了。

系统公告弹出：

```
[副本结算]
副本：第十一层
结算评价：A
原因：找到了第十一层的真相
积分奖励：+1500
纪念物品：「副本计划」原件
```

白光——出现。

但這次——白光沒有把他帶回去。

一雙手——從他身後——抓住了他的肩膀。然後——把他拽了出來。

周仓摔在地上。面前——是那張熟悉的臉。

獵犬。

「你又來了。」周仓說。

「你騙了我。」獵犬說——「我查了——守門人沒有懸賞你。」

「我知道。」

「那你——」

「我需要你——」周仓爬起來——「出現在這裡。」

獵犬——皺眉。

「清除者快到了。我需要一個——不被系統追蹤的窗口期。」

「你——想讓我拖住清除者？」

「不。清除者你擋不住。但我需要你在副本外面——接應。」

獵犬——沉默了幾秒。

「我為什麼要幫你？」

「因為林東。」周仓說——「你欠他的。」

檔案室的燈——閃了一下。

系統提示彈出：
```
[警告] 清除者即將離開失物檔案館
```

周仓——沒有時間了。

「五分鐘。」獵犬說——「你只有五分鐘。」

周仓——點了點頭。白光——重新亮起。

他——被傳送回主系統大廳。

他——看着系统背包里的「副本计划」原件。

他知道了副本是什么。

副本——是家。

是失败者选择留下的家。

他不再把副本当成必须通关的任务。

他把副本当成可以选择留下或者离开的地方。

## 4

周仓在主系统大厅休息了三天。

在他休息的第一天，大厅入口处两个守门人在交接班。新来的守门人看到周仓，表情变了。「你确定要接他的单？」新守门人小声问。老守门人没有回答。

周仓没有假装没听到。他直接走到两个守门人面前。

「我听到了。告诉我，你们系统里关于P-0217的备注是什么。」

两个守门人同时僵住。

「不说也可以。」周仓说，「我现在就去副本里死一次。到时候异常关联者的调查报告——会写上你们的编号。另外，我手上有副本计划的原件——里面提到'守门人权限层'。你觉得我应该把那块内容截给谁看？」

新守门人脸色发白。

老守门人叹了口气：「P-0217——异常关联者。系统的意思是——优先处理。」

「优先处理——是指清除者？」

老守门人没有回答，但他的沉默就是答案。

周仓点了点头。他转身走了——然后回头：「副本计划原件——我会留着。如果哪天我需要你们配合，那不是请求。」

三天里——他收到了——几条消息。

第一条——来自小宝。

```
[消息]
发件人：小宝
时间：您上次发送后 2 周

周仓：

我找到妈妈了。

我去了她在的城市。

我见到她了。

她抱着我哭。

她说她对不起。

我说我也对不起。

我没有等她。

我不怪她。

我谢谢你。

如果不是你收留了她的钱包我不会有勇气去找她。

谢——谢——你。

——小宝
2026-11-26
```

周仓看着这条消息。

小宝——找到了他的妈妈。

他——松了口气。

第二条——来自林小雨。

```
[消息]
发件人：林小雨
时间：您上次发送后 1 周

陌生人：

我带小燕去了末班地铁的最后一节车厢。

我们坐在那节车厢里。

我们告诉了大伟我们很想他。

我们告诉他小燕考上了高中。

我们告诉他我们会好好活下去。

我们不知道他能不能听到。

但我们说了。

谢——谢——你。

如果不是你我不会知道他已经不在了。

我会一直等他。

但现在我可以继续活下去了。

谢——谢。

——林小雨
2026-11-26
```

周仓看着这条消息。

林小雨——和她的女儿——去看了李大伟。

他——又松了一口气。

第三条——来自李福生。

```
[消息]
发件人：李福生
时间：您上次发送后 1 周

周仓：

你会来看我吗？

我等你李福生
2026-11-26
```

周仓看着这条消息。

李福生——在等他。

他——需要——去看他。

他打开了系统。

他接了「旧医院七楼」副本。

他要——再去看李福生。

白光——出现。

他——进入了——副本。

━━━

【章末钩子】周仓在第十一层副本里找到了「副本计划」原件——副本不是牢笼，是失败者选择的家。他收到了小宝、林小雨、李福生的消息。他决定再次进入旧医院七楼，去看望李福生。

【系统奖励】副本结算 A / 1500 积分 / 副本计划原件
