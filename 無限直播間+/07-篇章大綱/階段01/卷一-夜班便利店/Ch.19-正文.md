# Ch.19：再访

## 1

周仓睁开眼。

他站在了旧医院的大厅里。

和上次一样。绿色的墙壁，灰色的地板，消毒水的味道。

但这次大厅里有一个人——一个穿白色护士服的女人。她站在前台，看着周仓。

「你——」她说，「是来——看李福生的？」

「是。」

「他——」她说，「等你很久了。」

「多久？」

「从你——」她说，「离开——到现在——他每天——都——问——'那个年轻人会来吗'。」

周仓没有说话。他往楼上走。

一层。

二层。

三层。

四层。

五层。

六层。

七层。

他到了七楼，走进第三间病房。

李福生还坐在病床上。他老了——比上次更老了，脸上多了皱纹，眼睛也更浑浊了。但他看到周仓的时候眼睛亮了。

「你——」他说，「来了。」

「是。」周仓说。

「你——」李福生说，「真的——来了。」

「我说过——」周仓说，「我会来。」

李福生笑了，笑得很开心。

「我——」他说，「等你——很久——了。」

「我知道。」

「我——」李福生说，「以为——你不——会来——了。」

「我会来。」

「你——」李福生说，「每次——来我——都很——开心。」

周仓在病床旁边坐下来，看着李福生。

他想问李福生一个问题。

「李福生。」

「嗯。」

「你——」周仓说，「的女儿——她——为什么不来看你？」

李福生——沉默了很久。

「她不知道我在这里。」

「你没有告诉她？」

「没有。」

「为什么？」

「因为——」李福生说，「我不想让她知道我这么可怜。老了不能动了，需要别人照顾。别人不想照顾你——你就是可怜。」

「你不是可怜。你只是老了。需要照顾是人之常情。」

李福生看着他，低着头，声音在抖：「你不觉得我可怜、麻烦、是负担？」

「不觉得。」

李福生哭了。三年来第一次在别人面前哭。

周仓没有说话。他陪着李福生，听他哭。没有离开。

## 2

一个小时后。

李福生哭完了。他擦干眼泪，看着周仓。

「周仓。」

「嗯。」

「你能帮我做一件事吗？」

「什么事？」

「你能帮我找我女儿吗？」

「可以。」

「你能告诉她——我在这里等她吗？」

周仓看着李福生，点了点头。

李福生笑了。他笑得很开心——三年来第一次这么开心。

周仓打开了系统，查了李福生女儿的信息。

```
[查询结果]

姓名：李小梅
关系：李福生的女儿
状态：在世
位置：另一个城市
最后联系：2015 年
```

最后联系——2015 年。11 年前。

李福生 11 年没有和女儿联系。

周仓打开了「现实世界连接」。

他给李小梅发了一条消息。

```
[匿名消息]
收件人：李小梅

你爸爸在旧医院七楼等你。

他不敢联系你。

他怕麻烦你。

但他想你。

请去看看他。

——一个陌生人
```

他发送了，关掉系统，看着李福生。

「我——」他说，「已经——发消息了。」

李福生——看着他。

「她——」他说，「会来——吗？」

「我不知道。」周仓说，「但她——收到了消息。」

「谢谢——你。」李福生说。

「不谢。」

周仓站起来，准备离开。

「周仓。」

「嗯。」

「你——」李福生说，「以后——还会——来看——我吗？」

周仓看着李福生，笑了。

「会。」

「我——」他说，「会经——常来——看你。」

李福生笑了，笑得很开心。

周仓——走出第三间病房。

走廊里——站着一个人。
不是护士。
不是病人。
是一个——穿黑色雨衣的人。
黑色——和便利店的红雨衣——不一样。
但他的姿态——一样。
清除者。

「P-0217。」
清除者的声音——不是人的声音。
是金属的摩擦声像是生锈的齿轮。
「异常关联者——确认。」
「跟我走。」

周仓——退后一步。
清除者——向前一步。

周仓——看了一眼走廊尽头的窗户。
七楼。

他——没有犹豫。
他——冲过去——
他——推开窗——
冷风——灌进来。
他——没有看下面。
他——跳了出去——

风——在他耳边——呼啸。

白光——出现。

他——被传送回了主系统大厅。

## 3

三天后。

周仓收到了李小梅的回复。

```
[消息]
发件人：李小梅
时间：您上次发送后 3 天

陌生人：

我去了旧医院七楼看我爸爸。

他很老了。

他不认识我了。

但我认识他。

我抱着他哭了很久。

护士说他这三年来没有人来看过他。

但他每天都问有没有人来。

护士说最近有两个人来过。

一个是你。

你——是——谁？

你为什么要帮我爸爸？

我不认识你。

但谢谢你。

如果不是你我不会知道他在旧医院等我11年。

我不会来看他。

我会一直以为他不想见我。

但——他——想。

他每天都在想。

我不会再让他一个人了。

我会经常来看他。

谢——谢——你。

请问你是谁？

——李小梅
2026-11-29
```

周仓——看着这条消息。

李小梅去看她爸爸了。

她不再以为她爸爸不想见她了。

她会经常去看他。

他——帮了——李福生。

他也帮了李小梅。

他——松了一口气。

他——没有——回复。

他——不需要——回复。

他只是一个陌生人。

他做了他应该做的事。

他帮了一个需要帮助的人。

他不需要被感谢。

他只需要看到李福生在笑。

━━━

【章末钩子】周仓再次去旧医院七楼看李福生——他帮李福生联系到了 11 年没见的女儿李小梅。李小梅去医院看了她的父亲。父女重逢。周仓不再需要被感谢——他只是做了他应该做的事。
