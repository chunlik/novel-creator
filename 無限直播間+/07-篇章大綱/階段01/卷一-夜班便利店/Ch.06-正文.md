# Ch.06：许小小的异常

## 1

凌晨五点半。

距离副本结束还有一个半小时。

周仓没有睡。他在交接本上整理自己剩下的积分和情报。

积分余额：400。
已获得情报：1 条（冰柜警告）。
已观测异常：5 项。

他在本子上写：

```
许小小的异常：
1. 知道货架会自己变动，一直手动调整。
2. 拿牛奶盒但不喝——说「不记得味道」。
3. 说出副本剩余时间，语气像陈述而非猜测。
4. 存在时间长——说「在这里很久了」。
5. 观众视角——她真的是 NPC 吗？
```

他停在第五个问号上。

从进副本到现在，他从来没有确认过一件事：弹幕能不能看到许小小？

他记得有人提到过「店长」，但具体内容他不确定。

他抬起头。许小小坐在收银台旁边，还在刷手机。他看了一眼视野右角的弹幕区。

弹幕在正常滚动。

```
还有 一个半小时
猜他能活多久
他活了两个小时就破了值班表的秘密 很强了
但许小小到底是不是 NPC
她刚才那句话很奇怪
「你还有两个小时」——我听着像立flag
```

他们能看到她。

他们在讨论她。

但没有人说「她不是 NPC」。

周仓想了一下，开口了。

「许小小。」

「嗯？」

「你记得你第一天来这里上班是什么时候吗？」

她放下手机，看了他一眼。

「第一天？」

「对。你第一天值夜班。」

她沉默了一会儿。不是在想——像在找一段数据。

「2023年11月7号。」

周仓的动作顿住了。

2023年11月7号。

和他下午在货架上看到的那包过期泡面的日期一样。

同一串数字。

他的后背一阵发凉。

「你怎么记得这么清楚？」

许小小眨了眨眼睛。一副「这还需要记吗」的表情。

「因为那天之后，店里的灯就再也没关过。」

## 2

周仓没有立刻接话。

他在想。2023年11月7号——便利店从那天开始灯就没有关过。也就是说，它从那一天开始变成了 24 小时营业。或者说——从那一天起，这里的「时间」就不再正常了。

「那之前呢？」

「之前？」

「你之前在哪上班？」

许小小把手机屏幕按灭，把手机放在膝盖上。她像是在认真回想。

「之前……在一家奶茶店。」

「哪家？」

「忘了。不记得名字了。」

「怎么会忘？」

她歪了歪头。

「不知道。就是忘了。我记得我在那里工作过，记得穿围裙、做奶茶、收钱——但店名叫什么，在哪里，去了多久——都忘了。」

她说得很轻松。像在说一件无关紧要的小事。

但周仓不觉得轻松。

他开始注意到一些细节。

许小小的记忆——不是连续的记忆。她知道自己做过什么，但不知道在哪里、什么时候。她记得一些碎片——片段式的——但没有一条线把它们串起来。

弹幕也在讨论。

```
她不是 NPC 吧
NPC 不会有这么完整的背景断片
但她记得日期 2023-11-07
那是便利店有变化的日子
也就是说 她是「之后」才出现的
```

周仓决定做一个测试。

他从收银台抽屉里抽出一张便签纸，在上面写了一行字。然后他把便签纸折好，放在柜台下面。

「许小小，帮我拿一盒牛奶。」

「你刚才不是吃过了吗？」

「想喝。」

她站起来，走向冰柜。她打开柜门——不是那个里面有保鲜盒的冰箱——是饮料冰柜，拿出一盒牛奶。然后她转过身，正要走回来。

走了两步。

她停了。

她低头看手里的牛奶盒。

不是 CH-0217 编号的那盒——但她的表情变了。

「怎么了？」

「没。」

她把牛奶盒放在柜台上，没有离开。她站在那里，看着那盒牛奶。

「周仓。」

「嗯？」

「这个牛奶盒——我是不是喝过？」

「你下午拿过一次。」

「不是下午。」她说，「是很久以前。很久很久。我记得——我在喝这个牛奶的时候，那天在下雨。外面很吵。店里有人在吵架还是什么。」

她说不下去了。她的表情很困惑——不是痛苦的困惑，是「我明明记得有这件事，但为什么细节全没了」的那种困惑。

周仓看着她。

「你记得下雨。」

「对。」

「你记得有人在吵。」

「……对。」

「但你记得店在哪里吗？」

她沉默了很久。

然后她摇了摇头。

「不记得了。」

「你记得你自己当时穿的什么吗？」

她又想了一会儿。

「围裙。奶茶店的围裙。白色那条。」

「你不记得奶茶店的名字，但你记得围裙的颜色。」

「对。」

周仓没有再问。

他已经确认了一件事。

许小小的记忆——不是丢失，是被人为地删除了。

只留下了某些特定的碎片。

而那些碎片——和便利店有关。

和牛奶盒有关。

和这个副本有关。

他看着她。她没有注意到他的表情。她还在看那盒牛奶。

「你还好吗？」

她抬起头，笑了笑。笑得和之前一样——轻柔的、不设防的——

但眼里没有笑意。

「还好。」

她说。

「就是——有时候会觉得，有些事情我好像应该记得。但我不记得了。」

她说得很小声。

周仓没有回应。他把那盒牛奶接过来，没有喝。

他在那行「许小小的异常」下面写了一行新的字：

```
6. 她的记忆被删过。碎片指向便利店和这个副本。
```

他停了一下，又写了第七行：

```
7. 弹幕能看到她——但没人说她是 NPC。
   说她是 NPC 的，都用了「吧」。
   「她是 NPC 吧」——猜测。
   不是肯定句。
```

## 3

凌晨六点。

天快亮了。

便利店的玻璃外面开始透进一点点光——不是阳光，是黎明前的那一层灰蓝色。

许小小又回到了她的椅子上。她没有再刷手机。她抱着膝盖，看着门口的方向。

周仓坐在收银台后面，等着什么。

他没有等到时间到。

他等到的是另一件事。

他的视野左上角弹出一条提示——不是系统的蓝色。

是红色的。

```
⚠ 警告：副本状态异常。

检测到播放在未授权条件下运行。
检测到播放器存在数据残留。

建议主播在结束后主动联系系统客服。
```

周仓盯着那条红色警告。

他没有关掉它。

它在他的视野角落停留了五秒，然后自动消失。像是从来没有出现过一样。

但那条消息的内容留在了他的脑子里。

「未授权条件下运行。」
「数据残留。」

他看向许小小。

她坐在椅子上。抱着膝盖。看着门外，像是在等什么人。

他想起她说的那句话：

「总觉得这个盒子我应该喝过，但我不记得味道。」

「有一些事情我好像应该记得。但我不记得了。」

他明白了。

许小小不是 NPC。

她可能是这个副本的前任玩家。

她的档案没有被完整删除——她被残留在这里。

坐在收银台旁边，抱着膝盖，看着门外。

等一个人来，能把她从这里带出去。

但没有人来过。

直到周仓。

他没有说穿。他不能让任何人知道他已经猜到了。

他只是把那条警告的内容抄在交接本上，用很小的字，写在最后一面。

然后他合上本子，站起来。

「许小小。」

「嗯？」

「天亮了。」

她转头看窗外。灰蓝色的光——已经变成了更亮的蓝色。

她笑了一下。这一次——笑到眼睛里了。

「嗯。天亮了。」
