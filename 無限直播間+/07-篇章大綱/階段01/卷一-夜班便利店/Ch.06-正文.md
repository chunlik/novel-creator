# Ch.06：追杀

## 1

周仓盯着自己的手腕。

那两个字——「CH」——是浅灰色的。像打印上去的，但比普通的文字更细。像是刻在皮肤下面，从里面浮出来的。

他没有动。

他站在那里看了十秒。

他用右手指尖碰了一下那两个字。

触感——和普通皮肤一样。没有凸起，没有凹陷。但他能感觉到那两个字「在」。

他用袖子把它遮住。

弹幕刷过：

```
他的手腕上有字了
CH 开头
CH-0217？
他被「标记」了
是红雨衣标记的吗
它想把他变成商品
```

周仓没有看弹幕。

他打开任务系统。

一个新的任务已经触发：

```
[紧急] 处理手腕上的 CH 标记
      状态：标记正在生长
      预计完全浮现时间：6 小时
      后果：未确认
      建议：立即处理
```

6 小时。

他现在还剩 18 个小时。

但这个 CH 标记——6 小时后会完全浮现。

完全浮现后会怎样？系统没说。

他点开商城——

```
┌──────────────────────────────────┐
│  商城                            │
│──────────────────────────────────│
│  [紧急商品]                     │
│  CH 标记抑制贴     800 积分     │
│  状态：可购买                    │
│                                  │
│  [新] 紧急修复胶囊  500 积分     │
│  说明：可清除身体上的副本标记    │
│  警告：使用后 12 小时内不可再购买│
└──────────────────────────────────┘
```

CH 标记抑制贴——800 积分。

紧急修复胶囊——500 积分。

他现在有 1050 积分。

两个都能买。

但抑制贴只能抑制——不能清除。胶囊能清除——但有 12 小时冷却。

他选胶囊。

```
[已购买] 紧急修复胶囊 -500 积分
[剩余] 550 积分
```

他的视野里出现了一个小红点。点击后——一颗白色胶囊出现在他的右手掌心里。

他把胶囊放进嘴里。

咽下去。

手腕上的「CH」——

淡了。

从浅灰变成几乎看不见。

然后消失。

任务状态更新：

```
[已处理] CH 标记已清除
```

但他没有松一口气。

因为他知道——红雨衣标记他一次，会标记第二次。

他必须搞清楚一件事：它为什么标记他。

## 2

06:00。

便利店进入白班时段。

许小小从后场走出来。她的脸色比之前好了一点。

「我刚才——」她开口，「你手腕上是不是有字？」

周仓看了她一眼。

「你怎么知道？」

「林东以前也这样。」她说，「他手腕上先出现 CH，然后——」

她停了一下。

「然后怎样？」

「然后他就变成红雨衣了。」

周仓沉默。

红雨衣是先被标记 CH，然后变成那个样子的。

他被标记 CH——意味着它想让他变成下一个红雨衣。

「许小小。」

「嗯。」

「林东变成红雨衣之前——他做了什么？」

「他——」她想了想，「他连续三天没睡。」

「为什么？」

「他说他在算什么东西。他一直在算。」

「算副本的运行规则。」周仓说。

「对。他想找出规则里的漏洞。」

「他找到了吗？」

许小小看着他。

「他找到了。」她说，「所以他'死'了。」

弹幕飘过：

```
他找到了漏洞
但他死了
那他找的漏洞是——什么？
```

周仓在心里想了一件事。

林东设计了这个副本。他知道规则。他知道漏洞。

但他「死」了。

这意味着——他知道漏洞在哪里，但没有办法在不违反规则的情况下使用漏洞。

他要违反规则才能使用漏洞。

违反规则的代价——就是变成红雨衣。

「许小小。」

「嗯？」

「林东找到的漏洞——他告诉你了吗？」

她摇了摇头。

「他没说。但他留了一样东西。」

「什么东西？」

她从口袋里拿出一张纸条。

纸条很旧。边角都卷了。

上面写着几行字——手写的字迹，字有点抖：

```
如果有人能看到这张纸条：

副本有一个隐藏的存档点。
位置：CS 档案管理处。
权限：CS-0002。

我去拿我的存档。
如果你能拿到我的存档——
你就有权限把我带走。

——林东
2023-11-08
```

周仓看着那张纸条。

CS 档案管理处。

他在任务列表里看到过这个位置。

```
[线索] CS 档案管理处
      位置：便利店外东南方向 200 米
      状态：尚未探索
```

「许小小。」

「嗯。」

「三年了。这张纸条你一直带着？」

她点头。

「为什么不交给系统？」

「因为系统不收。」她说，「它只看编号，不看纸条。」

「那张纸条——」

「它是一把钥匙。」她说，「CS-0002 的权限钥匙。只要这张纸条在——」

她笑了一下。

「林东就有被带走的机会。」

弹幕刷过：

```
她三年没扔这张纸条
她每天带着林东的权限钥匙
她在等他
等一个能带她走的人
等一个能带林东走的人
```

周仓把纸条收起来。

「我会带你们两个离开。」

许小小看着他。

她没有说「别说这种话」。

她只是看着他。

「好。」她说。

## 3

06:30。

便利店进入白班。

周仓正准备去探索 CS 档案管理处——

便利店的自动门开了。

不是风铃响——是自动门。

周仓立刻抬头。

门口——

红雨衣。

但这次——它不是慢慢走进来。

它冲进来。

速度——

比他之前看到的快三倍。

弹幕炸裂：

```
它冲进来了
速度不对
它在追什么
它在追他
```

周仓没有时间反应。

他本能地蹲下来——从收银台后面——

红雨衣冲过收银台。

他的手——苍白的、像冰冻过的手——从周仓的头顶掠过。

差一点。

只差五厘米。

周仓滚到了货架区。

他蹲在货架的尽头——第一排和第二排之间——躲着。

红雨衣没有停。

它转向货架区。

它开始扫视。

它知道周仓在这里。

它开始往第一排货架走。

周仓的呼吸稳着——但他听到了红雨衣的脚步。

每一步都重。

像踩在地板上要把地板踩穿。

弹幕滚过：

```
他躲起来了
红雨衣在找他
它会找到他吗
```

周仓看了一眼视野里的小红点——紧急修复胶囊。

他已经用过了。12 小时内不能再用。

他看了一眼商城。

```
┌──────────────────────────────────┐
│  商城                            │
│──────────────────────────────────│
│  道具：                          │
│  烟雾弹           300 积分       │
│  说明：制造 3 米烟雾区域         │
│  持续时间：30 秒                  │
│                                  │
│  工具：                          │
│  手电筒           100 积分       │
│  说明：照亮 5 米范围              │
│                                  │
│  状态：                          │
│  紧急护盾         500 积分       │
│  说明：抵挡一次直接攻击          │
│  [紧急商品 - 暂时缺货]           │
└──────────────────────────────────┘
```

紧急护盾缺货。

烟雾弹 300 积分。

手电筒 100 积分。

他现在 550 积分。

他买了烟雾弹。

```
[已购买] 烟雾弹 -300 积分
[剩余] 250 积分
```

他的右手出现了一个黑色的圆筒。

他等着。

红雨衣走到了第一排货架的尽头。

它离他只有两米。

他的手——伸出货架的间隙——

按下烟雾弹的开关。

白色烟雾瞬间扩散。

红雨衣的脚步停了。

它看不见他。

周仓在烟雾中移动——他不知道往哪里走——他只能跑。

他冲向便利店的后场门。

他推开门。

后场是一个小房间——监控室、储物间、还有一个通向便利店后门的小门。

他冲向那个后门——

门锁着。

他推不开。

他转过身。

烟雾开始在房间里扩散。

红雨衣的轮廓在烟雾中出现。

它没有追他。

它站在后场门口。

它在看他。

雨帽下——

周仓第一次看清了它雨帽下的脸。

不是脸。

是空。

一片漆黑的洞。

没有眼睛。没有鼻子。没有嘴。

只有——黑色。

在那个黑色的洞里——

有一个声音。

很轻。

「你——」

是林东的声音。

但不是活人说话的方式。

是录音。

是循环播放的录音。

「你不应该——进来。」

周仓站在原地。

红雨衣——林东——站在他面前。

它的手伸出来。

它要碰他。

但——

叮——

系统的声音突然在他的视野里响起：

```
[被动触发] 永久能力【时间校准】

检测到副本时间异常波动。
已自动激活时差平衡。
主观时间与副本时间差异降为 0。
```

红雨衣的动作——

慢了下来。

不是停止。是变慢。

像有人在它的动作上加了一个减速器。

周仓看到了机会。

他从红雨衣的身边冲过去。

它伸出的手——从他背后掠过——

没有碰到他。

他冲回了便利店。

他冲过收银台。

他冲过自动门。

他冲到了便利店外面的街道上。

红雨衣没有追出来。

它站在门口。

它看着他。

然后它——

退回了便利店。

门关上了。

周仓站在便利店外面的街道上。呼吸粗重。

晨光打在他的脸上。

他活下来了。

他看着自己刚才被红雨衣摸过手腕的位置。

那里——

没有新标记。

但他注意到——

红雨衣的「没有脸」——

在他冲出便利店的那一瞬间——

有一秒——

那张黑色的洞里——

闪过了一张脸。

一张正常的脸。

一张——

他没有见过的脸。

但那张脸——

是笑着的。

━━━
