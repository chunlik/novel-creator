# Ch.17：旧医院七楼

## 1

一天后。

周仓在主系统大厅休息了一天。

他打开了系统。

他看到了林小雨的回复。

```
[消息]
发件人：林小雨
时间：您发送后 2 小时
状态：已读

陌生人：

我收到了你的消息。

你说大伟在末班地铁的最后一节车厢等过我。

他想我和小燕。

我——我不知道他已经不在了。

我以为他只是加班——我等了他三个月。

我以为他会回来。

我谢谢你。

我不知道你是谁。

但你告诉我了真相。

我会带小燕去看他。

我会告诉她她爸爸想她。

我谢谢你。

我永远记得你。

谢谢。

——林小雨
2026-11-19
```

周仓看着这条消息。

他帮了李大伟，帮了林小雨，帮了李小燕。

他松了一口气。

他打开了任务大厅。

他要接下一个副本。

```
[任务大厅 - 新副本列表]

1. 副本：旧医院七楼
   类型：恐怖类
   难度：四星（已降星）
   奖励：2000 积分
2. 副本：开学日
   类型：日常类
   难度：一星（已降星）
   奖励：800 积分
3. 副本：第十一层
   类型：探索类
   难度：三星
   奖励：1500 积分
4. 副本：最后一天
   类型：剧情类
   难度：三星
   奖励：1200 积分
```

周仓看着列表。

旧医院七楼——是恐怖类。但他已经降到了四星。他可以试试。

他接了旧医院七楼。

```
[已接受副本] 旧医院七楼
开始时间：现在
```

白光出现。

他进入了下一个副本。

## 2

周仓睁开眼。

他站在一个医院的大厅里。但这个医院不是普通的医院，是旧的医院。

墙壁是绿色的油漆，油漆在剥落。地板是灰色的瓷砖，瓷砖有裂缝。空气中有一股消毒水的味道，但消毒水的味道有点发霉。

周仓环顾四周。医院大厅没有其他人，只有他一个人。

他打开了系统。

```
[副本信息]
副本：旧医院七楼
类型：恐怖类
难度：四星（已降星）
时长：3 天
通关条件：在医院七楼找到「病人」
注：副本难度已降低
注：本副本没有规则限制
注：但有「心理压力」机制
```

通关条件——在七楼找到「病人」。

没有规则限制，但有「心理压力」。这个副本不是靠规则，是靠心理。他需要承受住心理压力。

大厅的挂号台没有人。但台面上放着一张纸。纸上写着一个名字：P-0217。下面有人用笔圈了两圈。

周仓看着那张纸。挂号台的笔还转着，刚放下不久。

他走向电梯。电梯是关着的。他按了按钮，没有反应。电梯坏了。

他需要走楼梯。

他找到了楼梯口。楼梯很旧，扶手是金属的，已经生锈。

他往上走。

一层。

二层。

三层。

四层。

五层。

六层。

每上一层，楼梯的灯光就暗一点。六层已经是半暗了。

他继续往上走。

七层到了。

楼梯口旁边的公告栏上贴着一张发黄的报纸剪报。剪报标题：「市第七人民医院精神科集体中毒事件——15人送医」。日期：2023年3月。

周仓看了一眼，没有多想。他把剪报折好放进口袋。

他走到护士站：「2023年集体中毒的名单——给我。」

护士抬头看他：「你是——」

「我是其中一人的家属。」周仓说。撒谎，但他没有犹豫。「15个人。我需要知道他们的名字。」

护士犹豫了一下，然后她打开了抽屉，拿出一份泛黄的文件夹。「你是第一个来问的人。」她说。

他走向楼梯口的那扇门。门上写着：「七楼 - 病人区」

周仓推开了门。

门后是一条走廊。走廊很长，两侧是病房。病房的门是关着的，但门上的小窗能看到病房里面。

周仓走进走廊。他开始看病房里面。

## 3

第一间病房是空的。病床是空的，没有病人。但病床上有被子的痕迹——像有人刚刚躺过。

周仓继续走。

第二间病房也是空的。但床头柜上有一个水杯，水杯是温的。有人刚刚在这里。

他继续走。

系统提示弹出：

```
[追踪状态更新]
清除者距离当前位置：约 1 天行程
```

还剩一天。

第三间病房有人。一个老人坐在病床上，看着周仓。他没有说话。

周仓也没有说话。

他需要找到「病人」。但他不知道这个老人是不是「病人」。

他走进第三间病房：「你是病人？」

老人看着他，点了点头：「是。」

「你在这里多久了？」

「很久。」老人说。

「多久？」

「我不记得了。」

「你为什么在这里？」

老人看着他，眼睛里有了泪：「因为我没有地方去。我的家人都走了，我没有家。」

「所以你来了这里？」

「医院会收留我。」

周仓看着老人。他明白了。这个「病人」不是生病的病人，是无家可归的病人。他没有家，在医院等死。

他想有人来看他，想有人记得他。但没有人来。他一个人在医院里等。三年。五年。十年。他不记得了。

弹幕安静了几秒：
```
……
太難受了
這副本太難了
不是戰鬥難——是心裡難
```

周仓在病床旁边坐下来。

「你叫什么名字？」

老人看着他，眼睛里有了光：「我叫李福生。」

「李福生。」周仓说。

「你是来看我的吗？」

「是。」

老人哭了：「我很久没人来看我了。」

周仓没有说话。他陪着老人坐了一会儿，听老人说话。

他说他年轻的时候在工厂工作。他说他有一个女儿。女儿很乖。女儿结婚了。女儿有了孩子。他说他想看外孙。

但他没有去。他不想给女儿添麻烦。他在医院等女儿来接他。但女儿没有来。他不知道为什么。他等了很久。他老了。他走不动了。他在医院继续等。

周仓听着。他没有说话。他只是陪着老人坐了三个小时。

然后他站起来了。

「李福生。」

「嗯。」

「我会来看你的。」

老人看着他，眼睛里有泪：「你会再来？」

「会。」

「你不骗我？」

「不骗。」

老人笑了。三年来他第一次笑。

周仓走出第三间病房。他继续看其他病房。他需要找到「病人」。

但他已经找到了。

他找到了一个需要被陪伴、需要被记得、需要被看的人。他不需要找更多的病人。他只需要让这个人被找到。

他打开了系统。

```
[通关条件达成]
找到病人：1/1
病人：李福生
位置：旧医院七楼 第三间病房
状态：已被玩家「找到」
```

副本完成了。但不是通过「找到」，是通过「陪伴」。

系统公告弹出：

```
[副本结算]
副本：旧医院七楼
结算评价：S
原因：通过「陪伴」而非「寻找」完成了副本
积分奖励：+3000
特殊成就：「陪伴者」
纪念物品：李福生的合照
```

白光出现。

周仓被传送回了主系统大厅。

他躺在休息室的床上，看着系统背包里的照片。

照片是周仓和李福生的合照——是周仓在病房里陪老人坐的时候，副本系统自动拍的。

他把照片设为了他休息室的背景图。

他每次进入休息室都能看到这个老人在笑。

━━━

【章末钩子】周仓在旧医院七楼「找到」的病人不是生病的病人——是一个无家可归、等了多年没人来看望的老人李福生。他没有「找到」他——他「陪伴」了他。副本以 S 评价结算。

【系统奖励】副本结算 S / 3000 积分 / 陪伴者称号
