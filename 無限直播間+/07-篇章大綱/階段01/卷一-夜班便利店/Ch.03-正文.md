# Ch.03：第一次异常

## 1

脚步声停了。

周仓站在第三排货架的尽头，盯着那个空位。CH-0217 的原味薯片——他确定早上还在。现在不在了。

他没有听到拿东西的声音。

他只听到了脚步声。走近。然后停住。

然后就安静了。

他等了十秒。

什么都没有发生。

便利店的日光灯继续嗡嗡响。关东煮的锅子继续冒热气。一切恢复正常——除了那包消失的薯片。

他慢慢后退，回到收银台。他的后背贴着柜台，视线没有离开过货架区那个方向。

什么都没有。

没有人。没有声音。没有动静。

但他知道有什么东西来过。

弹幕还在刷，但周仓没有在看。他在想一个问题：

那个东西——它拿薯片干什么？

不是吃。如果是吃的，它不会只拿一包原味的——而且是贴着 CH-0217 编号的那包。

它在拿和他有关的东西。

他低头看了看自己的手。手腕内侧，什么都没有。但他总觉得那里应该有什么——一个编号，一条线，一个标记。

他甩了甩头。不要自己吓自己。

傍晚过去了。夜晚降临。

便利店外面的路灯亮了。落地玻璃上映出店内的景象——货架、灯光、他自己的倒影。他站在收银台里，看着玻璃上自己的脸。脸色不太好。但还撑得住。

还有不到四个小时。

## 2

晚上十点。

便利店的门开了。

不是自动门的那种开关——是被人推开的。

风铃响了一声。

周仓抬起头。

门口站着一个人。

不——不是一个人。是一个穿着红色雨衣的身影。

红色。亮红色。像塑料雨衣的那种材质，在日光灯下反射出不自然的光泽。雨帽压得很低，完全遮住了脸。看不到眼睛、嘴巴、下巴——只有一片阴影。

周仓的第一反应：现在没下雨。

第二反应：便利店里有暖气，穿雨衣不热吗？

第三反应——

他看到了规则。

视野左上角弹出一条提示：

```
副本规则：
夜班期间，请勿直视门外。
```

他没有直视门外。

他在门口的人身上。

红雨衣站在门口，没有进来。就站在那里。像一个等待被服务的顾客——但他没有动。一步都没有移动。

周仓的视线从红雨衣身上移开，落在门口旁边的落地玻璃上。

规则说：不要看门外。

但这个人——这个东西——已经进来了。

这算「门外」吗？

他没有时间多想。红雨衣动了。

一步。

很慢。鞋底和瓷砖地面摩擦的声音——不响，但在安静的便利店里听得一清二楚。

两步。

三步。

红雨衣走进店里，走向货架区。

它的路线很固定。从门口直走，经过收银台，左转进入第三排货架。不偏不倚。像是走了很多次。

周仓的视线追着它——不，他没有「直视」它。他用余光看。他盯着自己面前的收银台屏幕，用视野边缘捕捉红雨衣的动作。

红雨衣在第三排货架前停下来。

就是那排。

CH-0217 的薯片消失的那排。

它站在那里，一动不动。

像在找东西。

又像在等人。

周仓的右手手指不自觉地敲着柜台表面。一下。两下。三下。他在数时间。

三十秒。

一分钟。

两分钟。

红雨衣站在那里，整整两分钟。然后它转身——同样的路线——走回门口，推开门，消失在夜色里。

风铃又响了一声。

然后安静了。

周仓呼出一口气——他不知道自己屏了多久的呼吸。

弹幕在刷屏。

```
那是什么？？？
红雨衣
规则里没说要下雨啊
不是 它进店了 但没买东西
它在找东西吧
它在找那包薯片
幹 你不要说出来
它出来的时候手里没拿东西
但它进去的时候也没拿
它到底来干嘛的
```

周仓也在想这个问题。

他抬起头——无意间、完全是习惯性地——扫了一眼墙上的值班表。

然后他停住了。

值班表上多了一个名字。

下午的时候还只有五天的排班：周一到周四林东，周五周仓。

现在多了一个。

```
周六：林东
```

周六。

但现在是周五。

而且这个副本只有七天。

他从哪里冒出来的周六？

周仓盯着那行字看了很久。墨水是干的。字迹和林东的其他字迹一模一样。像是本来就写在那里的。

但他确定，早上还没有。

弹幕也看到了。

```
周六？？？
这副本有七天？？？
不是 等等 刚才那张表有周六吗
没有 我截图了 刚才没有
那现在有了
他写的吧
谁？林东？
不是 值班表上的字是自己出现的
```

周仓伸手碰了一下那张纸。

指尖触摸到的瞬间——视野边缘闪了一下。

不是系统提示框。不是字幕。

是一个残影。

像是有另一个界面在他的视野里叠了一下，又消失了。

太快了。他没有看清是什么。

但他感觉到了。

疼痛。

不是头痛。是后脑勺靠左的位置——一下短暂的刺痛。像被针扎了一下，然后消失。

他揉了揉那个位置。没有肿块。没有伤口。

但痛是真的。

他在交接本上写下：

```
22:00 - 红雨衣顾客进店。
       停留约2分钟，未购物。
       离店后值班表出现周六栏位。
```

写完，他放下笔。

手在微微发抖。

不是因为害怕。

是因为那个刺痛——让他想起了一件事。

一件他以为自己忘记了的事。

## 3

凌晨一点。

便利店安静得像一个盒子。

周仓坐在收银台后面的椅子上，没有睡。他在等。

等下一波。等他不知道会发生的事。

他的积分还剩四百。他盯着商城界面，犹豫着要不要再买一条情报。

还是留着。

积分可能后面有用。

他关掉商城，站起来，走了一圈。货架上的商品没有明显变化。冰柜里的东西也没有多出来或少掉——除了那包薯片。

值班表上，周六的林东还在。

他没有再碰那张纸。

他走回收银台，经过柜台上的小显示器——四个监控画面，轮流切换。

他停下来。

画面的顺序变了。

下午的时候，顺序是：收银台、货架区、冰柜前、门口。

现在顺序是：收银台、货架区、冰柜前、门口——最后一个画面不一样。

不是门口了。

是一个他从来没有见过的角度。

像是监控摄像头被移动过——朝向货架区的深处。那个角度应该能拍到第三排货架尽头的位置。

但画面里，那里有一个货架编号的标签：

`CH-0217`

和他买的薯片编号一样。

和他看到的牛奶盒编号一样。

和他——他在想什么？

他盯着那个编号标签。

这是一个巧合吗？

还是系统在告诉他什么？

他的后脑勺又开始刺痛了——这次更轻。但确实有感觉。

他没有再盯着看。

他移开视线，走回收银台，坐下来。

还有两个小时。

他看了一眼值班表。

周六的林东还在那里。

但字的颜色——

变了。

从黑色变成了深灰色。

像有人在用橡皮擦，慢慢地、一点一点地，把它擦掉。

━━━

【章末钩子】红雨衣顾客第一次进店，没有购物，没有对视。离开后值班表上多了「周六：林东」。周仓触碰值班表时，后脑勺出现刺痛——第一次身体接触系统异常的反应。
