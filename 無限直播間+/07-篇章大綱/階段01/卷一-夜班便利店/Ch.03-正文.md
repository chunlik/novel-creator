# Ch.03：许小小

## 1

03:45。

便利店里很安静。冰柜的压缩机嗡嗡地响。关东煮的锅子还在冒热气。

周仓站在收银台后面，手里拿着交接本。

他做了三件事：

第一，他记下了系统说的「侧目」算一次服务——这意味着规则 #1 的「直视」有量化标准。他需要测试一下什么算「直视」，什么算「侧目」。

第二，他记下了「主观时间」和「副本时间」的差异。这意味着他发呆就是在缩短自己七天的时间。

第三，他开始用副本时间校准他的「体感时间」。每次他做一个动作，他就在交接本上记下系统时间。然后他对比自己的体感时间。

他写下了第一组对照：

```
03:00 副本时间 / 体感时间 03:00
03:10 副本时间 / 体感时间 03:09
03:20 副本时间 / 体感时间 03:18
03:30 副本时间 / 体感时间 03:27
03:40 副本时间 / 体感时间 03:35
```

体感时间比副本时间慢。每次记录他都「思考过」——但副本时间走得比他以为的快。

差距在拉大。

如果不思考——差距会更大。

弹幕在他写字的时候飘过。

```
他在做时间校准
这种活我只在 B+ 级主播身上看到过
一个新人
他会死在第二天。聪明人死得最快
但他好像知道这一点
他写在纸上不在脑子里
因为脑子会骗自己
```

## 2

03:50。

后场门开了。

周仓立刻抬头。

他看到的是一个女孩。

个子不高。大概一米六出头。穿着一件便利店店员的制服——深蓝色 polo 衫，左胸口绣着店名。头发扎在脑后，脸上有一点点困意，但眼睛很亮。

她手里端着一个马克杯。

她看到了周仓。

愣了一下。

「你——」她开口了，「你醒了？」

周仓没有回答。他盯着她。

不是用「直视」的方式。他注意到自己在不自觉地侧着看——因为规则 #1 说「请勿直视门外的任何人或物」。

但她在门里。不是门外。

后场门是开着的。她从门里出来。不是从外面进来。

所以直视她应该不违反规则。

但他还是很谨慎。

「你是谁？」

她笑了一下。

「我也在值夜班。我叫许小小。」

她走过来，把马克杯放在柜台上。热气冒上来——是咖啡。

「给你的。」

「你怎么知道我要醒了？」

「我不知道你什么时候醒。」她说，「但我每天这个时候都会泡一杯。给'还没醒'的你。」

周仓看着那杯咖啡。

「每天都泡？」

「嗯。」

「你在这里多久了？」

她想了想。

「三年。」

三年。

周仓的手指在柜台上敲了一下。

三年——这副本不是七天的吗？她怎么会在一个副本里待三年？

他没有问出口。他把这个疑问记在心里。

「你不喝吗？」她问。

他端起咖啡。

闻了一下——正常的咖啡味。没有编号。

他喝了一口。

温度刚好。糖加得刚好。

「谢谢。」

「不客气。」

她在收银台对面坐下来。和凌晨那些值夜班的店员一样自然。

她从口袋里拿出手机，开始刷视频。

便利店突然变得——正常了。

像是一个普通的、有店员的便利店。

但周仓知道不正常。

他看着许小小。她刷视频的样子很自然——手指在屏幕上划，嘴唇偶尔动一下像在默念什么。

太自然了。

他开口了。

「你刚才说每天这个时候都会泡一杯。你是说我以前也来过？」

许小小停下手里的动作。

她抬起头看他。

表情变了。

不是惊吓——是那种「你终于问了」的表情。

「你——」她说，「你不记得了？」

「记得什么？」

她看着他。看了很久。

然后她笑了。

「没什么。是我记错了。」

她继续刷视频。

但周仓注意到——她刷视频的动作慢了下来。屏幕在她的手指下没有翻动。

她在想事情。

他等了一会儿。

「许小小。」

「嗯？」

「你刚才说'你不记得了'——我应该记得什么？」

她没有立刻回答。

弹幕飘过：

```
她在试探
她不是普通 NPC
她好像在确认什么
```

许小小放下手机。

「你有没有——」她开口，「手腕内侧，有没有感觉什么？」

周仓下意识地看了一下自己的左手腕。

干干净净。什么都没有。

他看向许小小。

她慢慢地卷起自己的左袖子。

她的手腕内侧——有一行浅灰色的数字。

```
CS-0003
```

周仓的心跳漏了一拍。

「这是什么？」

「我的编号。」许小小说，「我是这个副本里的……'残留'。三年前的玩家。」

「三年前——」

「2023 年 11 月 7 号。我和另外六个人一起进了这个副本。我们没有通关。我们都死了。」

「死了——」

「至少系统认为我们死了。」她笑了。那种笑不是快乐的笑。是「无所谓」的笑。

「但你——」周仓说，「你现在坐在这里。」

「是。」她说，「因为我的回收流程没有完成。系统以为我死了——但它没有把我彻底删掉。我的数据残留在副本里。我就一直在这里。」

弹幕炸了。

```
CS-0003 ！！！
她真的是残留数据
我之前以为是任务描述
原来 NPC 是死人？
三年了
她在这个便利店待了三年
```

周仓看着她。

「你说的'另外六个人'——他们呢？」

许小小的表情变了。

「一个完整回收了。两个正在残留。三个——」

她停了一下。

「三个变成了别的东西。」

「别的东西？」

她没有立刻回答。

她看了一眼门口。

「你看到过红雨衣吗？」

周仓点头。

「那个——」她说话的时候声音变小了，「以前是我们的同伴。」

周仓的心沉了下去。

他想起红雨衣想起它没有脸。想起它在门口站着。

那是林东——她曾经的同伴。

「他叫什么？」

「林东。」许小小说。

他。

规则 #5 说：若听到有人在喊「林东」，请不要回头。

规则不是警告他。

规则是在保护他不要叫出这个名字。

因为一旦叫出——林东就会出来。
