# Ch.20b：继续前行

## 5

周仓睁开眼。

他又回到了主系统大厅。

但这次大厅和之前不一样。

大厅多了一个公告屏幕。

屏幕上——显示着：

```
[主系统大厅 - 公告]

P-0217 周仓 玩家
完成阶段一（卷一）所有副本。

阶段一（卷一）副本：
1. 便利店副本 [A]
2. 失物档案馆 [S]
3. 失物招领处 [A]
4. 末班地铁 [A]
5. 旧医院七楼 [S]
6. 第十一层 [A]
7. 最后一天 [A]

总评价：A
阶段一总积分：11010
阶段称号：「阶段完成者」

阶段一（卷一）正式结束。
玩家 P-0217 周仓
进入阶段二（卷二）。

阶段二预告：
下卷将开启新的副本。
副本类型：暂未公布。

感谢玩家 P-0217 周仓
对阶段一的贡献。
```

周仓——看着这个公告。

他——完成了——阶段一。

他——A 评价。

他——11010 积分。

他——进入了——阶段二。

他——很累。

他——很开心。

他——打开了——休息室。

他——躺在床上。

他看着休息室墙上的李福生的合照。

他想起了阶段一里遇到的所有人。

许小小。

林东。

红雨衣。

周远。

王可。

刘一鸣。

赵雪。

陈旧。

小宝。

他的妈妈。

林小雨。

李小燕。

李福生。

李小梅。

老婆婆。

年轻人。

妈妈。

孩子。

他们都在他的阶段一里。

他——帮了——他们。

他被他们帮助了。

他——笑了。

他——关掉——灯。

他——睡着了。

## 6

第二天。

周仓——醒来。

他收到一条消息。

```
[消息]
发件人：主系统
时间：阶段二开放时

P-0217 周仓 玩家：

阶段二（卷二）已开放。
新副本列表已更新。
请在准备好后进入下个阶段。

祝你好运。

——主系统
2026-12-01
```

周仓——看着这条消息。

阶段二——开放了。

他——打开——任务大厅。

```
[任务大厅 - 新副本列表（阶段二）]

1. [预告] 副本：第九区
   类型：剧情类
   难度：四星
   奖励：2000 积分
   备注：阶段二第一个副本

2. [预告] 副本：失物博物馆
   类型：探索类
   难度：四星
   奖励：1800 积分

3. [预告] 副本：完美的一天
   类型：日常类
   难度：三星
   奖励：1500 积分

4. [预告] 副本：再见
   类型：剧情类
   难度：五星
   奖励：3000 积分
```

周仓——看着这个列表。

他——看到了——副本 4——「再见」。

五星。

奖励 3000 积分。

他想接这个副本。

但他——不急。

他——先——休息。

他——躺下来。

他看着休息室的天花板。

他想起了许小小在便利店等了三年。

他想对她说一句话。

「许小小。」

「我——」

「会——」

「经——」

「常——」

「来——」

「看——」

「你——」

「和——」

「林东——」

「便——」

「利——」

「店——」

「永——」

「远——」

「开——」

「着——」

「我——」

「会——」

「回——」

「来——」

他——闭上眼睛。

他——继续——休息。

阶段二——在——等他。

他——想起了那行被涂黑的字。
「不是他的错——是我让他这么做的。」
那句话——还在他脑子里。
谁让他做的？
许小小？
还是——系统？
他不知道。
但他——会带进阶段二。

还有——守门人。
为什么要冒充许小小？
有人在监视他。
谁？
也是系统？
还是——清除者？
他——也会带进阶段二。

他会继续前行。

但便利店永远在他心里。

突然——系统弹出一条红色警告：
[安全协议异常]
检测到异常关联者标记覆盖主系统安全协议
清除者获得主系统大厅入场权限
预计到达时间：30 秒

休息室的门——被推开了。
不是被手推开的。
是被一股黑色的风。

周仓——睁开眼。

门口——站着一个人。
黑色雨衣。没有脸。
清除者。

「P-0217。」清除者的声音——冰冷的——机械的——「时间到了。」

周仓——从床上跳起来。
清除者——冲进来。

周仓——没有时间开门。
他撞破了休息室的窗户
玻璃碎片在空中散开

他——跳了出去——

进入阶段二

白光——吞没了他。

━━━

【章末钩子】周仓完成阶段一（卷一）——总评价 A，进入阶段二。但清除者追进了主系统大厅——他破窗跳入阶段二。便利店永远在他心里，但追猎——刚刚开始。阶段二「再见」五星副本在等待。

【阶段二预告】清除者的追杀不会停止。周仓必须在副本里找到摆脱异常关联者标记的方法——或者——面对清除者。

【系统奖励】阶段一完成 / A 评价 / 11010 积分 / 阶段完成者称号 / 阶段二开放

【积分余额】11010
