# 世界觀衝突檢查

## 文件狀態

【已定案】

本文用於檢查《無限直播間+》高風險設定是否與既有世界觀、角色功能、時間線與終局規劃衝突。

本文不是設定定案文件，而是修改前的審核文件。

---

## 檢查原則

【已定案】

任何會影響 Level A 設定的內容，都必須先經過本文件檢查。

主要檢查項目：

1. 是否違反既有世界公理；
2. 是否造成時間線矛盾；
3. 是否造成角色功能重疊；
4. 是否削弱周倉主角特殊性；
5. 是否提前揭露終局真相；
6. 是否破壞許小小情感主線；
7. 是否讓單一角色承擔過多功能；
8. 是否讓讀者理解負擔過重。

## C-001：林█權重過高風險

### 狀態

【已修復】（2026-06-02，[[CHG-014]]）

### 涉及設定

- [[林█]]
- [[0號樣本]]
- [[名字存續理論]]
- [[歸零事件完整版]]
- [[許小小]]
- [[周倉]]

### 風險描述

林█目前可能同時承擔：

- 名字存續體；
- 0號樣本線索；
- 歸零事件證據；
- 系統真相入口；
- 許小小／X 線關聯。

若所有功能同時成立，林█可能成為過度中心化角色，削弱周倉的主動解謎功能。

### 處理結果（2026-06-02）

林█正式定案為「繼承了理智與冷酷的另一塊碎片（0號樣本），混入系統高層，企圖吞噬許小小以重組完整形態」。功能邊界收斂：林█ = 系統高層線索 + 0號樣本碎片，**不**承擔歸零事件證據與名字存續體角色。周倉在終局（R04）主動定義自身存在，不被林█吸收。

## C-002：0號樣本搶主角功能風險

### 狀態

【已修復】（2026-06-02，[[CHG-014]]）

### 涉及設定

- [[0號樣本]]
- [[周倉]]
- [[未登記者]]
- [[存續人格]]
- [[歸零事件完整版]]

### 風險描述

若 0號樣本同時是：

- 第一個存續人格；
- 第一個未登記者；
- 第一個被刪除後走回來的人；
- 歸零事件導火線；

則會與周倉功能重疊。

### 處理結果（2026-06-02）

0號樣本正式定案為「第一個成功維持薛丁格狀態（兩個正確答案）的生命體，引發歸零事件後碎裂」。定位為**先驅非完成者**：0號樣本碎裂為多個碎片（X = 情感碎片 → 許小小；林█ = 理智碎片；其他殘餘）。周倉（R04）才是「帶著完整自我存續、主動定義自身存在」的第一例，與 0號樣本的「先驅碎裂」形成對比。

## C-003：歸零事件資訊過度集中風險

### 狀態

【已修復】（2026-06-02，[[CHG-014]]）

### 涉及設定

- [[歸零事件完整版]]
- [[第零修復科]]
- [[系統起源]]
- [[0號樣本]]
- [[林█]]
- [[存在校正]]
- [[存續人格]]

### 風險描述

歸零事件目前承載過多內容：

- 系統反應；
- 第零修復科研究成果；
- 0號樣本；
- 存續人格證明；
- 世界可以有兩個正確答案；
- 名字存續可能性。

若不拆分，會導致單一事件負擔過重，難以分層揭露。

### 處理結果（2026-06-02）

歸零事件完整版正式拆分為多檔分散呈現：

- [[歸零事件完整版]]：核心事件（系統邏輯死結 + 切除手術 + 黑水底層）
- [[第零修復科]]：創造悖論的責任方
- [[系統起源]]：系統為何如此反應
- [[0號樣本]]：誰證明了現象，碎裂成何
- [[林█]]：理智碎片的後續

揭露時點分散於階段六（事件重現）、階段七（終局抉擇）與階段八（規則重設）。

## C-004：周倉唯一性不足風險

### 狀態

【已修復】（2026-06-02，[[CHG-014]]）

### 涉及設定

- [[周倉]]
- [[未登記者]]
- [[0號樣本]]
- [[林█]]
- [[Q004-因果是否只有一種結果]]
- [[Q005-誰有資格定義正確存在]]

### 風險描述

目前周倉主要依賴「未登記者」身份作為特殊性。但若後期存在其他更早、更強或更完整的未登記者／存續人格，周倉的主角功能會被削弱。

### 處理結果（2026-06-02）

周倉唯一性於 [[CHG-014]] 補強：

- 周倉 = 「主動定義自身存在」的第一例（Q005 變體）
- 周倉 = 「帶著完整自我存續的主播」的第一例（0號樣本碎裂 = 先驅失敗）
- 周倉 = 「觀測臨界點 = 定位值為零」的首位新範式觀測者（R16）

R04 於 2026-06-02 升為已定案：周倉主動定義自身存在，不被校正、不被抹消、不被 0號樣本碎片吸收。

## C-005：許小小與 X 終局關係風險

### 狀態

【已修復】（2026-06-02，[[CHG-014]]）

### 涉及設定

- [[許小小]]
- [[X]]
- [[存在校正]]
- [[存續人格]]
- [[陳述]]

### 風險描述

許小小與 X 的關係會直接決定本作核心情感答案。

若處理不當，可能導致：

- 許小小被降格為 X 的附屬；
- 「不要把我還回去」失去重量；
- 周倉救她的目標變得模糊；
- 終局情感回收失敗。

### 處理結果（2026-06-02）

[[許小小]] 正式定案為「X 的情感殘留化身的獨立存續人格」。A-004 / R05 已定案：

- 許小小保持獨立人格價值（不降格為 X 附屬）
- 「不要把我還回去」對應階段七抉擇 2：「找回 X 完整碎片 vs 讓許小小以現在的樣子存在」
- 周倉的救援目標 = 讓許小小以現在的樣子繼續存在
