## 分層結構

【已定案】

《無限直播間+》World Bible 採用六層依賴結構：

| 層級 | 類型 | 說明 |
|---|---|---|
| Layer 0 | 核心問題 | 全書最上層問題，不直接提供答案 |
| Layer 1 | 世界公理 | 世界運作的底層規則 |
| Layer 2 | 世界現象 | 由世界公理衍生的現象與機制 |
| Layer 3 | 組織與事件 | 組織、派系、歷史事件與災難事件 |
| Layer 4 | 角色 | 主角、核心角色、配角、功能型角色 |
| Layer 5 | 敘事載體 | 大型篇章、卷／副本、章節 |

設定應由上往下依賴，不應讓低層設定反過來污染高層公理。
